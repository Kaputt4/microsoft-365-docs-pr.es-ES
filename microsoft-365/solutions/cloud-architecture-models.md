# <a name="microsoft-cloud-for-it-architects-illustrations"></a>Nube de Microsoft para ilustraciones de los arquitectos de TI

Estos pósteres de arquitectura en la nube proporcionan información sobre los servicios en la nube de Microsoft, incluidos Microsoft 365, Azure Active Directory (Azure AD), Microsoft Intune, Microsoft Dynamics 365 y soluciones híbridas locales y en la nube. 

Los arquitectos y responsables de la toma de decisiones de TI pueden usar estos recursos para determinar las soluciones ideales para sus cargas de trabajo y tomar decisiones sobre los componentes principales de la infraestructura, como redes, identidades y seguridad.

<a name="zero trust"></a>
### <a name="microsoft-365-zero-trust-deployment-plan"></a>Plan de implementación de Confianza cero de Microsoft 365


En esta ilustración se proporciona un plan de implementación para crear Confianza cero seguridad con Microsoft 365. Confianza cero es un nuevo modelo de seguridad que supone una infracción y comprueba cada solicitud como si se originase en una red no controlada. Independientemente de dónde se origine la solicitud o del recurso al que acceda, el modelo de Confianza cero nos enseña a "no confiar nunca, comprobar siempre".
