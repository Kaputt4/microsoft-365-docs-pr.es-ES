# <a name="microsoft-cloud-for-enterprise-architects-illustrations"></a>Ilustraciones de la nube de Microsoft para arquitectos empresariales

Estos pósteres de arquitectura de la nube le proporcionan información sobre los servicios en la nube de Microsoft, incluidos Microsoft 365, Azure Active Directory (Azure AD), Microsoft Intune, Microsoft Dynamics 365 y soluciones híbridas locales y en la nube. 

Los responsables de tomar decisiones de ti y los arquitectos pueden usar estos recursos para determinar las soluciones ideales para sus cargas de trabajo y para tomar decisiones sobre los componentes principales de la infraestructura, como la red, la identidad y la seguridad.
