# <a name="security-incident-management"></a>Administración de incidentes de seguridad

Después de configurar las funcionalidades de seguridad en Microsoft 365 Empresa Premium, el equipo de seguridad puede supervisar y solucionar las amenazas detectadas. A medida que se detectan amenazas, se generan alertas y se crean incidentes. Las acciones de corrección pueden entrar en juego para ayudar a mitigar las amenazas.
