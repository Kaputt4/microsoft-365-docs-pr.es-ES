# <a name="fortify-your-organization-with-microsoft-365-business-premium"></a>Fortalezca su organización con Microsoft 365 Empresa Premium

Para este primer cometido, configure las defensas de seguridad iniciales inmediatamente. Inicie sesión e inicie el [asistente de instalación](../business/set-up.md?toc=/microsoft-365/campaigns/toc.json) para configurar la seguridad general y las licencias de los equipos. 

Sus objetivos son:

- Registrarse e iniciar sesión en Microsoft 365 Empresa Premium
- Configurar el entorno de Microsoft 365 Empresa Premium

Una vez que haya logrado estos objetivos, puede pasar al [siguiente cometido](m365bp-security-overview.md).
