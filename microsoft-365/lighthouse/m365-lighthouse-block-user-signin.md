# <a name="block-user-sign-in"></a>Bloquear el inicio de sesión de usuario

Puedes bloquear una cuenta de usuario si crees que está en peligro. Cuando se bloquea una cuenta de usuario, se bloquea inmediatamente que cualquier usuario inicie sesión en esa cuenta. Si un usuario intenta iniciar sesión, se cerrará automáticamente de todos los servicios Microsoft 365 en 60 minutos. El bloqueo de una cuenta de usuario no eliminará ningún dato y no impedirá que la cuenta reciba correo.

## <a name="block-sign-in-for-a-user"></a>Bloquear el inicio de sesión para un usuario

1. En el panel de navegación izquierdo de Lighthouse, seleccione **Usuarios**.

2. Seleccione la **pestaña Buscar usuarios** .

3. En el cuadro de búsqueda, escriba el nombre de un usuario.

4. En la lista de resultados de búsqueda, seleccione el usuario.

5. En el panel de detalles del usuario, seleccione **Bloquear inicio de sesión**.

6. En el panel Administrar estado de inicio de sesión, seleccione **Bloquear usuarios para que no inicien sesión**.

7. Seleccione **Guardar**.

## <a name="block-sign-in-for-risky-users"></a>Bloquear el inicio de sesión para usuarios de riesgo

1. En el panel de navegación izquierdo de Lighthouse, seleccione **Usuarios**.

2. Seleccione la **pestaña Usuarios arriesgados** .

3. Seleccione un usuario arriesgado de la lista.

4. En el panel de detalles del usuario, seleccione **Bloquear inicio de sesión**.

5. En el panel Administrar estado de inicio de sesión, seleccione **Bloquear usuarios para que no inicien sesión**.

6. Seleccione **Guardar**.
