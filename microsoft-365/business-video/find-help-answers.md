# <a name="find-docs-and-training"></a>Buscar documentos y entrenamiento

Si tiene una pregunta o necesita ayuda con Microsoft 365 Empresa Premium, tiene varias opciones para obtener ayuda.

## <a name="browse-all-small-business-training"></a>Examinar toda la formación de pequeñas empresas

Para obtener vínculos a todo el soporte técnico de aprendizaje que necesitará usar Microsoft 365 con su pequeña empresa, vea [Microsoft 365 de aprendizaje para pequeñas empresas.](index.yml) 

## <a name="search-for-microsoft-365-admin-center-docs-and-resources"></a>Búsqueda de Centro de administración de Microsoft 365 documentos y recursos

Para buscar tareas y ayuda relacionadas con el administrador, como restablecer contraseñas y asignar suscripciones de Microsoft, [consulte Centro de administración de Microsoft 365 ayuda](../admin/index.yml).

## <a name="join-the-community"></a>Unirse a la comunidad

Inicie una conversación, siga las discusiones y comparta sus conocimientos con otros usuarios de Microsoft 365 pequeñas empresas uniéndose a microsoft [small business Community](https://smallbusiness.microsoft.com/).

Si tiene preguntas específicas sobre otros productos y servicios, publique preguntas y siga las discusiones relevantes en [Ask the Microsoft Community](https://answers.microsoft.com).
