# <a name="choose-a-microsoft-365-subscription"></a>Elegir una suscripción a Microsoft 365

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWfzL9?autoplay=false]

Elegir la suscripción correcta de Microsoft 365 es clave para sacar el máximo provecho del servicio. Aquí&#39;s cómo comparar las opciones y elegir un plan que&#39;s para su empresa.

## <a name="try-it"></a>¿Se atreve?

1. En un explorador, busque  **Microsoft 365 empresa Premium**.
2. Abra la página  **Microsoft 365 Business Premium**  y, a continuación, elija  **Ver planes y precios**. Aquí puede ver qué suscripciones se han adaptado a empresas más pequeñas.
3. Desplácese hacia abajo para ver las características que están disponibles con cada opción.
4. Si tiene una empresa más grande o tiene necesidades de TI complejas, desplácese y seleccione  **Microsoft 365 Enterprise**.
5. Seleccione  **Ver productos y planes** y revise las suscripciones empresariales y sus características.
6. Una vez que&#39;decidido en una suscripción, elija  **comprar ahora** y realice el proceso de registro.
