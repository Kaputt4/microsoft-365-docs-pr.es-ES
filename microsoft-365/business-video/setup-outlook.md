# <a name="set-up-outlook-for-microsoft-365-for-business-email"></a>Configurar Outlook para el correo electrónico de Microsoft 365 para empresas 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/9fe86884-8a83-42cc-bca9-61a12e6dad31?autoplay=false]

## <a name="try-it"></a>¿Se atreve?

Después de instalar las aplicaciones de Office,&#39;desea configurar Outlook para que empiece a usar el correo electrónico, el calendario y los contactos. Aquí&#39;s cómo.

1. Abra el menú Inicio. Busque Outlook y selecciónelo.
2. Escriba su dirección de correo electrónico de 365 de Microsoft y seleccione  **conectar**.
3. Escriba las direcciones de correo electrónico adicionales que quiera usar, como su dirección de correo electrónico personal o anterior. Seleccione  **siguiente**.
4. Si se le solicita, escriba una contraseña y, a continuación, seleccione  **iniciar sesión**.
5. Una vez agregadas todas las cuentas, elija si desea configurar Outlook Mobile o esperar hasta más adelante.
6. Seleccione  **listo**. Outlook puede tardar varios minutos en descargar el correo electrónico y otros datos.

Ahora puede ver el correo electrónico de las cuentas que ha agregado. También puede ver el calendario, los contactos y las tareas.
