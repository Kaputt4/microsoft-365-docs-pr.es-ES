# <a name="install-the-microsoft-365-admin-mobile-app-on-your-phone"></a>Instalar la aplicación móvil de administración de Microsoft 365 en el teléfono

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWe8Kw?autoplay=false]

Puede usar la aplicación móvil de administración de Microsoft 365 para administrar usuarios, supervisar el estado del servicio y realizar otras tareas administrativas desde el teléfono.

## <a name="try-it"></a>¿Se atreve?

1. Para obtener la aplicación móvil de administración de Microsoft 365, busque el  **Administrador de microsoft 365**  en la App Store de su dispositivo e instálelo.
2. Inicie sesión con su dirección de correo electrónico de administrador global y vea el panel para ver el estado del servicio, supervisar las licencias de usuario y ver los mensajes y las solicitudes de servicio.
3. Estas son algunas de las tareas comunes que puede hacer con la aplicación:

### <a name="reset-a-users-password"></a>Restablecer la contraseña de un usuario

1. En  **vínculos rápidos** , elija  **usuarios**.
2. En la página  **usuarios**  , elija el nombre del usuario y, a continuación, elija  **Restablecer contraseña**.
3. Si desea que el usuario deba cambiar su nueva contraseña la primera vez que inicie sesión, active la casilla  **cambiar en el primer inicio de sesión**  y, a continuación, elija  **restablecer**.
4. Cuando se le pregunte si desea confirmar, elija  **sí** y, después,  **Aceptar**.
5. En la ventana  **compartir contraseña**  , puede enviar opcionalmente la nueva contraseña al usuario en un correo electrónico o en un mensaje de texto.

### <a name="add-a-user-to-a-group"></a>Agregar un usuario a un grupo

1. En  **vínculos rápidos** , elija  **grupos**.
2. En la página  **grupos**  , elija el grupo al que desea agregar un usuario y, a continuación, elija  **Editar miembros** y, a continuación,  **Agregar un integrante**.
3. Elija el usuario que desea agregar.
4. Cuando se le pregunte si desea confirmar, elija  **sí** y, después,  **Aceptar**.

### <a name="turn-on-notifications"></a>Activar las notificaciones

1. En el menú principal, elija  **configuración**.
2. En  **configuración** de la aplicación, elija  **notificaciones**.
3. Active las casillas de verificación situadas junto a  **Estado del servicio**  y  **centro de mensajes**.
