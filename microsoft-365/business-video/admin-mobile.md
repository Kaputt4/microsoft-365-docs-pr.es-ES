# <a name="install-the-microsoft-365-admin-mobile-app-on-your-phone"></a>Instalar la Administración de Microsoft 365 móvil en el teléfono

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWe8Kw?autoplay=false]

Puedes usar la aplicación Administración de Microsoft 365 móvil para administrar usuarios, supervisar el estado del servicio y realizar otras tareas de administración desde el teléfono.

## <a name="try-it"></a>¿Se atreve?

1. Para obtener la Administración de Microsoft 365 móvil, busca  Administración de Microsoft 365 en la tienda de aplicaciones para tu dispositivo e instálla.
2. Inicie sesión con su dirección de correo electrónico de administrador global y vea el panel para ver el estado del servicio, supervisar las licencias de usuario y ver mensajes y solicitudes de servicio.
3. Estas son algunas tareas comunes que puedes hacer con la aplicación:

### <a name="reset-a-users-password"></a>Restablecer la contraseña de un usuario

1. En  **Vínculos rápidos,** elija  **Usuarios**.
2. En la  **página Usuarios,**  elija el nombre del usuario y, a continuación, elija  **Restablecer contraseña**.
3. Si desea que el usuario cambie su nueva contraseña al  iniciar sesión por primera vez, active la casilla Cambiar al iniciar sesión primero y, a continuación, elija **Restablecer**.
4. Cuando se le pida que confirme,  **elija Sí** y, a continuación,  **Aceptar**.
5. En la  **ventana Compartir contraseña,**  opcionalmente puede enviar la nueva contraseña al usuario en un correo electrónico o un mensaje de texto.

### <a name="add-a-user-to-a-group"></a>Agregar un usuario a un grupo

1. En  **Vínculos rápidos,** elija  **Grupos**.
2. En la  **página**  Grupos, elija el grupo al que desea agregar un usuario y, a continuación, elija  **Editar** miembros  **y, a continuación, Agregar un miembro**.
3. Elija el usuario que desea agregar.
4. Cuando se le pida que confirme,  **elija Sí** y, a continuación,  **Aceptar**.

### <a name="turn-on-notifications"></a>Activar notificaciones

1. En el menú principal, elija **Configuración**.
2. En  **Configuración de la aplicación,** elija  **Notificaciones**.
3. Active las casillas situadas junto **a Estado del servicio** y Centro de **mensajes.**
