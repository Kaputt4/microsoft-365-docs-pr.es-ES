## <a name="threat-protection"></a>Protección contra amenazas

Microsoft 365 Business Premium incluye Office 365 para la protección contra amenazas avanzada (ATP), un servicio de filtrado de correo electrónico basado en la nube que le protege de malware, ransomware, vínculos dañinos y mucho más. Los vínculos seguros ATP protegen de direcciones URL malintencionadas en documentos de Office o correo electrónico. Los datos adjuntos seguros de ATP protegen del malware y los virus adjuntos a mensajes o documentos.

La autenticación multifactor (MFA) o la verificación en dos pasos requieren la presentación de una segunda forma de autenticación, como un código de comprobación, para confirmar la identidad antes de que pueda obtener acceso a los recursos.  

Windows Defender proporciona protección completa para el sistema, los archivos y las actividades en línea de virus, malware, spyware y otras amenazas.

## <a name="data-protection"></a>Protección de datos

Las características de protección de datos de Microsoft 365 empresa Premium garantizan que los datos importantes permanecen seguros y que solo las personas autorizadas tienen acceso a ellos.

Puede usar las directivas de prevención de pérdida de datos (DLP) para identificar y administrar información confidencial, como la seguridad social o los números de tarjetas de crédito, de modo que no se comparta por error. 

El cifrado de mensajes de Office 365 combina capacidades de cifrado y derechos de acceso para ayudar a garantizar que solo los destinatarios deseados puedan ver el contenido de los mensajes. El cifrado de mensajes de Office 365 funciona con Outlook.com, Yahoo! y gmail y otros servicios de correo electrónico.

Archivado de Exchange Online es una solución de archivado basada en la nube que funciona con Microsoft Exchange o Exchange Online para proporcionar capacidades avanzadas de archivado, incluidas las retenciones y la redundancia de datos. Puede usar directivas de retención para ayudar a su organización a reducir las responsabilidades asociadas con el correo electrónico y otras comunicaciones. Si su empresa tiene que conservar las comunicaciones relacionadas con los litigios, puede usar retenciones de In-Place y retenciones por juicio para conservar el correo electrónico relacionado.

## <a name="device-management"></a>Administración de dispositivos

Las características de administración avanzada de dispositivos de Microsoft 365 Business Premium permiten supervisar y controlar lo que los usuarios pueden hacer con los dispositivos inscritos. Estas características incluyen el acceso condicional, la administración de dispositivos móviles (MDM), BitLocker y las actualizaciones automáticas.

Puede usar directivas de acceso condicional para requerir medidas de seguridad adicionales para determinados usuarios y tareas. Por ejemplo, puede requerir multi-factor Authentication (MFA) o bloquear a los clientes que no admitan el acceso condicional.

Con MDM, puede ayudar a proteger y administrar los dispositivos móviles de los usuarios, como iPhone, iPad, Androids y Windows Phone. Puede crear y administrar directivas de seguridad de dispositivos, borrar de forma remota un dispositivo para quitar todos los datos de la compañía, restablecer un dispositivo a la configuración de fábrica y ver informes detallados de dispositivos. 

Puede habilitar el cifrado de BitLocker para ayudar a proteger los datos en caso de pérdida o robo de un dispositivo, y habilitar la protección contra ataques de Windows para proporcionar protección avanzada contra ransomware.

Puede configurar actualizaciones automáticas para que las actualizaciones y las características de seguridad más recientes se apliquen a todos los dispositivos de usuario.
