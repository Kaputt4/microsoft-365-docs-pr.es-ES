# <a name="use-microsoft-planner-to-plan-an-event"></a>Usar Microsoft Planner para planear un evento

> [!VIDEO https://www.microsoft.com/videoplayer/embed/c8a41e1a-c30d-4adb-ab32-6eac922d0ec0?autoplay=false]

## <a name="try-it"></a>¿Se atreve?

Usa Microsoft Planner para planear un evento, como el seguimiento de un inicio de producto.

### <a name="work-with-buckets"></a>Trabajar con cubos

Usa cubos para ordenar las tareas por tipo, como la planeación de eventos, marketing y anuncios colaterales y relaciones públicas.

1. Seleccione  **Agregar nuevo cubo**  y escriba una nueva categoría de tarea.
2. Seleccione  **+**  para agregar tareas a un cubo y escriba un nombre.
3. Seleccione  **Agregar tarea**.

### <a name="work-with-labels"></a>Trabajar con etiquetas

Usa etiquetas para ver qué tareas necesitan tu atención.

1. Escriba los nombres de las etiquetas.
2. Seleccione un cuadro coloreado para quitar o agregar una etiqueta a una tarea.

## <a name="add-a-comment"></a>Agregar un comentario

Publique un comentario para analizar una tarea con las personas a las que está asignado.

1. Escriba un mensaje en el  **cuadro**  Comentarios.
2. Seleccione  **Enviar**.

### <a name="keep-track-of-tasks"></a>Realizar un seguimiento de las tareas

Use  **gráficos**  **y vista de**  panel para supervisar el progreso de cada miembro del equipo y el proyecto general, o para ordenar y organizar tareas.

- Seleccione  **Gráficos**, luego **Agrupar por** y, por **último, Fecha de vencimiento**  para ver qué tarea vence a continuación.
- Seleccione  **Panel**, luego **Agrupar por** y, por último, **Progreso**  para ordenar las tareas por progreso.

**Nota:**  Las tareas se organizan mediante  **Not started**,  **In progress** y  **Completed**  para que sepas cuándo estás listo para iniciar el producto.
