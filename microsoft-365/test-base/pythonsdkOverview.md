# <a name="test-base-sdk-for-python"></a>Prueba del SDK base para Python

## <a name="overview"></a>Información general
El SDK de Test Base se puede usar para interactuar con el recurso base de prueba de Azure. (Es decir, administre el paquete de aplicación, incluya crear paquete, editar paquete y eliminar paquete).

Con el SDK, el resumen de prueba y el resultado de análisis que se pueden obtener incluyen: scriptExecution, confiabilidad, memoryUtilization, cpuUtilization, memoryRegression, cpuRegression.

Con el SDK de Test Base, puede integrar la base de pruebas en la canalización de CI/CD.

## <a name="client-library"></a>Biblioteca cliente

Instale el paquete base de prueba con pip.

~~~
pip install  Microsoft.Testbase
~~~
