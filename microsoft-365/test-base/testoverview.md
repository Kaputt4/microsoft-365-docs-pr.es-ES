# <a name="test-your-application-on-test-base"></a>Probar la aplicación en Test Base

En esta sección, aprenderá a crear paquetes con diferentes tipos de aplicaciones para cargar y probar en **Test Base**. Los tipos de aplicación constan de los siguientes valores:

   > [!div class="mx-imgBorder"]
   > ![Mensajes de publicación de paquetes](Media/testoverview01.png)

   - **Probar archivos binarios**

      Consulte [Creación y prueba de archivos binarios en Test Base](testapplication.md) para preparar un paquete con un archivo de aplicación binaria (es decir, .exe, .msi) para cargar y probar.

   - **Aplicación Intunewin**

      Consulte [Prueba de la aplicación de Intune en Test Base](testintuneapplication.md) para preparar un paquete con un archivo de aplicación Intune (.intunewin) para la carga y las pruebas.

   - **Paquete Zip precompilado**

      Consulte [Carga del paquete zip precompilado](uploadapplication.md) si ya tiene un paquete integrado sin conexión en formato .zip.

   - **Clonación del paquete existente**

      Consulte [Clonación de un paquete existente](clonepackage.md) para crear un nuevo paquete duplicando el paquete publicado anteriormente.
