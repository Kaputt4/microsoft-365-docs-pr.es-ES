# <a name="microsoft-managed-desktop-devices"></a>Dispositivos de escritorio administrados por Microsoft 

El escritorio administrado de Microsoft evalúa con regularidad los dispositivos que deben incluirse en el servicio. En este tema se describen los requisitos de un dispositivo para su inscripción en el escritorio administrado por Microsoft.

Para obtener más información sobre el escritorio administrado por Microsoft, consulte [documentación y recursos del escritorio administrado de Microsoft](https://docs.microsoft.com/microsoft-365/managed-desktop/).

Además, el dispositivo también debe:     

- Tener una imagen de la fábrica o volver a crear una imagen que:
    - Tiene la versión más reciente de Windows 10 Pro o Enterprise o la versión inmediatamente anterior;
    - Tiene la versión más reciente de Microsoft Office 365 haga clic para ejecutar o inmediato la versión anterior;
    - Tiene instalados los controladores más recientes proporcionados por el OEM; y
    - No tiene software precargado adicional
    - Los dispositivos HP con 5VS04AV deben tener una imagen de dispositivos administrados de Microsoft
- No ha completado la experiencia de primera ejecución de Windows.
- Estar inscritos en [Windows AutoPilot](https://docs.microsoft.com/en-us/windows/deployment/windows-autopilot/user-driven-aad) y tener asignado el perfil de Microsoft Managed Desktop AutoPilot.
