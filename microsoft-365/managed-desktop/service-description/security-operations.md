# <a name="security-operations-in-microsoft-managed-desktop"></a>Operaciones de seguridad en el escritorio administrado por Microsoft

Microsoft Managed Desktop Security Operations Center (SOC) se asocia con su personal de seguridad de la información para mantener protegido su entorno de escritorio. Nuestro equipo recibe y responde a todas las alertas de seguridad de los dispositivos administrados con análisis experto y, cuando es necesario, controla las actividades de respuesta a incidentes de seguridad. Para obtener más información sobre cómo trabajar con SOC, revise la documentación operativa en el portal de administración.

El SOC ofrece cobertura 24/7/365 a los empleados de tiempo completo de Microsoft con experiencia en el panorama actual y emergente de amenazas, incluidos los métodos comunes de ataque a través del software, la red o la adversarios humana.

El SOC proporciona estos servicios:
- Respuesta rápida y precisa a los eventos detectados, con un análisis de los datos para identificar el impacto y evaluar el riesgo global para un dispositivo o su entorno
- Acciones de administración y aislamiento de dispositivos para proteger su entorno de los peligros conocidos o sospechosos, lo que reduce el riesgo al impedir la propagación
- Impulsar el proceso de respuesta ante incidentes de seguridad, garantizando una comunicación oportuna y precisa con el equipo de seguridad
- Análisis y recomendaciones basadas en datos de amenazas y vulnerabilidades para identificar y enfrentar los riesgos antes de que se aprovechen
- La búsqueda avanzada en los dispositivos administrados para identificar indicadores y entidades para las amenazas conocidas y potenciales

## <a name="processes"></a>Sus

- Las operaciones de seguridad de escritorio administradas por Microsoft son atendidas por empleados a tiempo completo de colaboración con el centro de operaciones de Microsoft [Cyber Defense](https://www.microsoft.com/msrc/cdoc). 
- Nuestro SOC usa señales colectivas de toda nuestra empresa, tanto interna como externa, para proteger sus dispositivos (incluso de cosas que todavía no hemos visto en el escritorio administrado de Microsoft).
- Las soluciones de seguridad de Microsoft se alinean con varios estándares de protección de Cybersecurity. Las operaciones de SOC se basan en la guía de administración de respuesta a incidentes de seguridad y de equipos tecnológicos del Instituto Nacional (NIST 800-61 R2).
- El proceso permite una recopilación de información y pruebas adecuadas para el análisis y la documentación y la información posterior a la recuperación de formas de defender mejor su entorno a través de estas fases:
    - Preparación, detección y análisis
    - Contención
    - Erradicación
    - Recuperación
    - Actividad posterior al incidente
- Los clientes de escritorio administrados por Microsoft pueden inscribirse en el servicio de Microsoft Threat Experts. El SOC relacionarse con este servicio para comprender mejor las complejas amenazas que afectan a su organización, incluidas las consultas de alertas, los dispositivos potencialmente comprometidos, la causa de origen de una conexión de red sospechosa y la inteligencia adicional de amenazas sobre las campañas de amenazas persistentes avanzadas en curso. Para obtener más información, consulte [Microsoft Threat Experts](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-threat-experts).
- El proceso de administración de amenazas y vulnerabilidades de SOC usa algunos de los servicios de Microsoft para ayudar a informar de las recomendaciones a su organización sobre la protección contra amenazas. El SOC consume datos de su centro de seguridad de Microsoft defender para extremo y de los orígenes de datos de vulnerabilidad pertinentes dentro y fuera de Microsoft para descubrir vulnerabilidades y configuraciones inadecuadas y proporcionar informes que requieren acción.
