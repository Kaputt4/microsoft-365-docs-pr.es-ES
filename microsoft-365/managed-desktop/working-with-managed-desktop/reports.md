# <a name="work-with-reports"></a>Trabajar con informes

El escritorio administrado de Microsoft proporciona informes que los administradores de TI de su inquilino pueden usar para comprender diversos aspectos de la población de dispositivos. Puede acceder a los informes en el [portal de administración ](https://aka.ms/mmdportal)en **informes**.


## <a name="device-compliance-report"></a>Informe de cumplimiento de dispositivos

La vista principal muestra el porcentaje de los dispositivos de escritorio administrados por Microsoft que se ajustan a las directivas de cumplimiento establecidas en Microsoft Intune. También puede ver información específica para todos los dispositivos de escritorio administrados de Microsoft o solo para los dispositivos que no se ajusten a las directivas de cumplimiento y para exportar un archivo con la misma información.

Obtenga más información sobre el [Informe de cumplimiento de dispositivos](device-compliance-report.md).

## <a name="device-inventory-report"></a>Informe de inventario de dispositivos

La vista de **dispositivos** ofrece una ficha **exportar todo** que [descarga un informe de inventario detallado](device-inventory-report.md).
