# <a name="work-with-reports"></a>Trabajar con informes

El escritorio administrado de Microsoft proporciona informes que los administradores de TI de su inquilino pueden usar para comprender diversos aspectos de la población de dispositivos. Puede acceder a los informes en el [portal de administración ](https://aka.ms/mmdportal)en **informes**.



## <a name="device-inventory-report"></a>Informe de inventario de dispositivos

La vista de **dispositivos** ofrece una ficha **exportar todo** que [descarga un informe de inventario detallado](device-inventory-report.md).
