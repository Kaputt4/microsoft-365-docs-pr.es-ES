# <a name="app-usage-report"></a>Informe de uso de aplicaciones

Este informe le ayuda a comprender cómo se usan las aplicaciones en los dispositivos de Escritorio administrado de Microsoft. También puede actuar como referencia para ayudarle a evaluar cualquier efecto en los usuarios cuando se detectan problemas de aplicación. 

La información de este informe incluye: 

- Nombre de la aplicación: las aplicaciones con cualquier cantidad de uso notificado aparecerán en esta lista 
- Uso de primer plano: tiempo invertido en interactuar con la aplicación en primer plano que se muestra en horas. 
- Uso medio del día de la semana: uso promedio por dispositivo excepto los fines de semana 
- Recuento de dispositivos: el número de dispositivos de informes que contribuyen al uso por aplicación 
- % de dispositivos de informes: el porcentaje del total de dispositivos de informes que han usado esta aplicación  

> [!IMPORTANT]
> Para que los dispositivos informen datos, deben establecerse en el nivel de datos de diagnóstico opcional. Obtenga más información [sobre cómo Microsoft Managed Desktop usa Windows de diagnóstico](../service-description/privacy-personal-data.md).
