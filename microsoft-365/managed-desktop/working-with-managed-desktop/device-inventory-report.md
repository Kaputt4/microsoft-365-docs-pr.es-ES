# <a name="device-inventory-report"></a>Informe de inventario de dispositivos

En la **vista Dispositivos,** puede seleccionar la pestaña Exportar **todo** para descargar un archivo delimitado por comas, incluida esta información:

- Número de serie
- Nombre del dispositivo
- Última entrada del dispositivo
- Administrado por
- Fabricante
- Model
- Sistema operativo
- Licencia del sistema operativo
- Versión del sistema operativo
- Ownership
- Fecha de inscripción
- Nombre del anillo
- Storage: gratuito en GB
- Storage: total en GB
- Inscrito por UPN de usuario
- Usuario asignado
- Estado
- Edad (meses)
- Perfil
- Usuarios que han iniciado sesión
- Versión de TPM
- Arranque seguro habilitado
- Tipo de disco principal
- Memoria física total
