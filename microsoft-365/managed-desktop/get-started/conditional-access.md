# <a name="adjust-conditional-access"></a>Ajustar el acceso condicional

Si usa directivas de [acceso condicional](https://docs.microsoft.com/azure/active-directory/conditional-access/overview) en su organización, tendrá que configurarlas para excluir determinadas cuentas para que Microsoft Managed Desktop pueda funcionar correctamente.

Para ello, siga estos pasos:

1. Consulte la sección "pasos de reversión" de [Cómo: planear la implementación de acceso condicional en Azure Active Directory](https://docs.microsoft.com/azure/active-directory/conditional-access/plan-conditional-access#rollback-steps).
2. Siga los pasos que se indican a continuación para excluir el grupo de *cuentas del servicio de lugar de trabajo moderno* para todas las directivas.


Si tiene algún problema con el acceso condicional, póngase en [contacto con el administrador.](../working-with-managed-desktop/admin-support.md)
