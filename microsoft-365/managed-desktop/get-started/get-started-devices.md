# <a name="get-your-users-ready-to-use-devices"></a>Prepare a los usuarios para que usen dispositivos

Una vez que un dispositivo de Escritorio administrado de Microsoft está en manos del usuario, empezar es rápido y fácil. Los dispositivos vienen preconfigurados con la versión actual de Windows y configuraciones, y las aplicaciones se instalan desde la nube a medida que el usuario completa la instalación.

Para facilitar aún más la introducción, ofrecemos una guía que guía a los usuarios a través de la configuración inicial. La guía proporciona recursos útiles tanto para la configuración como para su uso posterior, si es necesario. Puede personalizar la siguiente guía para incluir determinados detalles específicos de su organización. A continuación, distribuya la guía directamente a los usuarios junto con su dispositivo.

## <a name="prepare-the-guide"></a>Preparar la guía

**Para preparar la guía:**

1. Descargar el [Escritorio administrado de Microsoft: introducción a la guía del](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-custom-v2.pdf) dispositivo.
2. Usa cualquier aplicación capaz de abrir archivos PDF para especificar detalles relevantes para tu organización:
    - Nombre de la red a la que los usuarios deben conectarse para continuar con la instalación (paso 3 de la guía).
    - Nombre de la cuenta de inquilino de Azure de su organización (paso 4 de la guía).
    - Información de contacto para la compatibilidad interna de TI de su organización (parte superior de la segunda página).
3. Guarde el PDF editado y, a continuación, distribuya a los usuarios.

## <a name="ready-to-use-guide"></a>Guía lista para usar

También proporcionamos una versión más genérica de la guía para aquellas organizaciones que no necesitan personalizarla.

Solo tienes que [descargar la guía Escritorio administrado de Microsoft: introducción a tu dispositivo (listo para usar](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-v2.pdf) ).

En este momento, estás listo para pasar a implementar [aplicaciones](deploy-apps.md).

## <a name="steps-to-get-started-with-microsoft-managed-desktop"></a>Pasos para empezar con Microsoft Managed Desktop

1. Acceder al [portal de administrador](access-admin-portal.md).
1. [Agregar y verificar los contactos de administración en el portal de administrador](add-admin-contacts.md).
1. [Ajustar la configuración después de la inscripción](conditional-access.md).
1. Implementar y asignar el [Portal de empresa de Intune](company-portal.md).
1. [Asignar las licencias](assign-licenses.md).
1. [Implementar las aplicaciones](deploy-apps.md).
1. [Configurar los dispositivos](set-up-devices.md).
1. Configurar la [experiencia de primera ejecución con el Autopilot y la página de estado de inscripción](esp-first-run.md).
1. [Habilitar las características de soporte técnico para el usuario](enable-support.md).
1. Prepare a los usuarios para usar dispositivos (en este artículo).
1. [Comenzar a usar el control de aplicaciones](get-started-app-control.md).
