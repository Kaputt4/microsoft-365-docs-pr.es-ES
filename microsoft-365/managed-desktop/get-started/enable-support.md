# <a name="enable-user-support-features"></a>Habilitar características de soporte técnico para usuarios

Ya sea que proporciones soporte técnico a tu propio usuario o trabajes con un partner para proporcionar soporte técnico, sigue los pasos que se indican a continuación para permitir que el proveedor de soporte técnico solicite acceso a dispositivos elevados o escala los problemas a Escritorio administrado de Microsoft, si es necesario.

1. Si aún no tienen una, los usuarios necesitan una cuenta en el mismo dominio Azure Active Directory (AAD) que los dispositivos de Escritorio administrado de Microsoft.
1. Agregue las cuentas de usuario al grupo de seguridad De **Roles-Support De** trabajo moderno en el Azure Active Directory (AAD).
