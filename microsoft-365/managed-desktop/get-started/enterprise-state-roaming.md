# <a name="enable-enterprise-state-roaming"></a>Habilitar Enterprise State Roaming

Para obtener la mejor experiencia con el Escritorio administrado de Microsoft, habilite [Enterprise State Roaming,](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-overview)que permite a los usuarios sincronizar de forma segura los datos de configuración de usuario y aplicación en la nube. Esto significa que tendrán la misma experiencia independientemente del dispositivo Windows en el que inicien sesión. Por ejemplo, si reemplaza uno de sus dispositivos de Escritorio administrado de Microsoft por uno nuevo, tendrá el mismo aspecto y comportamiento que el último.

Para habilitar Enterprise State Roaming, sigue los pasos de [Habilitar Enterprise State Roaming](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-enable)en Azure Active Directory y, a continuación, vuelve a esta documentación.

Si tiene alguna dificultad con Enterprise State Roaming, póngase en contacto con el soporte técnico de [administración.](../working-with-managed-desktop/admin-support.md)
