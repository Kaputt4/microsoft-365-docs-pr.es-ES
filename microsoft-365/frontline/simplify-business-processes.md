# <a name="simplify-business-processes-for-frontline-teams"></a>Simplifique los procesos empresariales de los equipos de primera línea

Puede usar las aplicaciones de Teams, Power Apps y Power BI para simplificar los procesos empresariales de los equipos de primera línea.

> [!NOTE]
> Las nubes del sector de Microsoft ofrecen funcionalidades adicionales para [servicios financieros](/industry/financial-services), [asistencia sanitaria](/industry/healthcare), [ONG](/industry/nonprofit) y entornos [comerciales](/industry/retail), que incluyen funcionalidades de Dynamics 365 y Microsoft Azure, así como las características de Microsoft 365.

## <a name="create-custom-apps-for-your-organizations-needs"></a>Crear aplicaciones personalizadas para las necesidades de su organización

Agregue Power Apps para crear aplicaciones con poco código o sin código para su organización. Puede crear aplicaciones personalizadas para conectar los datos empresariales, de modo que pueda administrar el inventario o realizar recorridos por la tienda, por ejemplo.

> [!TIP]
> Se proporcionan ejemplos para los sectores de servicios financieros, asistencia sanitaria, ONG y comercio minorista, pero puede usar estas aplicaciones para una organización en cualquier sector.

|Industria |Ejemplo |
|:--------|:-------|
|Servicios financieros |Puede crear una aplicación para que los administradores de relaciones y cuentas realicen un seguimiento de las llamadas y los correos electrónicos a los clientes. |
|Salud |Puede crear una aplicación para realizar un seguimiento del inventario de consumibles en las salas de exámenes para asegurarse de que todas las salas están listas para los pacientes. |
|Venta minorista | Puede crear una aplicación para administrar y realizar un seguimiento del inventario, o para realizar recorridos de la tienda donde una persona responsable comprueba todas las áreas de una tienda antes de abrirla. |
|Industria |Puede crear una aplicación para realizar un seguimiento de las inspecciones de maquinaria y equipos, así como las necesidades de reparación. |

Más información: [Power Apps](/microsoftteams/manage-power-platform-apps) e [Integración de Power Apps y Microsoft Teams](/powerapps/teams/overview).

## <a name="track-key-performance-indicators-kpis-with-power-bi-reports"></a>Realice un seguimiento de los indicadores clave de rendimiento (KPI) con informes de Power BI.

Comparta y colabore en contenido interactivo de Power BI en canales y chats de Microsoft Teams. Puede agregar una [pestaña de Power BI](/microsoftteams/platform/tabs/what-are-tabs) a Teams para insertar informes interactivos y chatear en Teams sobre los informes, y recibir notificaciones cuando sucedan cosas importantes en Power BI directamente en la fuente de actividades de Teams.

Más información: [Colaborar en Microsoft Teams con Power BI](/power-bi/collaborate-share/service-collaborate-microsoft-teams).
