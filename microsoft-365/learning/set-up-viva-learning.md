# <a name="set-up-microsoft-viva-learning-in-the-teams-admin-center"></a>Configurar Aprendizaje Microsoft Viva en el Centro Teams administración

Viva Learning está habilitado de forma predeterminada para todos Microsoft Teams usuarios de la organización. Puedes desactivar o activar Viva Learning en el nivel  de la organización en la página Administrar aplicaciones del Centro de administración Microsoft Teams administración. Para obtener más información, vea [Manage your apps in the Microsoft Teams admin center](/microsoftteams/manage-apps).

Para permitir o impedir que usuarios específicos de la organización utilicen Viva Learning, asegúrese  de que Viva Learning esté activado para su organización en la página Administrar aplicaciones del Centro de administración de Microsoft Teams. A continuación, cree una directiva de permisos de aplicación personalizada y asígnela a esos usuarios. Para obtener más información, consulta [Administrar directivas de permisos de aplicaciones en Teams](/microsoftteams/teams-app-permission-policies).
