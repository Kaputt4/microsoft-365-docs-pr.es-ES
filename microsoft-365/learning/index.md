# <a name="introduction-to-microsoft-viva-learning"></a>Introducción a Aprendizaje Microsoft Viva

Aprendizaje Microsoft Viva permite a los empleados hacer que el aprendizaje sea una parte natural del día al incorporar el aprendizaje al flujo de trabajo dentro de las herramientas y plataformas que ya usan. Descubra, comparta y realice un seguimiento del aprendizaje en una variedad de orígenes de un centro de aprendizaje de Microsoft Teams.

Los recursos de esta sección le ayudarán a obtener más información sobre lo que Viva Learning es y cómo puede ayudar a su organización.
