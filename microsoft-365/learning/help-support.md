# <a name="help-and-support"></a>Ayuda y soporte técnico

Puede obtener ayuda con Viva Learning mediante la presentación de una solicitud de soporte técnico.

[Obtenga información sobre cómo presentar una solicitud de soporte técnico.](/microsoft-365/business-video/get-help-support)
