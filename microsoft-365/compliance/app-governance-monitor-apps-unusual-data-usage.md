# <a name="monitor-and-respond-to-apps-with-unusual-data-usage"></a>Supervisión y respuesta a aplicaciones con uso de datos inusual

La gobernanza de aplicaciones proporciona información de uso de datos que puede ayudarle a identificar actividades de aplicaciones no deseadas y potencialmente malintencionadas. Puede usar esta información a través de las secciones siguientes del portal:

- Tarjeta de uso de datos: proporciona el uso total de datos a lo largo del tiempo, resaltando picos repentinos en la actividad total de carga y descarga de todas las aplicaciones de Microsoft 365. Esta tarjeta también proporciona información de uso por separado para varios recursos, como archivos y correo electrónico, para que pueda identificar los recursos que las aplicaciones podrían estar usando de forma incorrecta.
- Panel de detalles de la aplicación: al seleccionar una aplicación en la página Aplicaciones, un panel flotante muestra información detallada sobre la aplicación. En el panel, obtenga información de uso de datos específica de la aplicación por tipo de recurso, así como patrones de carga y descarga a lo largo del tiempo.
- Condiciones de directiva: puede crear directivas que marquen y desactiven automáticamente las aplicaciones cuyo uso de datos coincida con las siguientes condiciones:
  - Uso de datos: la cantidad total de descargas y cargas supera el umbral especificado
  - Tendencia de uso de datos: el aumento porcentual de la cantidad total de descargas y cargas en comparación con el día anterior alcanza el umbral especificado
