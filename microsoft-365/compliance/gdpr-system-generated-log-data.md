Si quiere que estos datos seudonimizados estén disponibles para un usuario y necesita ayuda para ello, póngase en contacto con el equipo de [soporte técnico de Microsoft](https://support.microsoft.com/contactus/).</span><span class="sxs-lookup"><span data-stu-id="ab334-p103">**Workplace Analytics**: The data log export tool provides usage data for those users in your organization who have permission to run Workplace Analytics reports.
