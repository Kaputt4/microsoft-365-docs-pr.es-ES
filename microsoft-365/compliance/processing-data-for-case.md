# <a name="work-with-processing-errors-in-advanced-ediscovery"></a>Trabajar con errores de procesamiento en eDiscovery avanzado

El *procesamiento* es el proceso de identificación de archivos, expansión de documentos y datos adjuntos incrustados, extracción de texto y reconocimiento óptico de caracteres (OCR) de archivos de imagen y la posterior indización de ese contenido.  

Cuando agrega administradores y orígenes de datos que no  son administradores a un caso en la pestaña Orígenes, todos los elementos parcialmente indizados de Microsoft 365 se procesan para que sean totalmente de búsqueda. Del mismo modo, cuando se agrega contenido a un conjunto de revisión de orígenes de datos de Microsoft 365 y de orígenes de datos que no son de Microsoft 365, también se procesa este contenido.

La **pestaña** Procesamiento de eDiscovery avanzado proporciona información sobre el estado de la indización avanzada para diferentes escenarios de procesamiento.

Para más información, consulte los siguientes artículos:

- [Clasificación avanzada de los datos de administración](indexing-custodian-data.md)

- [Corrección de errores al procesar los datos](error-remediation-when-processing-data-in-advanced-ediscovery.md)

- [Corrección de errores de un único elemento](single-item-error-remediation.md)
