# <a name="work-with-processing-errors-in-ediscovery-premium"></a>Trabajar con errores de procesamiento en eDiscovery (Premium)

*El procesamiento* es el proceso de identificación de archivos, expansión de documentos y datos adjuntos incrustados, extracción de texto y reconocimiento óptico de caracteres (OCR) de archivos de imagen y la indexación posterior de ese contenido.  

Al agregar custodios y orígenes de datos no custodios a un caso en la pestaña **Orígenes** , todos los elementos indizados parcialmente de Microsoft 365 se procesan para que se puedan buscar por completo. Del mismo modo, cuando se agrega contenido a un conjunto de revisión de orígenes de datos de Microsoft 365 y que no son de Microsoft 365, este contenido también se procesa.

La pestaña **Procesamiento** en eDiscovery (Premium) proporciona información sobre el estado de la indexación avanzada para los diferentes escenarios de procesamiento.

Para más información, consulte los siguientes artículos:

- [Clasificación avanzada de los datos de administración](indexing-custodian-data.md)

- [Corrección de errores al procesar los datos](error-remediation-when-processing-data-in-advanced-ediscovery.md)

- [Corrección de errores de un único elemento](single-item-error-remediation.md)
