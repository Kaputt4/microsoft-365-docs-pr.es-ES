## <a name="restore-inactive-mailboxes"></a><span data-ttu-id="4e49a-127">Restaurar buzones inactivos</span><span class="sxs-lookup"><span data-stu-id="4e49a-127">Restore inactive mailboxes</span></span>

   ```

   > [!IMPORTANT]
   > <span data-ttu-id="4e49a-p107">En el comando anterior, use el valor de la propiedad **DistinguishedName** o **ExchangeGUID** para identificar el buzón inactivo. 2. <span data-ttu-id="4e49a-p108">Restaurar el contenido del buzón inactivo en un buzón existente.    ```

   <span data-ttu-id="4e49a-p109">Como alternativa, puede especificar una carpeta de nivel superior en el buzón de destino en el que se va a restaurar el contenido del buzón inactivo.    <span data-ttu-id="4e49a-137">En este ejemplo se copian los elementos del buzón y las subcarpetas de un buzón inactivo a una carpeta denominada "Buzón inactivo" en la estructura de carpetas de nivel superior del buzón de destino.</span><span class="sxs-lookup"><span data-stu-id="4e49a-137">This example copies mailbox items and subfolders from an inactive mailbox to a folder named "Inactive Mailbox" in the top-level folder structure of the target mailbox.</span></span>

   ```
