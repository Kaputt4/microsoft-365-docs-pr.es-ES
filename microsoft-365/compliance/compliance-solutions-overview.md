# <a name="compliance-solutions"></a>Soluciones de cumplimiento
En esta biblioteca, se proporcionan recursos técnicos para usar funciones en Microsoft 365 que permiten a muchas organizaciones cumplir distintos reglamentos, empezando con el [RGPD](gdpr.md). 

Hasta que agreguemos más contenido a esta biblioteca, le recomendamos que vea los recursos de cumplimiento del Centro de confianza de Microsoft:
- [Información general sobre cumplimiento](https://www.microsoft.com/trustcenter/compliance)
- [Ofertas de cumplimiento](https://www.microsoft.com/trustcenter/compliance/complianceofferings)
