# <a name="breach-notification-under-the-gdpr"></a>Notificación de infracciones de datos según el RGPD

Microsoft se toma muy en serio sus obligaciones relacionadas con el Reglamento general de protección de datos (RGPD). Para obtener información sobre cómo los servicios Microsoft le protegen frente una infracción de datos personales y cómo respondemos y le notificamos en caso de que se produzca una infracción, vea estos temas:

- [Office 365](gdpr-breach-Office365.md) 
- [Azure](gdpr-breach-Azure.md)
- [Dynamics 365](gdpr-breach-Dynamics365.md)
- [Soporte técnico y servicios profesionales de Microsoft](gdpr-breach-Microsoft-Support-Professional-Services.md)

Para obtener más información sobre cómo Microsoft detecta y responde ante una infracción de datos personales, vea [Notificación de infracciones de datos según el RGPD](https://servicetrust.microsoft.com/ViewPage/GDPRBreach) en el Portal de confianza del servicio.
