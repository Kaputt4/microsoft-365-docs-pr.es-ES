# <a name="ediscovery-solutions-in-microsoft-365"></a>Soluciones de exhibición de documentos electrónicos en Microsoft 365

La exhibición de documentos electrónicos, o eDiscovery, es el proceso de identificación y entrega de información electrónica que se puede usar como prueba en casos legales. Puede usar herramientas de exhibición de documentos electrónicos en Microsoft 365 para buscar contenido en buzones de Exchange Online, Grupos de Microsoft 365, Microsoft Teams, sitios de SharePoint Online y OneDrive para la Empresa, conversaciones de Skype Empresarial y equipos de Yammer. Puede buscar buzones y sitios en la misma búsqueda de exhibición de documentos electrónicos mediante la herramienta búsqueda de contenido. Además, puede usar los casos principales de exhibición de documentos electrónicos para identificar, retener y exportar el contenido que se encuentra en buzones y sitios. Si su organización tiene una suscripción a Office 365 E5 o Microsoft 365 E5 (o suscripciones de complementos E5 relacionadas), puede administrar más a los custodios y analizar el contenido mediante la solución de exhibición de documentos electrónicos avanzada en Microsoft 365.
  
Microsoft 365 proporciona las siguientes herramientas de exhibición de documentos electrónicos:
  
- [Búsqueda de contenido](#content-search)

- [Core eDiscovery](#core-ediscovery)

- [eDiscovery avanzado](#advanced-ediscovery)

## <a name="advanced-ediscovery"></a>eDiscovery avanzado

La solución de exhibición de documentos electrónicos avanzada de Microsoft 365 (también denominada *eDiscovery avanzada v2.0)* se basa en las capacidades de análisis y exhibición de documentos electrónicos existentes en Microsoft 365. Esta solución de exhibición de documentos electrónicos proporciona un flujo de trabajo completo para conservar, recopilar, revisar, analizar y exportar contenido que responda a las investigaciones internas y externas de la organización. También permite a los equipos legales administrar los custodios y todo el flujo de trabajo de notificación de retención legal para comunicarse con los custodios implicados en un caso.
