# <a name="download-selected-documents-from-a-review-set"></a>Descarga de documentos seleccionados de un conjunto de revisión

[!include[Purview banner](../includes/purview-rebrand-banner.md)]

Descargar ofrece una manera sencilla de descargar contenido de un conjunto de revisión en formato nativo. La herramienta de descarga de eDiscovery (Premium) usa las características de transferencia de datos del explorador. Aparecerá un mensaje del explorador cuando una descarga esté lista. Los archivos descargados mediante este método se comprimen en un archivo contenedor y contendrá archivos de nivel de elemento. Significa que si selecciona descargar un archivo adjunto, recibirá el mensaje de correo electrónico con los datos adjuntos incluidos. De forma similar, si exporta una hoja de cálculo de Excel incrustada en un documento de Word, el documento de Word y la hoja de cálculo Excel incrustada se incluyen en la descarga. Al descargar elementos, la propiedad Datos modificados por última vez se conserva y se puede ver como una propiedad de archivo.

Para descargar contenido de un conjunto de revisión en un caso de exhibición de documentos electrónicos (Premium), empiece por seleccionar los archivos que desea descargar y, a continuación, seleccione **Elementos** >  de **acciónDescargar**.
