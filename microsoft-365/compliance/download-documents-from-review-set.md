# <a name="download-selected-documents-from-a-review-set"></a>Descargar documentos seleccionados de un conjunto de revisión

La descarga ofrece una forma sencilla de descargar contenido de un conjunto de revisión en formato nativo. La herramienta de descarga de Advanced eDiscovery aprovecha las características de transferencia de datos del explorador. Aparecerá un mensaje del explorador cuando una descarga esté lista. Los archivos descargados con este método se comprimieron en un archivo contenedor y contendrán archivos de nivel de elemento. Esto significa que si selecciona descargar datos adjuntos, recibirá el mensaje de correo electrónico con los datos adjuntos incluidos. Del mismo modo, si exporta una hoja de cálculo Excel incrustada en un documento de Word, el documento de Word con la hoja de cálculo Excel incrustada se incluirá en la descarga. Al descargar elementos, la propiedad Last Modified Data se conserva y se puede ver como una propiedad de archivo.

Para descargar contenido de un conjunto de revisión en un caso Advanced eDiscovery, empiece seleccionando los archivos que desea descargar y seleccione **Elementos de** acción  >  **Descargar**.
