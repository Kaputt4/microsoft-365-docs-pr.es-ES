# <a name="download-selected-documents-from-a-review-set"></a>Descargar documentos seleccionados de un conjunto de revisión

La descarga ofrece una forma sencilla de descargar contenido de un conjunto de revisión en formato nativo. La herramienta de descarga en eDiscovery avanzado aprovecha las características de transferencia de datos del explorador. Cuando una descarga esté lista, aparecerá un mensaje del explorador. Los archivos descargados mediante este método se comprimen en un archivo contenedor y contienen archivos de nivel de elemento. Esto significa que si selecciona descargar un archivo adjunto, recibirá el mensaje de correo electrónico con los datos adjuntos incluidos. De forma similar, si exporta una hoja de cálculo de Excel que está incrustada en un documento de Word, el documento de Word con la hoja de cálculo de Excel incrustada se incluye en la descarga. Cuando se descargan elementos, la última propiedad de datos modificada se conserva y puede verse como una propiedad de archivo.

Para descargar contenido de un conjunto de revisión en un caso de exhibición avanzada de documentos electrónicos, comience seleccionando los archivos que desea descargar y, a continuación, seleccione descarga de **elementos de acción**  >  **Download**.
