# <a name="plan-for-security-amp-compliance"></a>Planeamiento del cumplimiento de seguridad &amp;

La administración de la seguridad y el cumplimiento es una asociación. Usted es responsable de proteger sus datos, identidades y dispositivos, mientras que Microsoft protege vigorosamente los servicios de Microsoft 365. Puede usar Microsoft 365 y Enterprise Mobility + Security (EMS) juntos para ayudarle a lograr el nivel adecuado de protección para su organización.
  
## <a name="step-1-review-capabilities"></a>Paso 1: Revisar funcionalidades

Orientarse a las capacidades de protección de la información en el Information Protection para Office 365 póster. 
  
[Implementación de la protección de la información para las regulaciones de privacidad de datos con Microsoft 365](../solutions/information-protection-deploy.md)
  
## <a name="step-2-check-your-secure-score"></a>Paso 2: Comprobar la puntuación de seguridad

Después de configurar la suscripción de Microsoft 365, tome nota de la puntuación inicial. Puntuación segura proporciona sugerencias de configuración que puede tomar para aumentar la puntuación. El objetivo es tener en cuenta las oportunidades que puede aprovechar para proteger el entorno, lo que no afectará negativamente a la productividad de los usuarios.
  
- [Puntuación de seguridad de Microsoft](../security/defender/microsoft-secure-score.md)
