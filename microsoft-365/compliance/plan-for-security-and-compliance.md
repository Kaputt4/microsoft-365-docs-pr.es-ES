# <a name="plan-for-security-amp-compliance"></a>Planear el cumplimiento de &amp; la seguridad

La administración de la seguridad y el cumplimiento es una asociación. Usted es responsable de proteger sus datos, identidades y dispositivos, mientras que Microsoft protege vigorosamente los servicios de Microsoft 365. Puede usar Microsoft 365 y Enterprise Mobility + Security (EMS) juntos para ayudarle a lograr el nivel de protección adecuado para su organización.
  
## <a name="step-1-review-capabilities"></a>Paso 1: Revisar funcionalidades

Orientese a las capacidades de protección de la información en el póster de Information Protection for Office 365. 
  
[Implementar la protección de la información para las normativas de privacidad de datos con Microsoft 365](../solutions/information-protection-deploy.md?view=o365-worldwide)
  
## <a name="step-2-check-your-secure-score"></a>Paso 2: Comprobar la puntuación segura

Después de configurar la suscripción a Microsoft 365, toma nota de la puntuación inicial. Puntuación segura proporciona sugerencias de configuración que puede tomar para aumentar la puntuación. El objetivo es tener en cuenta las oportunidades que puede aprovechar para proteger su entorno, lo que no afectará negativamente a la productividad de los usuarios.
  
- [Presentación de la puntuación segura de Office 365](../security/mtp/microsoft-secure-score.md)
    
## <a name="step-3-plan-access-protection-for-identity-and-devices"></a>Paso 3: Planear la protección de acceso para identidades y dispositivos

Proteger el acceso a los datos y servicios de Microsoft 365 es fundamental para protegerse contra ataques cibernéticos y protegerse contra la pérdida de datos.
