# <a name="introduction-to-microsoft-whiteboard"></a>Introducción a Microsoft Whiteboard

Microsoft Whiteboard en Microsoft 365 es un lienzo digital de forma libre donde las personas, el contenido y las ideas se unen. Whiteboard permite a los miembros del equipo colaborar en tiempo real, dondequiera que se encuentre. También ofrece a tus ideas espacio para crecer con un lienzo infinito diseñado para lápiz, tacto y teclado.
