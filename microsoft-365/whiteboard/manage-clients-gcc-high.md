# <a name="manage-clients-for-microsoft-whiteboard-in-gcc-high-environments"></a>Administración de clientes para Microsoft Whiteboard en entornos GCC high

>[!NOTE]
> Esta guía se aplica a los entornos de alta Government Community Cloud (GCC) de EE. UU.

Los clientes de pizarra se están actualizando actualmente para admitir One Drive for Business.

## <a name="clients-supported"></a>Clientes admitidos

Actualmente se admiten los siguientes clientes en Whiteboard:

- Aplicación web de pizarra independiente en [https://whiteboard.office365.us](https://whiteboard.office365.us)
- Microsoft Teams reuniones, chats y canales mediante Teams escritorio y web
- Aplicación de pizarra independiente para dispositivos móviles

## <a name="clients-planned"></a>Clientes planeados

Los siguientes clientes están previstos para futuras versiones de Whiteboard:

- Aplicación de pizarra independiente para Windows 10 o versiones posteriores
- Aplicación de pizarra independiente para Surface Hub (actualmente se puede usar en modo anónimo)
- Pizarra en el iniciador de aplicaciones de Office.com
- Pizarra en Teams reuniones en Surface Hub y Teams salas de reuniones
- Pizarra en llamadas 1:1 en Teams

>[!NOTE]
> Aunque los usuarios pueden instalar el cliente Windows, no podrán iniciar sesión hasta que se actualice el cliente.
