# <a name="introduction-to-sharepoint-syntex"></a>Introducción a SharePoint Syntex


</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4CSu7] 

</br>

SharePoint Syntex es un servicio de Office 365 que ayuda a las organizaciones a:

- Usar la enseñanza avanzada de AI y mecanizado para amplificar las experiencias humanas
- Automatizar el procesamiento de contenido
- Transformación del contenido en el conocimiento

Los recursos de esta página están diseñados para ayudarle a empezar a conocer y usar SharePoint Syntex en su organización.
