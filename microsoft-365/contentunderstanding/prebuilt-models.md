# <a name="create-and-configure-a-prebuilt-model-in-microsoft-sharepoint-syntex"></a>Crear y configurar un modelo precompilado en Microsoft SharePoint Syntex

> [!NOTE]
> El contenido de esta característica está actualmente en desarrollo.

Los modelos precompilados están preentrenados para reconocer documentos y los componentes de los documentos. También funcionan en archivos de imagen y archivos PDF. Puede aplicar inmediatamente modelos precompilados a las bibliotecas tal como están. No necesitan formación ni etiquetado adicionales y no se necesita ningún archivo de aprendizaje.

También puede personalizar un modelo precompilado, revisarlo para ver cómo funciona en los documentos y personalizar los nombres para que coincidan con los documentos o la convención de nomenclatura. También puede desactivar los componentes sugeridos que no son aplicables.
