# <a name="learn-about-document-understanding-models-through-a-sample-model"></a>Obtenga información sobre cómo documentar los modelos a través de un modelo de ejemplo

Microsoft SharePoint Syntex proporciona un modelo de ejemplo que puede usar para examinar, lo que le ofrece una mejor comprensión sobre cómo crear sus propios modelos. El modelo de ejemplo también le permite examinar los componentes del modelo, como su clasificador, extractores y explicaciones. También puede usar los archivos de ejemplo para entrenar el modelo.

## <a name="import-the-sample-model"></a>Importación del modelo de ejemplo

Para obtener acceso al modelo de ejemplo, primero debe importar el modelo al centro de contenido.

1. En el centro de contenido, seleccione **modelos** para ver la lista de modelos.</br>
2. En la página **modelos** , seleccione **importar modelo de ejemplo**.</br>

    ![Modelo de ejemplo de importación](../media/content-understanding/import-sample-model.png) </br>

3. Busque el modelo de ejemplo titulado *BenefitsChangeNotice. Classifier* que aparece en la lista de modelos.</br>

    ![Modelo de ejemplo](../media/content-understanding/sample-model.png) </br>

4. Seleccione *BenefitsChangeNotice. Classifier*para abrir la Página principal del modelo.</br>
  
     ![Página principal de ejemplo](../media/content-understanding/sample-home-page.png)

5. En la Página principal del modelo, examine el modelo más detenidamente para ver cómo se ha creado.
 
- Vea los archivos de ejemplo que se usan para entrenar el modelo.
- Examine los extractores de entidad del modelo más detenidamente para ver cómo el modelo de ejemplo ha configurado las explicaciones.

   ![Extractores de modelo de ejemplo](../media/content-understanding/entity-extractors.png)
