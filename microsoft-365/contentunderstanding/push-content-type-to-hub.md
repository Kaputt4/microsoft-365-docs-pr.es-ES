# <a name="push-content-types-to-a-hub"></a><span data-ttu-id="c12bb-103">Insertar tipos de contenido en un concentrador</span><span class="sxs-lookup"><span data-stu-id="c12bb-103">Push content types to a hub</span></span>

<span data-ttu-id="c12bb-104">Para que los tipos de contenido importantes estén disponibles de forma más coherente en las listas y bibliotecas de SharePoint, puede insertarlos en los concentradores que elija.</span><span class="sxs-lookup"><span data-stu-id="c12bb-104">To make important content types more consistently available to SharePoint libraries and lists, you can push them to the hubs that you choose.</span></span> <span data-ttu-id="c12bb-105">Esto los agrega automáticamente a todas las listas y bibliotecas nuevas que se creen en los sitios asociados con el concentrador y a los nuevos sitios que se agreguen al concentrador.</span><span class="sxs-lookup"><span data-stu-id="c12bb-105">This automatically adds them to any new lists and libraries created on the sites associated with the hub, and to any new sites added to the hub.</span></span>

<span data-ttu-id="c12bb-106">Para que esta característica funcione, los tipos de contenido que se insertan deben estar ya publicados.</span><span class="sxs-lookup"><span data-stu-id="c12bb-106">For this feature to work, The content types being pushed must already be published.</span></span>

<span data-ttu-id="c12bb-107">Para insertar tipos de contenido en los concentradores</span><span class="sxs-lookup"><span data-stu-id="c12bb-107">To push content types to hubs</span></span>

1. <span data-ttu-id="c12bb-108">En el centro de administración de SharePoint, expanda **servicios de contenido**y, a continuación, haga clic en Galería de **tipos de contenido**.</span><span class="sxs-lookup"><span data-stu-id="c12bb-108">In the SharePoint admin center, expand **Content services**, and then click **Content type gallery**.</span></span>

2. <span data-ttu-id="c12bb-109">Haga clic en el tipo de contenido que desee insertar en los concentradores.</span><span class="sxs-lookup"><span data-stu-id="c12bb-109">Click the content type that you want to push to hubs.</span></span>

3. <span data-ttu-id="c12bb-110">Haga clic en **Editar** en la barra de comandos.</span><span class="sxs-lookup"><span data-stu-id="c12bb-110">Click **Edit** in the command bar.</span></span>
 
4. <span data-ttu-id="c12bb-111">Haga clic en **elegir sitios concentradores**.</span><span class="sxs-lookup"><span data-stu-id="c12bb-111">Click **Choose hub sites**.</span></span>
 
5. <span data-ttu-id="c12bb-112">Seleccione los sitios concentradores que desee y, a continuación, haga clic en **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="c12bb-112">Select the desired hub sites and then click **OK**.</span></span>
 
6. <span data-ttu-id="c12bb-113">Haga clic en \*\*Guardar \*\*.</span><span class="sxs-lookup"><span data-stu-id="c12bb-113">Click **Save**.</span></span>

<span data-ttu-id="c12bb-114">Cuando se inserta un tipo de contenido en un concentrador existente & sus sitios asociados existentes por primera vez, puede tardar hasta una hora desde que se visita el concentrador o los sitios asociados, para que la configuración se actualice en el sitio.</span><span class="sxs-lookup"><span data-stu-id="c12bb-114">When pushing a content type to an existing hub & its existing associated sites for the first time, it can take up to an hour from when the hub or associated sites are visited, for the settings to update in the site.</span></span> <span data-ttu-id="c12bb-115">Las nuevas asociaciones al concentrador no requerirán esta espera y la configuración se reflejará en unos minutos.</span><span class="sxs-lookup"><span data-stu-id="c12bb-115">Any new associations to the hub will not require this wait and will have the settings reflected in a few minutes.</span></span> 

<span data-ttu-id="c12bb-116">Una vez configurado, el tipo de contenido con esta configuración estará disponible en cualquier sitio recién asociado con el concentrador en unos minutos.</span><span class="sxs-lookup"><span data-stu-id="c12bb-116">Once this is configured, the content type with these settings will be available in any newly associated site with the hub in a few minutes.</span></span> <span data-ttu-id="c12bb-117">Una vez disponible, cualquier lista o biblioteca nueva creada tendrá el tipo de contenido agregado automáticamente en unos minutos de creación.</span><span class="sxs-lookup"><span data-stu-id="c12bb-117">Once available, any new list or library created will have the content type automatically added to it within a few minutes of creation.</span></span> <span data-ttu-id="c12bb-118">Un tipo de contenido insertado se agregará a una biblioteca de documentos solo si se deriva directa o indirectamente del tipo de contenido de documento y un tipo de contenido se agregará a una lista sólo si no se deriva del tipo de contenido del documento, ya sea directa o indirectamente.</span><span class="sxs-lookup"><span data-stu-id="c12bb-118">A pushed content type will be added to a document library only if it derives directly or indirectly from the Document content type, and a content type will be added to a list only if it does not derive from the Document content type directly or indirectly.</span></span>
