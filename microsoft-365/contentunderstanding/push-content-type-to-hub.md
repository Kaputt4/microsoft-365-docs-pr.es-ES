<span data-ttu-id="f0472-104">Para que los tipos de contenido importantes estén disponibles de forma más coherente en las listas y bibliotecas de SharePoint, puede insertarlos en los centros que elija.</span><span class="sxs-lookup"><span data-stu-id="f0472-104">To make important content types more consistently available to SharePoint libraries and lists, you can push them to the hubs that you choose.</span></span> <span data-ttu-id="f0472-105">Esto los agregará automáticamente a las nuevas listas y bibliotecas creadas en los sitios asociados con el centro, y a los nuevos sitios que se agreguen al centro.</span><span class="sxs-lookup"><span data-stu-id="f0472-105">This automatically adds them to any new lists and libraries created on the sites associated with the hub, and to any new sites added to the hub.</span></span>

<span data-ttu-id="f0472-106">Para que esta característica funcione, los tipos de contenido que se insertan ya deben haber sido publicados.</span><span class="sxs-lookup"><span data-stu-id="f0472-106">For this feature to work, The content types being pushed must already be published.</span></span>

<span data-ttu-id="f0472-107">Insertar tipos de contenido en un centro</span><span class="sxs-lookup"><span data-stu-id="f0472-107">To push content types to hubs</span></span>

1. <span data-ttu-id="f0472-108">En el Centro de administración de SharePoint, expanda **Servicios de contenido** y, después, haga clic en **Galería de tipo de contenido**.</span><span class="sxs-lookup"><span data-stu-id="f0472-108">In the SharePoint admin center, expand **Content services**, and then click **Content type gallery**.</span></span>

2. <span data-ttu-id="f0472-109">Haga clic en el tipo de contenido que desea insertar en el centro.</span><span class="sxs-lookup"><span data-stu-id="f0472-109">Click the content type that you want to push to hubs.</span></span>

3. <span data-ttu-id="f0472-110">Haga clic en **Editar** en la barra de comandos.</span><span class="sxs-lookup"><span data-stu-id="f0472-110">Click **Edit** in the command bar.</span></span>
 
4. <span data-ttu-id="f0472-111">Haga clic en **Elegir sitios centrales**.</span><span class="sxs-lookup"><span data-stu-id="f0472-111">Click **Choose hub sites**.</span></span>
 
5. <span data-ttu-id="f0472-112">Seleccione los sitios centrales deseados y, a continuación, haga clic en **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="f0472-112">Select the desired hub sites and then click **OK**.</span></span>
 
6. <span data-ttu-id="f0472-113">Haga clic en **Guardar**.</span><span class="sxs-lookup"><span data-stu-id="f0472-113">Click **Save**.</span></span>

<span data-ttu-id="f0472-114">Cuando se inserta un tipo de contenido en un centro existente y sus sitios asociados existentes por primera vez, puede tardar hasta una hora desde la visita al centro o a los sitios asociados para que la configuración se actualice en el sitio.</span><span class="sxs-lookup"><span data-stu-id="f0472-114">When pushing a content type to an existing hub & its existing associated sites for the first time, it can take up to an hour from when the hub or associated sites are visited, for the settings to update in the site.</span></span> <span data-ttu-id="f0472-115">Ninguna nueva asociación al centro requerirá esta espera y todas reflejarán la configuración en unos cuantos minutos.</span><span class="sxs-lookup"><span data-stu-id="f0472-115">Any new associations to the hub will not require this wait and will have the settings reflected in a few minutes.</span></span> 

<span data-ttu-id="f0472-116">Una vez configurado, el tipo de contenido con estas opciones de configuración estará disponible en cualquier sitio recientemente asociado con el centro en cuestión de minutos.</span><span class="sxs-lookup"><span data-stu-id="f0472-116">Once this is configured, the content type with these settings will be available in any newly associated site with the hub in a few minutes.</span></span> <span data-ttu-id="f0472-117">Cuando esté disponible, el tipo de contenido se agregará automáticamente a las nuevas listas o bibliotecas creadas luego de unos minutos después de la creación.</span><span class="sxs-lookup"><span data-stu-id="f0472-117">Once available, any new list or library created will have the content type automatically added to it within a few minutes of creation.</span></span> <span data-ttu-id="f0472-118">Un tipo de contenido insertado se agregará a una biblioteca de documentos solo si se deriva directa o indirectamente del tipo de contenido de documento, y un tipo de contenido se agregará a una lista solo si no deriva de forma directa o indirecta del tipo de contenido del documento.</span><span class="sxs-lookup"><span data-stu-id="f0472-118">A pushed content type will be added to a document library only if it derives directly or indirectly from the Document content type, and a content type will be added to a list only if it does not derive from the Document content type directly or indirectly.</span></span>
