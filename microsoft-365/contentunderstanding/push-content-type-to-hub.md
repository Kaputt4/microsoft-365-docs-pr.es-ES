<span data-ttu-id="8886f-104">Para que los tipos de contenido importantes estén disponibles de forma más coherente en las listas y bibliotecas de SharePoint, puede insertarlos en los centros que elija.</span><span class="sxs-lookup"><span data-stu-id="8886f-104">To make important content types more consistently available to SharePoint libraries and lists, you can push them to the hubs that you choose.</span></span> <span data-ttu-id="8886f-105">Al insertar los tipos de contenido se agregarán automáticamente a las nuevas listas y bibliotecas creadas en los sitios asociados con el centro, y a los nuevos sitios que se agreguen al centro.</span><span class="sxs-lookup"><span data-stu-id="8886f-105">Pushing the content types automatically adds them to any new lists and libraries created on the sites associated with the hub, and to any new sites added to the hub.</span></span>

<span data-ttu-id="8886f-106">Para que esta característica funcione, los tipos de contenido que se insertan ya deben haber sido publicados.</span><span class="sxs-lookup"><span data-stu-id="8886f-106">For this feature to work, the content types being pushed must already be published.</span></span>

<span data-ttu-id="8886f-107">Insertar tipos de contenido en un centro</span><span class="sxs-lookup"><span data-stu-id="8886f-107">To push content types to hubs</span></span>

1. <span data-ttu-id="8886f-108">En el Centro de administración de SharePoint, expanda **Servicios de contenido** y, después, seleccione **Galería de tipo de contenido**.</span><span class="sxs-lookup"><span data-stu-id="8886f-108">In the SharePoint admin center, expand **Content services**, and then select **Content type gallery**.</span></span>
2. <span data-ttu-id="8886f-109">Seleccione el tipo de contenido que quiere insertar en el centro.</span><span class="sxs-lookup"><span data-stu-id="8886f-109">Select the content type that you want to push to hubs.</span></span>
3. <span data-ttu-id="8886f-110">Seleccione **Editar** en la barra de comandos.</span><span class="sxs-lookup"><span data-stu-id="8886f-110">Select **Edit** in the command bar.</span></span>
4. <span data-ttu-id="8886f-111">Seleccione **Elegir sitios del sitio central**.</span><span class="sxs-lookup"><span data-stu-id="8886f-111">Select **Choose hub sites**.</span></span>
5. <span data-ttu-id="8886f-112">Seleccione los sitios del sitio central que quiera y después **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="8886f-112">Select the hub sites you want and then choose **OK**.</span></span>
6. <span data-ttu-id="8886f-113">Seleccione **Guardar**.</span><span class="sxs-lookup"><span data-stu-id="8886f-113">Choose **Save**.</span></span>

<span data-ttu-id="8886f-114">Cuando inserta un tipo de contenido en un centro existente y sus sitios asociados existentes por primera vez, puede tardar hasta una hora desde la visita al centro o a los sitios asociados para que la configuración se actualice en el sitio.</span><span class="sxs-lookup"><span data-stu-id="8886f-114">When you push a content type to an existing hub & its existing associated sites for the first time, it can take up to an hour from when the hub or associated sites are visited, for the settings to update in the site.</span></span> <span data-ttu-id="8886f-115">Ninguna nueva asociación al centro requerirá esta espera y todas reflejarán la configuración en unos cuantos minutos.</span><span class="sxs-lookup"><span data-stu-id="8886f-115">Any new associations to the hub won't require this wait and will have the settings reflected in a few minutes.</span></span>

<span data-ttu-id="8886f-116">Una vez actualizada la configuración, el tipo de contenido con estas opciones de configuración estará disponible en cualquier sitio recientemente asociado con el centro en cuestión de minutos.</span><span class="sxs-lookup"><span data-stu-id="8886f-116">After the settings are updated, the content type with these settings will be available in any newly associated site with the hub in a few minutes.</span></span> <span data-ttu-id="8886f-117">Después, el tipo de contenido se agregará automáticamente a las nuevas listas o bibliotecas creadas unos minutos después de la creación.</span><span class="sxs-lookup"><span data-stu-id="8886f-117">Then, any new list or library created will have the content type automatically added to it within a few minutes of creation.</span></span> <span data-ttu-id="8886f-118">Un tipo de contenido insertado se agregará a una biblioteca de documentos solo si se deriva directa o indirectamente del tipo de contenido de documento, y un tipo de contenido se agregará a una lista solo si no deriva de forma directa o indirecta del tipo de contenido del documento.</span><span class="sxs-lookup"><span data-stu-id="8886f-118">A pushed content type will be added to a document library only if it derives directly or indirectly from the Document content type, and a content type will be added to a list only if it does not derive from the Document content type directly or indirectly.</span></span>
