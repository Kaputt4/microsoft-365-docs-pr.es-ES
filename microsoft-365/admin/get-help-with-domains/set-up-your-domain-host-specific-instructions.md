# <a name="set-up-your-domain-host-specific-instructions"></a>Configurar su dominio (instrucciones específicas del host)

Para empezar a usar un dominio personalizado (contoso.com) con Microsoft 365, debe comprobar el dominio y configurar los registros DNS del dominio. 
  
Puede agregar y administrar registros DNS con las herramientas administrativas del host de dominio o dar a Microsoft el control de los registros de dominio y los configuraremos por usted.
  
Seleccione el host de dominio que aparece a continuación para obtener los pasos exactos. Si no está seguro de quién es su host, consulte [Find your domain registrar](find-your-domain-registrar.md).
