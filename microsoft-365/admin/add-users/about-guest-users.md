# <a name="guest-users-in-microsoft-365-admin-center"></a>Usuarios invitados en el centro de administración de Microsoft 365

Cuando se agregan invitados a Microsoft Teams o a SharePoint, estos usuarios se agregan a la lista de usuarios invitados en el centro de administración. También puede Agregar usuarios invitados a Azure Active Directory para que puedan colaborar con los usuarios de la organización. Los usuarios invitados pueden asistir a reuniones, ver los documentos y chatear en los equipos a los que están invitados.
Una vez que se agrega un usuario a la lista de usuarios invitados, puede quitar el acceso a él.

Para ver los usuarios invitados, en el centro de administración de Microsoft 365, en el panel de navegación de la izquierda, expanda **usuarios**y, a continuación, elija **usuarios invitados**.

## <a name="add-guest-users-in-azure-active-directory"></a>Agregar usuarios invitados en Azure Active Directory

Para agregar usuarios invitados a Azure Active Directory, consulte [Add guestd users](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal).

Al agregar un usuario, también puede asignarlos a un grupo o darles acceso a una aplicación de su organización. Una vez que haya agregado un usuario en el portal de Azure AD, dicho usuario también aparecerá en la página usuarios invitados del centro de administración de Microsoft 365.
Una vez que un usuario se agrega a la lista de usuarios invitados, se puede [Agregar a los grupos](../create-groups/manage-guest-access-in-groups.md#add-guests-to-a-microsoft-365-group-from-the-admin-center) del centro de administración de Microsoft 365.

Consulte [Agregar usuarios invitados de forma masiva](https://docs.microsoft.com/azure/active-directory/b2b/tutorial-bulk-invite) para invitar a varios invitados para colaborar con su organización.


## <a name="remove-a-guest-user"></a>Quitar un usuario Guest

Para quitar un usuario Guest:

1. En el centro de administración de Microsoft 365, expanda **usuarios** y, a continuación, elija **usuarios invitados**.
1. En la página **usuarios invitados** , elija el usuario que desea quitar y, a continuación, elija **eliminar un usuario**. 

Para quitar usuarios en el portal de Azure AD, vea [quitar un usuario y recursos de invitado](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal#clean-up-resources).
