# <a name="guest-users-in-microsoft-365-admin-center"></a>Usuarios invitados en Microsoft 365 de administración

Los invitados que agregue a su Microsoft Teams, SharePoint o Azure Active Directory también se  agregarán a la lista de usuarios invitados en el Centro de administración de Microsoft. Los invitados pueden asistir a reuniones, ver documentos y chatear en Teams a la que están invitados.
Una vez que un usuario aparece en la **lista De usuarios invitados,** puede quitar su acceso allí.

Para ver usuarios invitados, en el centro de administración Microsoft 365, en la navegación izquierda, expanda **Usuarios** y, a continuación, elija **Usuarios invitados.**

## <a name="before-you-begin"></a>Antes de empezar

Debe ser un administrador global para realizar esta tarea.

## <a name="steps-add-guests-in-azure-active-directory"></a>Pasos: Agregar invitados en Azure Active Directory

Para agregar invitados en el Azure Active Directory, vea [Agregar usuarios invitados](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal).

Después de agregar un usuario, también puedes asignarlos a un grupo o darles acceso a una aplicación de la organización. Una vez que haya agregado un usuario en el Portal de  Azure AD, ese usuario también aparecerá en la página Usuarios invitados en el Centro de administración de Microsoft 365 usuario.
Después de agregar un usuario a la lista de usuarios **invitados,** se pueden [agregar](../create-groups/manage-guest-access-in-groups.md#add-guests-to-a-microsoft-365-group-from-the-admin-center) a grupos en el centro de administración Microsoft 365 invitado.

Consulta [Agregar invitados en masa](https://docs.microsoft.com/azure/active-directory/b2b/tutorial-bulk-invite) para invitar a varios invitados a colaborar con la organización.

## <a name="next-steps-remove-a-guest"></a>Pasos siguientes: Quitar un invitado

Una vez que haya terminado de colaborar con un usuario invitado, puede quitarlos y ya no tendrán acceso a su organización.

1. En el centro Microsoft 365 administración, expanda **Usuarios** y, a continuación, elija **Usuarios invitados.**
1. En la **página Usuarios invitados,** elija el usuario que desea quitar y, a continuación, **elija Eliminar un usuario**.

Para quitar usuarios en El portal de Azure AD, vea [quitar un usuario invitado y recursos](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal#clean-up-resources).
