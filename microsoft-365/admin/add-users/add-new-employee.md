# <a name="add-a-new-employee-to-microsoft-365"></a>Agregar un nuevo empleado a Microsoft 365

Este artículo le ayuda a incorporar un nuevo empleado a Microsoft 365 para empresas. Damos por hecho que es un administrador y que ya ha completado la configuración de [Microsoft 365](../setup/setup.md)y ahora tiene a alguien nuevo que se une a su empresa.
  
Está en el lugar adecuado si su nuevo empleado necesita Microsoft 365 y está usando un plan de [Microsoft 365](https://products.office.com/business/compare-office-365-for-business-plans) que le permite instalar aplicaciones de Office como Word y Excel en un equipo. 
  
 **¿No es un administrador?** [Obtenga información sobre Microsoft 365](https://support.microsoft.com/office/396b8d9e-e118-42d0-8a0d-87d1f2f055fb) para ayudar a los usuarios empresariales y de su hogar a configurarse. 
  
 **¿No hay aplicaciones de Office en su plan?** Sigue los pasos siguientes, pero omite las secciones para instalar aplicaciones. En su [lugar, use las versiones en línea de Office.](https://support.microsoft.com/office/91a4ec74-67fe-4a84-a268-f6bdf3da1804) 
  
A continuación se ofrece una introducción rápida: 
  
|**Paso**|**¿Por qué hacerlo?**|
|:-----|:-----|
|[Paso 1: Crear una cuenta de Microsoft 365 para el empleado](#step-1-create-a-microsoft-365-account-for-the-employee) <br/> |Cada vez que un nuevo empleado se una a su empresa, cree una cuenta para ellos para que pueda empezar a usar Microsoft 365.  <br/> |
|[Paso 2: Proporcionar al empleado su id. de usuario y contraseña](#step-2-give-the-employee-their-user-id-and-password) <br/> |Al crear una cuenta, obtienes un identificador y una contraseña que puedes pasar a tu empleado para que puedan iniciar sesión.  <br/> |
|[Paso 3: Explicar dónde iniciar sesión](#step-3-explain-where-to-sign-in) <br/> |La ubicación de inicio de sesión es [https://www.office.com](https://www.office.com) <br/> |
|[Paso 4: Ayudar a los empleados a empezar](#step-4-help-your-employee-get-started) <br/> |Permita a sus empleados saber cómo usar OneDrive o cualquier sitio de grupo de su organización.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Paso 1: Crear una cuenta de Microsoft 365 para el empleado


Para obtener instrucciones, [vea Agregar usuarios y asignar licencias al mismo tiempo.](add-users.md) Al configurar el nuevo empleado, puede enviar los detalles de inicio de sesión a la cuenta personal del empleado. De esta forma, recibirán un correo electrónico del equipo de Microsoft Online Service que les indica cómo iniciar sesión en Microsoft 365.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Paso 2: Proporcionar al empleado su id. de usuario y contraseña


A menos que lo envíe a su dirección de correo electrónico personal, imprima el nombre de inicio de sesión y la contraseña del empleado y pásenlo a ellos. Or tell them the information over the phone.
  
Como aún no tendrán acceso a su correo electrónico, no envíe la información a esa dirección de correo electrónico.
  
## <a name="step-3-explain-where-to-sign-in"></a>Paso 3: Explicar dónde iniciar sesión 


Al igual que Facebook, Amazon o Gmail, su empleado inicia sesión para usar Microsoft 365. Conséles la siguiente información de inicio de sesión:
  
- Inicie sesión en [https://www.office.com](https://www.office.com) .
    
- Seleccione **Iniciar sesión y,** a continuación, escriba el identificador de usuario y la contraseña.
