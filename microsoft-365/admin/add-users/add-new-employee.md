# <a name="add-a-new-employee-to-microsoft-365"></a>Agregar un nuevo empleado a Microsoft 365

Este artículo le ayuda a incorporar un nuevo empleado a Microsoft 365 para empresas. Suponemos que es un administrador y que ya ha completado la configuración de [Microsoft 365](../setup/setup.md)y ahora tiene a alguien nuevo uniéndose a su empresa.
  
Está en el lugar correcto si su nuevo empleado necesita Microsoft 365 y está usando un plan de [Microsoft 365](https://products.office.com/business/compare-office-365-for-business-plans) que le permite instalar aplicaciones de Office como Word y Excel en un equipo. 
  
 **¿No es un administrador?** [Obtenga información acerca de Microsoft 365](https://support.microsoft.com/office/396b8d9e-e118-42d0-8a0d-87d1f2f055fb) y ayude a los usuarios empresariales y de su hogar a configurarse. 
  
 **¿No hay aplicaciones de Office en el plan?** Siga los pasos siguientes, pero omita las secciones para instalar aplicaciones. En su [lugar, use las versiones en](https://support.microsoft.com/office/91a4ec74-67fe-4a84-a268-f6bdf3da1804) línea de Office. 
  
A continuación se ofrece una introducción rápida: 
  
|**Paso**|**¿Por qué hacerlo?**|
|:-----|:-----|
|[Paso 1: Crear una cuenta de Microsoft 365 para el empleado](#step-1-create-a-microsoft-365-account-for-the-employee) <br/> |Cada vez que un nuevo empleado se una a su empresa, cree una cuenta para ellos para que pueda empezar a usar Microsoft 365.  <br/> |
|[Paso 2: Proporcionar al empleado su id. de usuario y contraseña](#step-2-give-the-employee-their-user-id-and-password) <br/> |Al crear una cuenta, obtienes un identificador y una contraseña que puedes pasar a tu empleado para que puedan iniciar sesión.  <br/> |
|[Paso 3: Explicar dónde iniciar sesión](#step-3-explain-where-to-sign-in) <br/> |La ubicación de inicio de sesión es [https://www.office.com](https://www.office.com) <br/> |
|[Paso 4: Ayudar a su empleado a empezar](#step-4-help-your-employee-get-started) <br/> |Permite a los empleados saber cómo usar OneDrive o cualquier sitio de grupo de la organización.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Paso 1: Crear una cuenta de Microsoft 365 para el empleado


Para obtener instrucciones, vea [Agregar usuarios y asignar licencias al mismo tiempo.](add-users.md) Al configurar el nuevo empleado, puede elegir enviar detalles de inicio de sesión a la cuenta personal del empleado. De esta forma, recibirán un correo electrónico del equipo de servicio de Microsoft Online que les indica cómo iniciar sesión en Microsoft 365.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Paso 2: Proporcionar al empleado su id. de usuario y contraseña


A menos que lo envíe a su dirección de correo electrónico personal, imprima el nombre y la contraseña de inicio de sesión del empleado y pásenlo a ellos. O bien, díles la información por teléfono.
  
Dado que aún no tendrán acceso a su correo electrónico, no envíe la información a esa dirección de correo electrónico.
  
## <a name="step-3-explain-where-to-sign-in"></a>Paso 3: Explicar dónde iniciar sesión 


Al igual que Facebook, Amazon o Gmail, el empleado inicia sesión para usar Microsoft 365. Déles la siguiente información de inicio de sesión:
  
- Inicie sesión en [https://www.office.com](https://www.office.com) .
    
- Seleccione **Iniciar sesión** y, a continuación, escriba el id. de usuario y la contraseña.
