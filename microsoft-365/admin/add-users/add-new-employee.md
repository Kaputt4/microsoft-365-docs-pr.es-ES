# <a name="add-a-new-employee-to-microsoft-365"></a>Agregar un nuevo empleado a Microsoft 365

Este artículo le ayuda a incorporar un nuevo empleado para Microsoft 365 para empresas. Suponemos que es administrador y que ya [ha completado Microsoft 365 configuración](../setup/setup.md) y ahora tiene a alguien nuevo que se une a su empresa.
  
Está en el lugar correcto si el nuevo empleado necesita Microsoft 365 y usa un [plan de Microsoft 365](https://products.office.com/business/compare-office-365-for-business-plans) que le permite instalar aplicaciones Office como Word y Excel en un equipo. 
  
 **¿No es un administrador?** [Obtenga información sobre Microsoft 365](https://support.microsoft.com/office/396b8d9e-e118-42d0-8a0d-87d1f2f055fb) ayuda a los usuarios empresariales y domésticos con la configuración. 
  
 **¿No hay aplicaciones Office en el plan?** Siga los pasos siguientes, pero omita las secciones para instalar aplicaciones. En su lugar, use las [versiones en línea de Office](https://support.microsoft.com/office/91a4ec74-67fe-4a84-a268-f6bdf3da1804). 
  
Esta es una introducción rápida: 
  
|**Paso**|**¿Por qué hacer esto?**|
|:-----|:-----|
|[Paso 1: Crear una cuenta de Microsoft 365 para el empleado](#step-1-create-a-microsoft-365-account-for-the-employee) <br/> |Cada vez que un nuevo empleado se una a su empresa, cree una cuenta para ellos para que pueda empezar a usar Microsoft 365.  <br/> |
|[Paso 2: Dar al empleado su id. de usuario y contraseña](#step-2-give-the-employee-their-user-id-and-password) <br/> |Al crear una cuenta, obtendrá un identificador y una contraseña que puede pasar a su empleado para que pueda iniciar sesión.  <br/> |
|[Paso 3: Explicar dónde iniciar sesión](#step-3-explain-where-to-sign-in) <br/> |La ubicación de inicio de sesión es [https://www.office.com](https://www.office.com) <br/> |
|[Paso 4: Ayudar a su empleado a empezar](#step-4-help-your-employee-get-started) <br/> |Deje que su empleado sepa cómo usar OneDrive o cualquier sitio de equipo de su organización.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Paso 1: Crear una cuenta de Microsoft 365 para el empleado


Para obtener instrucciones, consulte [Agregar usuarios y asignar licencias al mismo tiempo](add-users.md). Al configurar el nuevo empleado, puede elegir enviar los detalles de inicio de sesión a la cuenta personal del empleado. De este modo, recibirán un correo electrónico del equipo de servicio en línea de Microsoft que les indica cómo iniciar sesión en Microsoft 365.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Paso 2: Dar al empleado su id. de usuario y contraseña


A menos que lo envíe a su dirección de correo electrónico personal, imprima el nombre y la contraseña de inicio de sesión del empleado y páselo. O diles la información por teléfono.
  
Dado que aún no tendrán acceso a su correo electrónico, no envíe la información a esa dirección de correo electrónico.
  
## <a name="step-3-explain-where-to-sign-in"></a>Paso 3: Explicar dónde iniciar sesión 


Al igual que Facebook, Amazon o Gmail, el empleado inicia sesión para usar Microsoft 365. Proporcione la siguiente información de inicio de sesión:
  
- Inicie sesión en [https://www.office.com](https://www.office.com).
    
- Seleccione **Iniciar sesión** y escriba el identificador de usuario y la contraseña.
    
## <a name="step-4-help-your-employee-get-started"></a>Paso 4: Ayudar a su empleado a empezar
