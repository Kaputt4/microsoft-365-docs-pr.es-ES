# <a name="office-365-is-now-microsoft-365"></a>Office 365 ahora es Microsoft 365

Cambiamos algunos de nuestros nombres de suscripción de Office 365 y Microsoft 365. No necesita hacer nada. Sus servicios, aplicaciones y características serán los mismos, al igual que el precio de su suscripción. El nombre de la suscripción se actualizará automáticamente en el <a href="https://go.microsoft.com/fwlink/p/?linkid=2166757" target="_blank">Centro de administración de Microsoft 365</a> y las instrucciones de facturación se actualizarán el 21 de abril de 2020 o después.

- **Office 365 Empresa Essentials** ahora está **Microsoft 365 Empresa Básico**
- **Office 365 Business Premium** está **ahora Microsoft 365 Empresa Estándar**
- **Microsoft 365 Empresa** está **ahora Microsoft 365 Empresa Premium**
- **Office 365 Empresa** está **ahora Aplicaciones Microsoft 365 para negocios**
- **Office 365 ProPlus** ahora está **Aplicaciones Microsoft 365 para empresas**

[¡Aprende más!](https://go.microsoft.com/fwlink/?linkid=2120533)
