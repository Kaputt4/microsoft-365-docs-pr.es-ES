# <a name="message-center"></a>Centro de mensajes


Para realizar un seguimiento de los próximos cambios, incluidas las características nuevas y modificadas, el mantenimiento planeado u otros anuncios importantes, vaya al <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">Centro de mensajes.</a> 
  
Para abrir el Centro de mensajes:

::: moniker range="o365-worldwide"

- En el centro de administración, vaya al **Centro de** mensajes > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">de estado.</a>

::: moniker-end

::: moniker range="o365-germany"

- En el <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">centro de administración,</a>vaya al **Centro de** mensajes > **de estado.**

::: moniker-end

::: moniker range="o365-21vianet"
 
- En el <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">centro de administración,</a>vaya al **Centro de** mensajes > **de estado.**

::: moniker-end

También puede usar la aplicación de administración de [Microsoft 365](https://go.microsoft.com/fwlink/p/?linkid=627216) en su dispositivo móvil para ver el Centro de mensajes, que es una excelente manera de mantenerse al día con las notificaciones de inserción.

### <a name="major-updates"></a>Actualizaciones principales

Las actualizaciones principales se pueden revisar seleccionando la **pestaña** Importancia alta en la parte superior del Centro de mensajes.

Las actualizaciones principales se comunican con al menos 30 días de antelación cuando se requiere una acción y pueden incluir:
  
- Cambios en la productividad diaria, como bandeja de entrada, reuniones, delegados, uso compartido y acceso

- Cambios en temas, elementos web y otros componentes que pueden afectar a las características personalizadas

- Aumenta o disminuye la capacidad visible, como el almacenamiento, el número de reglas, elementos o duraciones.

- Cambios en la personalción de marca del producto que pueden:

  - Causa confusión al usuario final,

  - Resultado en cambios en los procesos del servicio de ayuda y material de referencia, o

  - Cambiar una dirección URL

- Un nuevo servicio o aplicación

- Cambios que requieren una acción de administrador (excluyen los problemas de prevención o corrección)

- Cambios en el lugar donde se almacenan los datos
  
### <a name="preferences"></a>Preferencias

Si la administración se distribuye en toda la organización, es posible que no desee o necesite ver publicaciones sobre todos los servicios de Microsoft 365. Cada administrador puede:

- Establecer preferencias que controle qué mensajes se muestran en el Centro de mensajes.
- Filtrar mensajes
- Establecer las preferencias de correo electrónico para recibir una síntesis semanal de todos los mensajes, correos electrónicos solo para actualizaciones principales y correos electrónicos para mensajes de privacidad de datos.  

::: moniker range="o365-worldwide"

1. Seleccione **Editar preferencias** en la parte superior del centro de mensajes.

2. Asegúrese de que la casilla esté activada para cada servicio que desee supervisar. Desactive las casillas de los servicios que desea filtrar fuera de la vista Centro de mensajes.

3. Los correos electrónicos de resumen están activados de forma predeterminada y se envían a su dirección de correo electrónico principal. Para dejar de recibir el resumen semanal, desactive la casilla Enviar una síntesis **de correo electrónico semanal de mis** mensajes. <br/><br/>La notificación por correo electrónico para actualizaciones principales es un control independiente. Si desea recibir avisos por correo electrónico sobre las actualizaciones principales, compruebe que la opción **Enviarme mensajes** de correo electrónico para actualizaciones principales esté seleccionada. Desactive la casilla para dejar de recibir correo electrónico sobre las actualizaciones principales. <br><br/>Active la casilla  junto a Enviarme correos electrónicos para que los mensajes de privacidad de datos reciban notificaciones por correo electrónico independientes de mensajes de privacidad de datos (los mensajes de privacidad de datos no se incluyen en el resumen semanal). <br><br/>Puede seleccionar o borrar la dirección de correo electrónico principal, pero no puede cambiarla. Para especificar otras direcciones de correo electrónico a  las que se envía el correo electrónico del mensaje de privacidad de datos, active la casilla Otras direcciones de correo electrónico y escriba las otras direcciones de correo electrónico a las que desea que se envíe el correo electrónico. Escriba la dirección de correo electrónico de un grupo de Microsoft 365 o una lista de distribución si más de dos personas deben obtener el correo electrónico de privacidad de datos.

4. Seleccione **Guardar** para conservar los cambios.
