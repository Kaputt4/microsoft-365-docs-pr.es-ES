# <a name="testing-the-servicenow-configuration"></a>Prueba de la configuración de ServiceNow

Si la aplicación requiere una comunicación correcta con sistemas externos, describa cómo probar la conexión para garantizar una configuración correcta.

Siga estos pasos para probar la configuración de la integración de soporte técnico de Microsoft 365:

1. Inicie sesión en el portal de ServiceNow como administrador.

2. Abra cualquier incidente.

3. Céntrese en la pestaña **Soporte técnico de Microsoft 365** y seleccione **Microsoft 365 Insights** para determinar si las soluciones recomendadas se devolvieron correctamente.
