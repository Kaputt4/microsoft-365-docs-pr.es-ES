# <a name="enroll-your-mobile-device-using-basic-mobility-and-security"></a>Inscribir el dispositivo móvil con la seguridad y la movilidad básicas

Usar su teléfono, la tableta y otros dispositivos móviles para trabajar es una buena forma de mantenerse informado y trabajar en proyectos profesionales mientras está fuera de la oficina. Para poder usar los servicios de Microsoft 365 con el dispositivo, es posible que deba inscribirse primero en la movilidad básica y la seguridad de Microsoft 365 mediante el portal de empresa de Microsoft Intune.

Las organizaciones eligen la movilidad y la seguridad básicas para que los empleados puedan usar sus dispositivos móviles para acceder de forma segura a los mensajes de correo electrónico, calendarios y documentos del trabajo, mientras que la empresa protege los datos importantes y cumple sus requisitos de cumplimiento.Para obtener más información, consulte [Overview of Basic Mobility and Security for Microsoft 365](overview-of-basic-mobility-and-security-for-microsoft-365.md). Para obtener más información, consulte [¿Qué información puede ver mi organización cuando inscribo mi dispositivo?](https://docs.microsoft.com/intune-user-help/what-info-can-your-company-see-when-you-enroll-your-device-in-intune).

>[!IMPORTANT] 
>Cuando inscriba su dispositivo en Basic Mobility and Security for Microsoft 365, es posible que deba configurar una contraseña, además de permitir que la opción para su organización de trabajo borre el dispositivo. Se puede realizar un borrado de dispositivo desde el centro de administración de Microsoft 365, por ejemplo, para quitar todos los datos del dispositivo si la contraseña se ha escrito incorrectamente demasiadas veces o si se rompen los términos de uso.

## <a name="supported-devices"></a>Dispositivos admitidos

Basic Mobility and Security for Microsoft 365 hospedado por el servicio Intune funciona con la mayoría de los dispositivos móviles, pero no todos. Los siguientes elementos son compatibles con la movilidad y la seguridad básicas:

- iOS 10,0 o posterior
    
- Android 4,4 o posterior
    
- Windows 8,1 y Windows 10 (teléfono y PC)
    
Si el dispositivo no aparece en la lista anterior y necesita usarlo con movilidad y seguridad básica, póngase en contacto con el administrador de su trabajo o escuela.

>[!TIP] 
>Si tiene problemas para inscribir el dispositivo, consulte [troubleshoot Basic Mobility and Security](troubleshoot-basic-mobility-and-security.md).

## <a name="set-up-your-mobile-device-with-intune-and-basic-mobility-and-security"></a>Configurar el dispositivo móvil con Intune y movilidad y seguridad básicos

El portal de empresa de Intune permite que Microsoft 365 administre un dispositivo y la seguridad y movilidad básicos.

### <a name="iphone-or-ipad"></a>iPhone o iPad

>[!TIP]
>No podrá enviar ni recibir correo electrónico hasta que complete este paso.

Vaya a la App Store de Apple y descargue e instale el portal de empresa de Intune.

Para conectar y configurar su teléfono o tableta iOS con el portal de empresa en Office 365, consulte [configurar el acceso de dispositivos iOS para los recursos de la empresa](https://go.microsoft.com/fwlink/?linkid=875316).

### <a name="android-phone-or-tablet"></a>Tableta o teléfono Android

>[!TIP]
>No podrá enviar ni recibir correo electrónico hasta que complete este paso.

Vaya a Google Play Store y descargue e instale el portal de empresa de Intune.

Para conectar y configurar su teléfono o tableta Android con el portal de la compañía a Microsoft 365, consulte [inscribir el dispositivo con el portal de la empresa](https://go.microsoft.com/fwlink/?linkid=875317).

### <a name="windows-81-and-windows-10"></a>Windows 8,1 y Windows 10

Vaya a Microsoft Store y descargue e instale el portal de empresa de Intune.

Para conectar y configurar su Windows Phone o PC con el portal de empresa en Microsoft 365, consulte [inscripción de dispositivos de Windows en Intune Company Portal](https://docs.microsoft.com/intune-user-help/windows-enrollment-company-portal).

## <a name="whats-next"></a>¿Cuál es el siguiente paso?

Una vez inscrito el dispositivo en la movilidad y la seguridad básicas, puede empezar a usar las aplicaciones de Office en el dispositivo para trabajar con el correo electrónico, el calendario, los contactos y los documentos.
