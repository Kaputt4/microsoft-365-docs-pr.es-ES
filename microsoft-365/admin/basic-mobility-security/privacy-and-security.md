# <a name="privacy-and-security-in-basic-mobility-and-security"></a>Privacidad y seguridad en la movilidad básica y la seguridad

Basic Mobility and Security es un servicio basado en la nube con tecnología de Microsoft Intune que le ayuda a administrar y proteger los dispositivos móviles usados en su organización. Después de activar la movilidad y la seguridad básica, puede crear directivas de administración de dispositivos móviles. Estas directivas se pueden implementar en dispositivos móviles que hayan sido inscritos por usuarios con licencia de Microsoft 365 en su organización.

Microsoft Intune envía información a Microsoft 365 sobre el estado de cumplimiento de cada dispositivo administrado y, a continuación, puede generar informes que muestren si los dispositivos administrados de su organización son compatibles según las directivas establecidas. Para obtener más información sobre el compromiso de Microsoft con la privacidad y la seguridad, consulte el [centro de confianza de Microsoft](https://www.microsoft.com/trust-center).
