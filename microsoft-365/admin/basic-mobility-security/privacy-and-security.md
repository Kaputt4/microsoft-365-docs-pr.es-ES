# <a name="privacy-and-security-in-basic-mobility-and-security"></a>Privacidad y seguridad en movilidad y seguridad básicas

Basic Mobility and Security es un servicio basado en la nube con tecnología de Microsoft Intune que le ayuda a administrar y proteger dispositivos móviles en su organización. Después de activar Basic Mobility and Security, puede crear directivas de administración de dispositivos móviles. Estas directivas se pueden implementar en dispositivos móviles inscritos por usuarios con licencia de Microsoft 365 en su organización.

Microsoft Intune envía información a Microsoft 365 sobre el estado de cumplimiento de cada dispositivo administrado y, a continuación, puede generar informes que muestren si los dispositivos administrados de su organización son compatibles en función de las directivas establecidas. Para obtener más información sobre el compromiso de Microsoft con la privacidad y la seguridad, consulte el [Centro de confianza de Microsoft](https://www.microsoft.com/trust-center).
