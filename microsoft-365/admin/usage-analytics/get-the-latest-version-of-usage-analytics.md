# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Obtener la versión más reciente del análisis de uso de Microsoft 365

La aplicación de plantilla se puede actualizar con nuevos datos o visualizaciones nuevas varias veces al año. La instancia existente seguirá funcionando, pero si desea obtener la versión más reciente, debe crearse una nueva instancia y se deben aplicar las personalizaciones a la nueva instancia. Vea Habilitar análisis de uso de [Microsoft 365.](enable-usage-analytics.md)
