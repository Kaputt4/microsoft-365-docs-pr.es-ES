# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Obtener la versión más reciente de Microsoft 365 análisis de uso

La aplicación de plantilla puede actualizarse con nuevos datos o nuevas visualizaciones varias veces al año. La instancia existente seguirá funcionando, pero si desea obtener la versión más reciente, se debe crear una nueva instancia y se deben aplicar las personalizaciones a la nueva instancia. Consulte [Enable Microsoft 365 usage analytics](enable-usage-analytics.md).
