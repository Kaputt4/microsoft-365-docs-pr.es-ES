# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Obtener la versión más reciente del análisis de uso de Microsoft 365

La aplicación de plantilla puede actualizarse con nuevos datos o nuevas visualizaciones varias veces al año. La instancia existente seguirá funcionando, pero si desea obtener la versión más reciente, se debe crear una nueva instancia y se debe aplicar cualquier personalización a la nueva instancia. Consulte [Habilitación del análisis de uso de Microsoft 365](enable-usage-analytics.md).
