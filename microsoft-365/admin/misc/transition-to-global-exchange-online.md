# <a name="update-your-mx-records-to-transition-to-the-global-exchange-online-service"></a>Actualizar los registros MX para realizar la transición al servicio de Exchange Online global

1. Inicie sesión en Centro de administración de Microsoft 365 y vaya a **Dominios de configuración** > .<a href="https://go.microsoft.com/fwlink/p/?linkid=834818" target="_blank"></a>

2. El estado se mostrará en el lado derecho de cada dominio. Si los dominios de su organización apuntan a Microsoft Cloud Germany Exchange Online, deberá actualizar el registro MX.

3. Haga clic en el dominio y, a continuación, haga clic en **Errores DNS detectados y haga clic aquí para verlo**.

4. Esta página tendrá instrucciones para mostrar cómo corregir el registro MX. Si el registrador de su dominio usa [Domain Connect](../setup/add-domain.md#registrars-with-domain-connect), puede hacer clic en "Corregir mis registros" en la parte superior. De lo contrario, puede seguir el vínculo del asistente para **obtener instrucciones paso a paso** para el registrador.
