# <a name="update-your-mx-records-to-transition-to-the-global-exchange-online-service"></a>Actualizar los registros MX para realizar la transición al servicio global de Exchange Online

1. Inicie sesión en el [portal de administración de Microsoft 365](https://admin.microsoft.com)y vaya a Dominios **de**  >  **configuración**

2. El estado se mostrará en el lado derecho de cada dominio. Si los dominios de su organización apuntan a Microsoft Cloud Alemania Exchange Online, tendrá que actualizar el registro MX.

3. Haga clic en el dominio y, a continuación, haga clic en **errores dns detectados, haga clic aquí para ver.**

4. Esta página tendrá instrucciones para mostrar cómo corregir el registro MX. Si el registrador de su dominio usa [La conexión de](../setup/add-domain.md#registrars-with-domain-connect)dominio, puede hacer clic en "Corregir mis registros" en la parte superior. De lo contrario, puede seguir el vínculo del asistente para obtener **instrucciones paso** a paso para su registrador.
