# <a name="set-up-outlook-to-read-email"></a>Configuración de Outlook para leer correo electrónico

La suscripción de Microsoft 365 viene con una outlook basada en web, pero si su suscripción incluye aplicaciones cliente de Office, también obtendrá Outlook que puede instalar en su equipo o dispositivos personales.
  
## <a name="set-up-an-email-account-in-outlook"></a>Configuración de una cuenta de correo electrónico en Outlook

La primera vez que abra Outlook, se abrirá el Asistente para cuentas automáticas. Si no es así, elija **Archivo** y, a continuación, **Agregar cuenta**.
  
1. En la página **Cuentas de correo electrónico** , elija **Siguiente** \> **agregar cuenta**.
    
2. En la página **Configuración automática de la cuenta** , escriba su nombre, dirección de correo electrónico y contraseña y, a continuación, elija **Siguiente** \> **finalizar**.
    
Lea [el artículo completo](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b).
  
## <a name="how-do-i-install-the-office-apps-including-outlook"></a>Cómo instalar las aplicaciones de Office, incluido Outlook?

1. Ve a [admin.microsoft.com/OLS/MySoftware](https://admin.microsoft.com/OLS/MySoftware.aspx).
    
2. Inicie sesión con su cuenta profesional o educativa.
    
3. En Administrar instalaciones, seleccione **Instalar**.
    
## <a name="set-up-an-email-account-in-outlook-2016-for-mac"></a>Configurar una cuenta de correo electrónico en Outlook 2016 para Mac

La primera vez que abra la aplicación outlook, se abrirá el Asistente para configurar la bandeja de entrada. En el asistente: 
  
1. En la página **Configurar mi bandeja de entrada** , seleccione **Agregar cuenta**.
    
2. En la página **Cuentas**, seleccione **Exchange o Office 365**.
    
3. En la página **Escriba la información de la cuenta de Exchange** , escriba su nombre, dirección de correo electrónico y contraseña y, a continuación, seleccione **Agregar cuenta**.
    
Lea [el artículo completo](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b#PickTab=Outlook_for_Mac). 
  
## <a name="how-do-i-access-my-mail-online"></a>Cómo acceder a mi correo en línea?

Después de iniciar sesión en Microsoft 365, seleccione **Outlook**.
  
![Página principal de Microsoft 365 con la aplicación outlook resaltada.](../../media/3ceee838-9d85-4af3-95a6-fbcee11036f4.png)
  
¿No encuentra la aplicación que busca? En el iniciador de aplicaciones, seleccione **Todas las aplicaciones** para ver una lista alfabética de las aplicaciones de Microsoft 365 disponibles. Ahí podrá buscar una aplicación específica. 
  
## <a name="how-do-i-know-if-my-subscription-includes-office-apps"></a>Cómo saber si mi suscripción incluye aplicaciones de Office?

Microsoft 365 Empresa Estándar y Aplicaciones Microsoft 365 para negocios incluyen aplicaciones de Office. Para obtener más información, consulte [Planes de Microsoft 365 para empresas](https://go.microsoft.com/fwlink/p/?LinkId=723731) o [planes de Office 365 Enterprise](https://go.microsoft.com/fwlink/p/?LinkId=800029).
  
## <a name="how-do-i-determine-what-subscription-i-have"></a>Cómo determinar qué suscripción tengo?

Para determinar la suscripción, consulte [¿Qué suscripción tengo?](../admin-overview/what-subscription-do-i-have.md)
