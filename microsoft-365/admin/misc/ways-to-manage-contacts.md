# <a name="quick-help-ways-to-manage-contacts"></a>Ayuda rápida: formas de administrar contactos

Hay varias formas de trabajar con contactos de Microsoft 365 y la mayoría dependen de cómo se usan los contactos. Esta es una comparación de las diferentes formas de ayudarle a elegir la adecuada para su organización.
  
|||||
|:-----|:-----|:-----|:-----|
|**Característica** <br/> |**Descripción** <br/> |**Ventajas** <br/> |**Desventajas** <br/> |
|**Contactos de Microsoft 365** <br/> |Use los contactos de Microsoft 365 para almacenar sus clientes o contactos empresariales y aparecerán en Contactos en Microsoft 365. Ver [contactos](contacts.md) <br/> |Todos los usuarios de la organización pueden ver los contactos directamente desde Outlook, Outlook en la Web y dispositivos móviles.  <br/> |Solo los administradores pueden crear y actualizar los contactos.  <br/> No se permiten campos personalizados (por ejemplo: fecha de nacimiento, universitario, agente de referencia).  <br/> |
|**Contactos de SharePoint** <br/> |Use una aplicación contactos de SharePoint para almacenar los clientes o contactos empresariales y aparecerán en el sitio de SharePoint y en las aplicaciones móviles de SharePoint. Vea [listas y aplicaciones de SharePoint.](https://support.microsoft.com/office/0a1c3ace-def0-44af-b225-cfa8d92c52d7)  <br/> |Todos los usuarios con acceso al sitio de grupo de SharePoint y las aplicaciones móviles de SharePoint pueden ver y actualizar estos contactos.  <br/> Permite campos personalizados, vistas, flujos de trabajo y mucho más.  <br/> |Estos contactos no aparecen en Outlook ni en Contactos de Microsoft 365.  <br/> Requiere conocimientos básicos de la infraestructura de SharePoint.  <br/> |
|**Obtener acceso a contactos** <br/> |Use una aplicación contactos de Access para almacenar los clientes o contactos empresariales y aparecerán en Access o en un sitio de SharePoint si están hospedados allí. Vea [aplicaciones de Access.](https://support.microsoft.com/office/25f3ab3e-510d-44b0-accf-b976c0813e71)  <br/> |Todos los usuarios con acceso a Access o al sitio de SharePoint que hospeda la aplicación pueden ver y actualizar los contactos.  <br/> Administración de contactos de aspecto profesional y altamente personalizable creada con Access.  <br/> |Debe comprar Microsoft Access o cambiar a un plan de Microsoft 365 que incluya Access.  <br/> Requiere conocimientos básicos de Microsoft Access y cómo crear aplicaciones.  <br/> |
|**Dynamics CRM Online** <br/> |Usa Microsoft Dynamics CRM Online para almacenar y administrar tus clientes y posibles oportunidades de negocio. Vea [Dynamics CRM.](https://dynamics.microsoft.com)  <br/> |Potentes capacidades de administración de relaciones con el cliente (CRM).  <br/> Se integra con Microsoft 365 para que pueda trabajar desde un solo lugar.  <br/> |Incluye complejidad con la incorporación y personalización para satisfacer sus necesidades.  <br/> Su coste es significativamente mayor que el de cualquiera de las otras opciones de administración de contactos.  <br/> |
|**Aplicación de terceros** <br/> |Si ninguna de las opciones anteriores le interesa, puede considerar una aplicación de terceros en la [Tienda Office](https://store.office.com) o externa a Microsoft.  <br/> |Muchos proveedores, características y precios para elegir para los clientes empresariales.  <br/> |No hay garantía de que se integre con Microsoft 365, lo que requiere que trabaje con dos servicios diferentes, inicios de sesión, etc.  <br/> |
