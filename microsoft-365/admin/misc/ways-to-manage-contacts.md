# <a name="quick-help-ways-to-manage-contacts"></a>Ayuda rápida: formas de administrar contactos

Hay varias formas de trabajar con los contactos en Office 365 y gran parte de ellos depende de cómo use los contactos. Esta es una comparación de las diferentes formas de ayudarle a elegir la correcta para su organización.
  
|||||
|:-----|:-----|:-----|:-----|
|**Característica** <br/> |**Descripción** <br/> |**Ventajas** <br/> |**Desventajas** <br/> |
|**Office 365 contactos** <br/> |Use Office 365 contactos para almacenar sus clientes o contactos profesionales y aparecerán en personas en Office 365. Ver [contactos](contacts.md) <br/> |Todos los usuarios de la organización pueden ver los contactos directamente desde Outlook, Outlook en la web y dispositivos móviles.  <br/> |Solo los administradores pueden crear y actualizar los contactos.  <br/> No se permiten campos personalizados (por ejemplo: FechaNacimiento, Universidad, agente de referencia).  <br/> |
|**Contactos de SharePoint** <br/> |Use una aplicación de contactos de SharePoint para almacenar los clientes o contactos profesionales y aparecerán en el sitio de SharePoint y en las aplicaciones móviles de SharePoint. Vea [apps and lists de SharePoint](https://support.office.com/article/0a1c3ace-def0-44af-b225-cfa8d92c52d7.aspx).  <br/> |Todos los usuarios con acceso al sitio de grupo de SharePoint y las aplicaciones móviles de SharePoint pueden ver y actualizar estos contactos.  <br/> Permite los campos personalizados, vistas, flujos de trabajo y más.  <br/> |Estos contactos no aparecen en Outlook ni en las personas en Office 365.  <br/> Requiere conocimientos básicos de la infraestructura de SharePoint.  <br/> |
|**Obtener acceso a contactos** <br/> |Use una aplicación de contactos de Access para almacenar los clientes o contactos profesionales y aparecerán en Access o en un sitio de SharePoint si se hospedan allí. Consulte [Access apps](https://support.office.com/article/25f3ab3e-510d-44b0-accf-b976c0813e71.aspx).  <br/> |Todos los usuarios con acceso a Access o al sitio de SharePoint que hospeda la aplicación pueden ver y actualizar contactos.  <br/> Administración de contactos de aspecto profesional y muy personalizable integrada con Access.  <br/> |Debe comprar Microsoft Access o cambiar a un plan de Office 365 que incluya Access.  <br/> Requiere un conocimiento básico de Microsoft Access y de cómo crear aplicaciones.  <br/> |
|**Dynamics CRM Online** <br/> |Usar Microsoft Dynamics CRM Online para almacenar y administrar los clientes y las posibles oportunidades de negocio. Consulte [Dynamics CRM](https://dynamics.microsoft.com).  <br/> |Potentes capacidades de administración de relaciones con el cliente (CRM).  <br/> Se integra con Office 365 para que pueda trabajar desde un único punto.  <br/> |Incluye la complejidad con la incorporación y la personalización para satisfacer sus necesidades.  <br/> Es considerablemente mayor en costo que cualquiera de las demás opciones de administración de contactos.  <br/> |
|**Aplicación de terceros** <br/> |Si ninguna de las opciones anteriores le interesa, puede considerar una aplicación de terceros en la [tienda Office](https://store.office.com) o externa a Microsoft.  <br/> |Muchos proveedores, características y precios para elegir a los clientes de su empresa.  <br/> |No hay garantía de que esté integrada con Office 365, lo que requiere trabajar con dos servicios diferentes, inicios de sesión, etc.  <br/> |
