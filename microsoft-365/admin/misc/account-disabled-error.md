# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Error de obtención de una cuenta deshabilitada en Outlook en la Web

Si recibe el **error** con el que se ha deshabilitado la cuenta al intentar abrir Outlook en la Web (anteriormente conocido como Outlook Web App), es posible que el administrador haya deshabilitado el acceso a `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` Outlook en la Web.

El administrador puede corregir este error siguiendo los pasos del tema Habilitar o deshabilitar [Outlook Web App para un buzón.](https://docs.microsoft.com/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app)
