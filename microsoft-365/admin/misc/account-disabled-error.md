# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Obtención de un error de cuenta deshabilitada en Outlook en la Web

Si recibe el error **Su cuenta se ha deshabilitado** `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` al intentar abrir Outlook en la Web (anteriormente conocida como Outlook Web App), es posible que el administrador haya deshabilitado el acceso a Outlook en la Web.

El administrador puede corregir este error siguiendo los pasos del tema [Habilitar o deshabilitar Outlook Web App para un buzón](/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app).
