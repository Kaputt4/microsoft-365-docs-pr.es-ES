# <a name="empower-your-small-business-with-remote-work"></a>Potencie su pequeña empresa con el trabajo a distancia

A medida que las empresas se adaptan a la mayor necesidad de que las personas trabajen de forma remota y se conecten con sus clientes de forma virtual, este sitio se actualiza con la información de procedimientos más reciente, sugerencias, recursos e instrucciones sobre el trabajo remoto para empresas que usan Microsoft 365.

> [!TIP]
> ¿No tienes Microsoft_Teams? Obtenga 6 meses de Microsoft Teams en Office gratis (cuando se registre durante 1 año). Obtenga las tecnologías descritas en este artículo como parte de la oferta. Para obtener más información, consulte [Probar 1 mes gratis](https://aka.ms/SMBTeamsOffer).

## <a name="transitioning-to-a-remote-workforce"></a>Transición a un personal remoto

A la luz del brote de COVID-19 (Novel Coronavirus), muchos propietarios de negocios se encuentran con un personal de trabajo completamente remoto. Esto es lo que puede hacer para realizar una transición al trabajo remoto seguro, seguro y productivo.

Para obtener más información, consulte [Comenzar con Microsoft Teams en su pequeña empresa](https://support.microsoft.com/office/6723dc43-dbc0-46e6-af49-8a2d1c5cb937).

¿Ya tiene una suscripción, pero necesita configurarla? Consulte [Microsoft 365 formación para pequeñas empresas](../../business-video/index.yml).

## <a name="connect-with-employees-and-customers"></a>Conectar con empleados y clientes

Todavía puede conectarse con empleados, clientes, clientes y asociados, incluso si no puede encontrarse cara a cara. Use Microsoft Teams para seguir haciendo negocios y conectándose con sus clientes.

### <a name="meet-up-in-teams"></a>Reunirse en Teams

Hospedar reuniones de audio, vídeo y web con empleados, clientes o asociados a través de Microsoft Teams.

Para obtener más información, consulte [Organizar reuniones en línea](https://support.microsoft.com/office/65748808-a403-462c-a6e1-b169e5bc6c92) y [Chatear con empleados y clientes](https://support.microsoft.com/office/65748808-a403-462c-a6e1-b169e5bc6c92).

### <a name="store-and-share-files"></a>Almacenar y compartir archivos

Coloque los documentos empresariales en la nube y compártelos de forma segura, tanto interna como externamente.

Para obtener más información, consulte esta [Introducción al uso compartido de archivos empresariales](../../business-video/overview-file-sharing.md) y [Upload y compartir archivos](https://support.microsoft.com/office/upload-and-share-files-57b669db-678e-424e-b0a0-15d19215cb12).

### <a name="collaborate-with-customers"></a>Colaboración con clientes

Invite a empleados y clientes a un equipo para que pueda trabajar directamente con ellos para coordinar y colaborar en un solo lugar.

Para obtener más información, consulte [Creación de un equipo](https://support.microsoft.com/office/fccb4fa6-f864-4508-bdde-256e7384a14f) y [Habilitación del acceso de invitado](/MicrosoftTeams/guest-access).

## <a name="manage-and-secure-your-business-to-run-remotely"></a>Administración y protección de su negocio para que se ejecute de forma remota

Solo unos pocos pasos pueden ayudarle a mantener su empresa segura, incluso con empleados remotos y usuarios invitados.

### <a name="secure-your-users"></a>Protección de los usuarios

Use la autenticación multifactor para proteger los datos y los dispositivos.

Para obtener más información, vea [Activar la autenticación multifactor](../security-and-compliance/set-up-multi-factor-authentication.md).

### <a name="secure-your-devices"></a>Proteger sus dispositivos

Administre equipos y teléfonos de forma remota y asegúrese de que están protegidos y actualizados. Requiere Microsoft 365 Empresa.

Para obtener más información, consulte [Administración de dispositivos](../../business-video/secure-win-10-pro-devices.md).

### <a name="more-for-admins-and-partners"></a>Más para administradores y asociados

El centro de documentación técnica para Microsoft 365 Empresa se actualiza con nuevas instrucciones de trabajo remoto seguras.

Para obtener más información, consulte [Microsoft 365 Recursos empresariales](/microsoft-365/business).

## <a name="need-to-ask-a-question"></a>¿Necesita hacer una pregunta?

Pregunte en el [foro de Teams](https://answers.microsoft.com/msteams/forum) o en el [foro de administradores de Office](https://answers.microsoft.com).

> La mayoría de las tareas de este artículo y vídeo se pueden realizar con una suscripción a Microsoft 365 Empresa Básico (anteriormente Office 365 Empresa Essentials), pero algunas requieren una suscripción Premium.
