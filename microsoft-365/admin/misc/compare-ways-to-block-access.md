# <a name="compare-ways-to-block-access-to-microsoft-365"></a>Comparar formas de bloquear el acceso a Microsoft 365

Cuando un empleado deja la organización, en buenos términos o en mal estado, debe bloquear su acceso a Microsoft 365. Estas son algunas maneras de hacerlo.
  
|Forma de bloquear el acceso|Definición|Procedimiento recomendado|
|:-----|:-----|:-----|
|Bloquear inicio de sesión  <br/> |Una manera de impedir que un usuario acceda a Microsoft 365 es cambiar su estado de inicio de sesión a **Inicio de sesión bloqueado**. Esto les impide iniciar sesión en Microsoft 365 desde sus equipos y dispositivos móviles, aunque todavía pueden ver el correo electrónico y los documentos descargados o sincronizados anteriormente. Si usa Blackberry Enterprise Service, también puede deshabilitar su acceso allí.  <br/> |Use cuando un empleado planee abandonar la organización o tenga previsto tomar una licencia de ausencia a largo plazo.  <br/> |
|Restablecer contraseña de usuario  <br/> |Otra manera de evitar que un usuario acceda a Microsoft 365 es restablecer su contraseña. Esto les impide usar su cuenta, aunque todavía pueden ver el correo electrónico y los documentos descargados o sincronizados anteriormente. A continuación, puede iniciar sesión como ellos y cambiar la contraseña a una de sus opciones.  <br/> |Use cuando un empleado se vaya de repente y permanentemente y sienta que hay preocupación por los datos empresariales.  <br/> |
|Eliminación de todas las licencias asignadas  <br/> |Otra opción es quitar las licencias de Microsoft 365 asignadas al usuario. Esto impide que usen aplicaciones y servicios como el conjunto de aplicaciones de Office, las aplicaciones de Office para la web, Yammer y SharePoint Online. Todavía pueden iniciar sesión, pero no pueden usar estos servicios.  <br/> |Úselo cuando considere que este usuario ya no necesita acceso a características específicas de Microsoft 365.  <br/> <br> **Importante:** Al quitar una licencia, el buzón del usuario se eliminará en 30 días.
