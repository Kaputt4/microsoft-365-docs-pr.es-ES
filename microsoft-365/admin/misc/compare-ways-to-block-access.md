# <a name="compare-ways-to-block-access"></a>Comparar formas de bloquear el acceso

Cuando un empleado deja su organización, en condiciones buenas o malas, debe bloquear su acceso a Microsoft 365. Estas son algunas maneras de hacerlo.
  
|Forma de bloquear el acceso|Definición|Procedimiento recomendado|
|:-----|:-----|:-----|
|Bloquear inicio de sesión  <br/> |Una forma de impedir que un usuario acceda a Microsoft 365 es cambiar su estado de inicio de sesión a **Inicio de sesión bloqueado.** Esto les impide iniciar sesión en Microsoft 365 desde sus equipos y dispositivos móviles, aunque aún pueden ver documentos y correo electrónico previamente descargados o sincronizados. Si usas el servicio de Enterprise Blackberry, también puedes deshabilitar su acceso allí.  <br/> |Se usa cuando un empleado planea abandonar la organización o tiene previsto tomar un permiso de ausencia a largo plazo.  <br/> |
|Restablecer contraseña de usuario  <br/> |Otra forma de impedir que un usuario acceda a Microsoft 365 es restablecer su contraseña. Esto les impide usar su cuenta, aunque aún pueden ver documentos y correo electrónico descargados o sincronizados anteriormente. A continuación, puede iniciar sesión como ellos y cambiar la contraseña a una de las opciones que elija.  <br/> |Úselo cuando un empleado se vaya de forma repentina y permanente y sienta que hay preocupación por los datos empresariales.  <br/> |
|Quitar todas las licencias asignadas  <br/> |Otra opción es quitar cualquier Microsoft 365 licencias asignadas al usuario. Esto les impide usar aplicaciones y servicios como el conjunto de Office, Office aplicaciones para la web, Yammer y SharePoint Online. Todavía pueden iniciar sesión, pero no pueden usar estos servicios.  <br/> |Úselo cuando sienta que este usuario ya no necesita acceso a características específicas de Microsoft 365.  <br/> <br> **Importante:** Al quitar una licencia, el buzón del usuario se eliminará en 30 días.
