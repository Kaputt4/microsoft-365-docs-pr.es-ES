# <a name="compare-ways-to-block-access"></a>Comparar formas de bloquear el acceso

Cuando un empleado abandona su organización, en condiciones buenas o mal, debe bloquear su acceso a Microsoft 365. Estas son algunas maneras de hacerlo.
  
||||
|:-----|:-----|:-----|
|**Forma de bloquear el acceso** <br/> |**Definición** <br/> |**Procedimiento recomendado** <br/> |
|Bloquear el inicio de sesión  <br/> |Una forma de impedir que un usuario acceda a Microsoft 365 es cambiar su estado de inicio de sesión a Inicio de sesión **bloqueado.** Esto les impide iniciar sesión en Microsoft 365 desde sus equipos y dispositivos móviles, aunque aún pueden ver el correo electrónico y los documentos previamente descargados o sincronizados. Si usa el servicio Blackberry Enterprise, también puede deshabilitar su acceso allí.  <br/> |Se usa cuando un empleado planea abandonar la organización o tiene previsto tomar una ausencia a largo plazo.  <br/> |
|Restablecer contraseña de usuario  <br/> |Otra forma de evitar que un usuario acceda a Microsoft 365 es restablecer su contraseña. Esto les impide usar su cuenta, aunque aún pueden ver el correo electrónico y los documentos previamente descargados o sincronizados. A continuación, puede iniciar sesión como ellos y cambiar la contraseña a una de las opciones que elija.  <br/> |Se usa cuando un empleado se va de forma repentina y permanente y cree que hay preocupación por los datos empresariales.  <br/> |
|Quitar todas las licencias asignadas  <br/> |Otra opción es quitar las licencias de Microsoft 365 asignadas al usuario. Esto les impide usar aplicaciones y servicios como el conjunto de aplicaciones de Office, las aplicaciones de Office para la web, Yammer y SharePoint Online. Aún pueden iniciar sesión, pero no pueden usar estos servicios.  <br/> |Úse cuando sienta que este usuario ya no necesita acceso a características específicas de Microsoft 365.  <br/> <br> **Importante:** Al quitar una licencia, el buzón del usuario se eliminará en 30 días.
