# <a name="switch-from-google-workspace---overview"></a>Cambio de Google Workspace - Información general

Puede seguir los siguientes pasos para mover sus datos, correo electrónico y usuarios de Google Workspace a Microsoft 365 empresarial. Utilice los artículos y vídeos de cada paso para ayudarse a preparar y configurar su entorno para la migración y a usar las herramientas de migración disponibles en el centro de administración de Microsoft.


| Paso  |Descripción  |
|---------|---------|
|Paso 1 | [Suscribirse](../admin-overview/sign-up-for-office-365.md) a Microsoft 365 Empresa Premium       |
|Paso 2 | [Configurar Microsoft 365 para la migración a Google Workspace](set-up-microsoft-365-forgoogle.md). </br> En este paso podrá validar que es el propietario del dominio que utiliza en Google Workspace, agregar usuarios y configurar la seguridad básica de los dispositivos. |
|Paso 3 | [Configurar directivas de seguridad para dispositivos Windows.](../setup/secure-win-10-pcs.md)</br> La seguridad de Windows se configura por separado en la página de configuración del Centro de administración de Microsoft 365. |
|Paso 4 | [Agregue su dominio de Google Workspace a Microsoft 365](add-google-domain.md) </br> Una vez que verifique que es el propietario del dominio que usa para el correo electrónico, usted y todos los demás usuarios pueden empezar a usar su antiguo correo electrónico para iniciar sesión en Microsoft 365 Empresa Premium. |
|Paso 5 | [Instalar las aplicaciones de Office y Microsoft Teams](../setup/install-applications.md).</br> Todas las personas que tengan licencias de Microsoft 365 deben instalar las apps de Office en sus dispositivos de trabajo.|
|Paso 6 | [Migrar el correo electrónico y los elementos del calendario de todos](migrate-email.md).</br> En este paso, realizará una migración de Exchange Online para trasladar el correo electrónico, el calendario y los contactos de todos los usuarios desde Google Workspace.  |
|Paso 7 | [Conectar el dominio a Microsoft 365](connect-domain-tom365.md). </br> Después de conectar el correo electrónico del dominio comenzará a ir a Microsoft 365, y todos sus servicios de Microsoft 365 funcionarán.|
|Paso 8|Use el [Administrador de migraciones para mover los datos de todos](migrate-files-migration-manager.md) de Drive a OneDrive y de las unidades compartidas a los sitios de equipo.</br> En este paso, todos los datos de las unidades personales y compartidas se copian y se trasladan a Microsoft 365.|
|Paso 9| [Interrumpir el uso de Google Workspace](cancel-google.md) pero conservar el dominio. </br> Si Google administra su dominio, puede mantenerlo allí incluso después de interrumpir su suscripción a Google Workspace. También puede optar por moverlo a otro host DNS si lo desea.|
