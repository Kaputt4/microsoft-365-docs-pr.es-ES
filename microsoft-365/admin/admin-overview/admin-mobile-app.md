# <a name="about-the-microsoft-365-admin-mobile-app"></a>Acerca de la aplicación móvil de administración de Microsoft 365

¿Es usted un administrador que normalmente está fuera de casa? A continuación, consulte la aplicación de administración de [Microsoft 365.](https://go.microsoft.com/fwlink/?LinkID=627216) Puede usar la aplicación para administrar su organización en línea desde su teléfono o tableta. Puedes obtener la aplicación de Apple Store y de Google Play. <br> 

![Captura de pantalla de la página principal de la aplicación móvil de administración, que muestra la búsqueda, los mensajes, el estado y los vínculos rápidos.](../../media/admin-mobile-app-darkbg.png)

La aplicación de administración es una aplicación con características enriquecciones que tiene más de 80 características que le ayudarán a administrar su organización cuando no puede acceder a un equipo. Esta es una lista de algunas de las tareas que puedes realizar desde la aplicación:

- Obtener notificaciones de cambio y estado del servicio
- Crear una solicitud de servicio
- Restablecer la contraseña de un usuario
- Asignar licencias
- Agregar o quitar pertenencias a grupos
- Agregar usuarios 

Y si es alguien responsable de más de una organización en línea, inicia sesión en varias organizaciones y cambia rápidamente entre ellas. 
  
> [!IMPORTANT]
> Si tiene problemas para usar la aplicación móvil Admin en iOS o Android, envíenos un correo [electrónico a feedback365@microsoft.com](mailto:feedback365@microsoft.com) para darnos información.

## <a name="frequently-asked-questions"></a>Preguntas frecuentes

A continuación están las respuestas a algunas de las preguntas más frecuentes sobre la aplicación.
  
### <a name="what-do-i-need-to-do-to-be-able-to-use-the-app"></a>¿Qué necesito hacer para poder usar la aplicación?

Debe tener una suscripción válida de Aplicaciones de Microsoft 365 para empresas o Microsoft 365 Empresa Premium con privilegios de administrador para iniciar sesión y usar la aplicación. La aplicación móvil Administrador no es un cliente de correo electrónico. Es una aplicación administrativa enriquecte con más de 80 características para ayudarle a dar soporte a su organización mientras está fuera.
  
### <a name="can-i-use-the-app-with-my-microsoft-365-family-subscription"></a>¿Puedo usar la aplicación con mi suscripción a Microsoft 365 Family?

No, la aplicación solo funciona con las suscripciones de Aplicaciones de Microsoft 365 para empresas y Microsoft 365 Empresa Premium. 
  
### <a name="will-the-app-work-if-my-organization-has-directory-synchronization-enabled"></a>¿Funcionará la aplicación si la organización tiene habilitada la sincronización de directorios?

Sí, pero con funcionalidad reducida. Podrá iniciar sesión y ver la información del servicio, pero la mayoría de las funciones de administración de usuarios serán de solo lectura. No podrá agregar, editar ni eliminar usuarios. Sin embargo, podrá asignar licencias a los usuarios de su organización y recibir notificaciones.
  
### <a name="what-languages-are-supported-by-the-app"></a>¿Qué idiomas son compatibles con la aplicación?

La aplicación admite los 39 idiomas que admite el Centro de administración de Microsoft 365 basado en web. Para cambiar el idioma, seleccione el icono de inicio de la aplicación en la barra superior y elija **Idioma de**  >  **configuración.**
  
### <a name="why-does-the-messages-tile-on-the-home-screen-show-numbers-even-after-ive-read-the-new-messages"></a>¿Por qué el icono Mensajes de la pantalla de inicio muestra números a pesar de que ya he leído los mensajes nuevos?

Los números son el total de los mensajes de la semana pasada, no los mensajes no leídos.
  
### <a name="how-can-i-share-the-service-incidents-and-messages-with-the-rest-of-my-organization"></a>¿Cómo puedo compartir los mensajes e incidencias de servicio con el resto de mi organización?

Si selecciona un mensaje o incidente de servicio específico, la opción compartir estará en la esquina superior derecha. Actualmente se admite el uso compartido de correo electrónico.
  
### <a name="can-i-use-this-app-with-multiple-accounts-or-tenants"></a>¿Puedo usar esta aplicación con varias cuentas o inquilinos?

Sí, puede configurar varias cuentas o inquilinos.
  
### <a name="my-app-is-acting-funny-what-can-i-do-to-troubleshoot-weird-app-behavior"></a>Mi aplicación actúa de forma extraña. ¿Qué puedo hacer para solucionar problemas de comportamiento extraño de la aplicación?

Puede probar algunos pasos para la solución de problemas de aplicaciones móviles comunes:
  
1. Cierre y vuelva a abrir la aplicación.
    
2. Desinstale y vuelva a instalar la aplicación.

3. Si eso no funciona, puede enviarnos un correo [electrónico a feedback365@microsoft.com](mailto:feedback365@microsoft.com) para darnos información.
    
### <a name="what-do-i-do-if-my-question-isnt-answered"></a>¿Qué hago si no encuentro la respuesta a mi pregunta?

Enviar [feedback365@microsoft.com](mailto:feedback365@microsoft.com) para informar de un problema con la aplicación. También puede enviar comentarios en la parte inferior de este artículo.
