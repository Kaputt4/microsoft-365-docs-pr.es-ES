# <a name="windows-365-business-sizing-options"></a>Opciones de tamaño de Windows 365 Empresa

Windows 365 Empresa ofrece licencias de precio fijo (a través de Microsoft 365) para diferentes tamaños de PC en la nube. Al asignar una licencia a un usuario, debe seleccionar una de varias opciones de tamaño. Cada una tiene un número diferente de CPU, RAM y almacenamiento, y está diseñada para admitir diferentes escenarios de uso. Evalúe los requisitos empresariales para determinar qué tamaños tienen sentido para los usuarios. 

En esta tabla se muestran ejemplos de los diferentes tamaños disponibles para un equipo en la nube.

> [!NOTE]  
> Si Aplicaciones Microsoft 365 y Microsoft Teams se incluyen en la **columna Aplicaciones** admitidas, están preinstaladas para esas opciones de PC en la nube. 


|CPU, RAM y almacenamiento|Escenarios de ejemplo|Aplicaciones compatibles|
|:-------------------------------------------------------------------------------|:----------------------------------|:----------------------------------|
|1vCPU/2 GB/64 GB |Trabajadores de primera línea, centros de llamadas, educación, formación y acceso a CRM.|Office ligera (basada en web), Microsoft Edge, OneDrive, aplicación ligera de línea de negocio (aplicación de centro de llamadas – aplicaciones web), soporte técnico de Defender. |
|2vCPU/4 GB/256 GB<br/> 2vCPU/4 GB/128 GB<br/> 2vCPU/4 GB/64 GB |Fusiones y adquisiciones, servicios al cliente a corto plazo y estacionales |Aplicaciones Microsoft 365, Microsoft Teams ligera (solo chat y audio), OneDrive, Adobe Reader, Microsoft Edge, aplicaciones de línea de negocio, soporte técnico de Defender.  |
|2vCPU/8 GB/256 GB<br/>2vCPU/8 GB/128 GB |Bring-your-own-PC, work from home, market researchers, government, consultants. |Aplicaciones Microsoft 365, Microsoft Teams, Outlook, Excel, Access, PowerPoint, OneDrive, Adobe Reader, Microsoft Edge, aplicaciones de línea de negocio, soporte de Defender.  |
|4vCPU/16 GB/512 GB<br/>4vCPU/16 GB/256 GB<br/> 4vCPU/16 GB/128 GB|Finanzas, gobierno, consultores, servicios de salud, bring-your-own-PC, work-from-home. |Aplicaciones Microsoft 365, Microsoft Teams, Outlook, Excel, Access, PowerPoint, Power BI, Dynamics 365, OneDrive, Adobe Reader, Microsoft Edge, aplicaciones de línea de negocio, soporte técnico de Defender. |
|8vCPU/32 GB/512 GB<br/>8vCPU/32 GB/256 GB<br/>8vCPU/32 GB/128 GB |Desarrolladores de software, ingenieros, creadores de contenido, estaciones de trabajo de diseño e ingeniería. |Aplicaciones Microsoft 365, Microsoft Teams, Outlook, Access, OneDrive, Adobe Reader, Microsoft Edge, Power BI, Visual Studio Code, aplicaciones de línea de negocio, Defender soporte técnico.  |
