# <a name="microsoft-home-use-program-frequently-asked-questions-faq"></a>Preguntas más frecuentes (P+F) del Programa de uso en el hogar de Microsoft

Los clientes de licencias por volumen usan el Programa de uso en casa (HUP) de Microsoft en el sitio web de [la Microsoft Store para Empresas.](https://go.microsoft.com/fwlink/?linkid=2139192) 

Todos los demás clientes acceden a HUP mediante este paso:

- En el Centro de administración de Microsoft, vaya a la página Facturación de sus productos, seleccione la pestaña Beneficios y, a continuación,  >  [](https://go.microsoft.com/fwlink/p/?linkid=842054)seleccione la **pestaña HUP.** 

## <a name="what-is-the-microsoft-home-use-program"></a>¿Qué es el Programa de uso en el hogar de Microsoft?

El Programa de uso en el hogar (HUP) de Microsoft permite a los empleados de las empresas aptas comprar productos de consumidor de Microsoft con un descuento. Puede comprobar los descuentos actuales de cada oferta de producto visitando la página de productos del Programa de uso [principal.](https://www.microsoft.com/home-use-program)

## <a name="in-which-markets-is-the-home-use-program-offered"></a>¿En qué mercados se ofrece el Programa de uso en el hogar?

Los descuentos en las suscripciones familiares y personales de Microsoft 365 se ofrecen a las empresas aptas en todo el mundo con estas excepciones:

- Macao, Rusia y Venezuela: esta oferta no está disponible en estos mercados.
- China y Hong Kong: hay disponible una suscripción de un año en estos mercados. Para China, visite [Microsoft 365 en China.](https://www.microsoftstore.com.cn/home-use-program/invite) For Hong Kong, visit [Buy Microsoft 365 at a discounted price](https://www.microsoftestore.com.hk/partner/hup?locale=en_HK).
- Japón: Microsoft 365 Family no está disponible.

## <a name="how-does-an-organization-qualify-for-hup-discounts"></a>¿Cómo califica una organización para los descuentos de HUP?

Para que los empleados aprovechen el descuento del Programa de uso en casa de Microsoft, necesitan una dirección de correo electrónico corporativa válida y una cuenta de Microsoft activa, y su organización debe cumplir uno de los siguientes criterios:

- Clientes que tienen Software Assurance activo para las aplicaciones de Office que reúnen los requisitos 

- Clientes que tienen Microsoft 365/Microsoft 365 E3/E5 desde Software Assurance

- Los clientes comerciales y gubernamentales que ≥ 2.000 licencias de una o una combinación de Microsoft 365 E3/E5

- Clientes educativos y sin ánimo de lucro

## <a name="why-do-i-have-to-create-an-azure-active-directory-aad-account-to-use-the-home-use-program-microsoft-store-for-business"></a>¿Por qué tengo que crear una cuenta de Azure Active Directory (AAD) para usar el Programa de uso principal o la Microsoft Store para Empresas?

Para actualizar y modernizar el Programa de uso en casa a una gama más amplia de ventajas de licencia y producto, tuvimos que cambiar la forma en que cumplimos con HUP. La Microsoft Store para Empresas se usa para la validación de elegibilidad y para asignar ventajas de HUP, y para ello se basa en AAD para conectar información desde VSLC y para validar los dominios de correo electrónico en los que se encuentran los empleados de la organización. Para obtener más información, vea Para registrarse en cuentas de Azure AD a través [de Office 365 para empresas.](https://docs.microsoft.com/microsoft-store/sign-up-microsoft-store-for-business#o365-welcome)

## <a name="how-do-i-become-the-assigned-it-admin-of-a-managed-tenant"></a>¿Cómo puedo convertirme en el administrador de TI asignado de un inquilino administrado?

Debe crear un registro especial en DNS para demostrar que es el propietario del dominio de su empresa, ex:contoso.com.

Para obtener un proceso paso a paso para convertirse en un inquilino administrado, vea Tomar el control de un directorio no administrado como [administrador en Azure Active.](https://docs.microsoft.com/azure/active-directory/users-groups-roles/domains-admin-takeover)

## <a name="what-resources-are-available-to-help-promote-the-home-use-program-benefit-to-employees"></a>¿Qué recursos están disponibles para ayudar a promover las ventajas del Programa de uso en el hogar para los empleados?

Use los recursos que se encuentran en la [página principal](https://www.microsoft.com/home-use-program/resources) del Programa de uso en casa para ayudarle a promover el programa a los empleados. Encontrará folletos digitales e imprimibles, artículos para su uso en correos electrónicos y boletines, así como instrucciones sobre cómo promocionar mejor el Programa de uso en el hogar.

Puedes acceder a los materiales de incorporación en varios idiomas en [HUP Material para clientes.](https://microsofteur.sharepoint.com/teams/HUPMaterial) Selecciona un idioma para empezar a descargar los archivos. El archivo contiene activos que puedes usar para ayudar a enseñar a los empleados sobre el programa y promocionarlo internamente en tu empresa.

## <a name="what-are-the-steps-for-an-eligible-employee-to-make-a-microsoft-home-use-program-purchase"></a>¿Cuáles son los pasos para que un empleado apto realice una compra del Programa de uso en el hogar de Microsoft?

1. **Registrarse** Vaya a la [página Programa de](https://www.microsoft.com/home-use-program)uso en casa de Microsoft y seleccione el botón Introducción. Los empleados escriben su dirección de correo electrónico corporativa.

2. **Confirmar**  Se envía un correo electrónico a la bandeja de entrada de su dirección de correo electrónico corporativa con un vínculo que autentica su elegibilidad para el programa.

3. **Iniciar sesión** Una vez autenticados, comienzan el proceso de compra proporcionando la cuenta de Microsoft asociada a su dirección de correo electrónico personal, lo que les permite ver los descuentos de los productos disponibles en función de las ventajas del Programa de uso en casa de la organización. Para obtener más información, consulta [Cómo iniciar sesión en tu cuenta de Microsoft.](https://support.microsoft.com/help/4028195/microsoft-account-sign-in)

4. **Tienda** La compra se procesa y se completa en Microsoft Store.

## <a name="if-an-employee-subscribes-to-microsoft-365-through-the-microsoft-home-use-program-how-does-this-impact-our-organizations-microsoft-365-business-subscription-and-device-limit"></a>Si un empleado se suscribe a Microsoft 365 a través del Programa de uso en el hogar de Microsoft, ¿cómo afecta esto a la suscripción de Microsoft 365 Empresa y al límite de dispositivos de nuestra organización?

Las instalaciones a través del Programa de uso en casa no tienen ningún impacto en la suscripción de Microsoft 365 Empresa de su organización. Cuando los empleados activan Microsoft 365 en sus dispositivos personales, usan su cuenta personal de Microsoft, no su cuenta de trabajo. Los administradores de TI de la organización no administran ni admiten las compras realizadas a través del Programa de uso en el hogar.

## <a name="im-having-trouble-managing-my-organizations-microsoft-home-use-program-benefit-where-can-i-get-support"></a>I'm having trouble managing my organization's Microsoft Home Use Program benefit. ¿Dónde puedo obtener soporte técnico?

Para obtener soporte técnico para problemas de migración desde el Centro de servicios de licencias por volumen o para acceder a la Microsoft Store para Empresas, consulta [soporte técnico de VLSC.](https://www.microsoft.com/Licensing/servicecenter/default.aspx?wa=wsignin1.0)

Para obtener problemas para ver las ventajas después de haber accedido a la Microsoft Store para Empresas, consulta El soporte técnico para administradores [de la Tienda para empresas](https://docs.microsoft.com/microsoft-store/)

## <a name="why-am-i-not-seeing-an-option-to-purchase-visio-professional-2019-andor-project-professional-2019"></a>¿Por qué no veo una opción para comprar Visio Professional 2019 o Project Professional 2019?

Si uno de sus empleados ha aprovechado el programa anterior y está buscando descuentos en Visio Professional 2019 o Project Professional 2019, vaya a través del proceso de validación de elegibilidad y, una vez que haya iniciado sesión con su cuenta personal de Microsoft, vaya a Otros productos de **Office** para comprar estos productos. Si no ve los descuentos esperados aplicados, póngase en contacto con el administrador del programa de uso en casa de su empresa para preguntar sobre estos beneficios adicionales.

- Facturación

- Instalación de Office
