# <a name="microsoft-home-use-program-frequently-asked-questions-faq"></a>Preguntas más frecuentes del Programa de uso principal de Microsoft (FAQ)

Los clientes de licencias por volumen usan el Programa de uso local (HUP) de Microsoft en el sitio [web de microsoft Store para empresas](https://go.microsoft.com/fwlink/?linkid=2139192).

Todos los demás clientes tienen acceso a HUP mediante este paso:

- En el Centro de administración de Microsoft, vaya a la página **Facturación** de sus productos, seleccione la pestaña Beneficios  >  [](https://go.microsoft.com/fwlink/p/?linkid=842054)y, a continuación, seleccione la **pestaña HUP.** 

## <a name="what-is-the-microsoft-home-use-program"></a>¿Qué es el Programa de uso en casa de Microsoft?

El Programa de uso en casa (HUP) de Microsoft permite a los empleados de las empresas elegibles comprar productos de consumidor de Microsoft con un descuento. Para comprobar los descuentos actuales de cada oferta de productos, visite la página [de productos del Programa de uso principal](https://www.microsoft.com/home-use-program).

## <a name="in-which-markets-is-the-home-use-program-offered"></a>¿En qué mercados se ofrece el Programa de uso en casa?

Los descuentos en las suscripciones familiares y personales de Microsoft 365 se ofrecen a empresas elegibles de todo el mundo con estas excepciones:

- Macao, Rusia y Venezuela: esta oferta no está disponible en estos mercados.
- China y Hong Kong: una suscripción de un año está disponible en estos mercados. Para China, visite [Microsoft 365 en China](https://www.microsoftstore.com.cn/home-use-program/invite). Para Hong Kong, visite [Comprar Microsoft 365 a un precio con descuento.](https://www.microsoftestore.com.hk/partner/hup?locale=en_HK)
- Japón: Microsoft 365 Family no está disponible.

## <a name="how-does-an-organization-qualify-for-hup-discounts"></a>¿Cómo califica una organización para los descuentos de HUP?

Para que los empleados aprovechen el descuento del Programa de uso en casa de Microsoft, necesitan una dirección de correo electrónico corporativa válida y una cuenta de Microsoft activa, y su organización debe cumplir uno de los siguientes criterios:

- Clientes que tienen Software Assurance activo para aplicaciones de Office que califiquen
- Clientes que tienen Microsoft 365/Microsoft 365 E3/E5 From Software Assurance
- Clientes comerciales y gubernamentales que han comprado ≥ 2.000 licencias de una o una combinación de Microsoft 365 E3/E5
- Clientes educativos y sin ánimo de lucro

## <a name="why-do-i-have-to-create-an-azure-active-directory-aad-account-to-use-the-home-use-program-microsoft-store-for-business"></a>¿Por qué tengo que crear una cuenta de Azure Active Directory (AAD) para usar el Programa de uso principal/Microsoft Store para empresas?

Para actualizar y modernizar el Programa de uso en casa a una gama más amplia de ventajas de licencia y producto, tuvimos que cambiar la forma en que cumplimos con HUP. Microsoft Store para empresas se usa para la validación de elegibilidad y para asignar ventajas hup, y para ello se basa en AAD para conectar información de VSLC y validar los dominios de correo electrónico en los que están los empleados de la organización. Para obtener más información, vea [To sign up for Azure AD accounts through Office 365 for Business](/microsoft-store/sign-up-microsoft-store-for-business#o365-welcome).

## <a name="how-do-i-become-the-assigned-it-admin-of-a-managed-tenant"></a>¿Cómo puedo convertirme en el administrador de TI asignado de un inquilino administrado?

Debe crear un registro especial en DNS para demostrar que es el propietario del dominio de su empresa, ex:contoso.com.

Para obtener un proceso paso a paso para convertirse en un inquilino administrado, consulte [Take over an unmanaged directory as administrator in Azure Active](/azure/active-directory/users-groups-roles/domains-admin-takeover).

## <a name="what-resources-are-available-to-help-promote-the-home-use-program-benefit-to-employees"></a>¿Qué recursos están disponibles para ayudar a promover el beneficio del Programa de uso en casa para los empleados?

Use los recursos que se encuentran en la página principal del [Programa](https://www.microsoft.com/home-use-program/resources) de uso principal para ayudarle a promover el programa a sus empleados. Encontrará volantes digitales e imprimibles, documentos para su uso en correos electrónicos y boletines, así como instrucciones sobre cómo promover mejor el Programa de uso en casa.

Puede acceder a los materiales de incorporación en varios idiomas en [HUP Material for Customers](https://microsofteur.sharepoint.com/teams/HUPMaterial). Seleccione un idioma para empezar a descargar los archivos. El archivo contiene activos que puede usar para ayudar a educar a los empleados sobre el programa y promoverlo internamente en su empresa.

## <a name="what-are-the-steps-for-an-eligible-employee-to-make-a-microsoft-home-use-program-purchase"></a>¿Cuáles son los pasos para que un empleado elegible realice una compra del Programa de uso en casa de Microsoft?

1. **Registrarse** Vaya a la [página Programa de uso principal de Microsoft](https://www.microsoft.com/home-use-program)y seleccione el botón Introducción. Los empleados escriben su dirección de correo electrónico corporativa.
2. **Confirmar**  Un correo electrónico se envía a la bandeja de entrada de su dirección de correo electrónico corporativa con un vínculo que autentica su elegibilidad para el programa.
3. **Iniciar sesión** Una vez autenticados, comienzan el proceso de compra proporcionando la cuenta de Microsoft asociada a su dirección de correo electrónico personal, lo que les permite ver los descuentos en los productos disponibles en función de las ventajas del Programa de uso en casa de la organización. Para obtener más información, [consulta How to sign in to your Microsoft account](https://support.microsoft.com/help/4028195/microsoft-account-sign-in).
4. **Tienda** La compra se procesa y se completa en microsoft store.

## <a name="if-an-employee-subscribes-to-microsoft-365-through-the-microsoft-home-use-program-how-does-this-impact-our-organizations-microsoft-365-business-subscription-and-device-limit"></a>Si un empleado se suscribe a Microsoft 365 a través del Programa de uso en casa de Microsoft, ¿cómo afecta esto a la suscripción empresarial y el límite de dispositivos de nuestra organización?

Las instalaciones a través del Programa de uso en casa no tienen ningún impacto en la suscripción empresarial de Microsoft 365 de su organización. Cuando los empleados activan Microsoft 365 en sus dispositivos personales, usan su cuenta personal de Microsoft, no su cuenta de trabajo. Los administradores de TI de su organización no administran ni admiten las compras realizadas a través del Programa de uso en casa.

## <a name="im-having-trouble-managing-my-organizations-microsoft-home-use-program-benefit-where-can-i-get-support"></a>Tengo problemas para administrar las ventajas del Programa de uso en casa de Microsoft de mi organización. ¿Dónde puedo obtener soporte técnico?

Para obtener soporte técnico para problemas de migración desde el Centro de servicios de licencias por volumen o para obtener acceso a Microsoft Store para empresas, consulte [VlSC Support](https://www.microsoft.com/Licensing/servicecenter/default.aspx?wa=wsignin1.0).

Para ver los problemas para ver las ventajas después de haber accedido ya a microsoft store para empresas, consulta Soporte técnico de la Tienda [para administradores empresariales](/microsoft-store/)

## <a name="why-am-i-not-seeing-an-option-to-purchase-visio-professional-2019-andor-project-professional-2019"></a>¿Por qué no veo una opción para comprar Visio Professional 2019 o Project Professional 2019?

Si uno de sus empleados se ha aprovechado del programa anterior y está buscando descuentos en Visio Professional 2019 o Project Professional 2019, consulte el proceso de validación de elegibilidad y, una vez que haya iniciado sesión con su cuenta personal de Microsoft, vaya a Otros productos de **Office** para comprar estos productos. Si no ve los descuentos esperados aplicados, póngase en contacto con el administrador del programa de uso en casa de su empleador para preguntar sobre estos beneficios adicionales.

## <a name="what-if-my-employees-have-questions-about-this-program-or-need-support-with-purchasing-and-installation"></a>¿Qué ocurre si mis empleados tienen preguntas sobre este programa o necesitan soporte técnico con la compra y la instalación?

Si los empleados tienen preguntas, refiéralas a los siguientes recursos:

- [Preguntas más frecuentes sobre la instalación y la compra posteriores](https://products.office.com/microsoft-office-for-home-and-school-faq)

- [Preguntas más frecuentes sobre los empleados del programa de uso principal](https://www.microsoft.com/home-use-program/frequently-asked-questions)

 Para obtener ayuda con problemas técnicos, consulte [Soporte técnico de Microsoft](https://support.microsoft.com/). Encontrará ayuda para:

- Estado del pedido
- Facturación
- Instalación de Office
- Iniciar sesión en Office
