# <a name="billing-information-for-microsoft-365-for-business-in-mexico"></a>Información de facturación para Microsoft 365 para empresas en México

## <a name="1-where-can-i-get-an-invoice-for-my-microsoft-365-for-business-purchase"></a>1. ¿Dónde puedo obtener una factura para la compra de Microsoft 365 para empresas?

La factura se envía a la dirección de correo electrónico de notificación de facturación 24 horas después de que se confirme la compra. Para descargar la factura en el centro de administración de Microsoft 365, vaya **a la**  >  página <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">facturas & pagos</a> .

> [!IMPORTANT]
> Si desea que la factura incluya su RFC, debe agregarla antes de la confirmación de compra.

## <a name="2-how-can-i-add-my-rfc-registro-federal-de-contribuyentes-to-the-invoice-i-get-for-the-purchase-of-microsoft-365-for-business"></a>2. ¿Cómo puedo agregar mi RFC (Registro Federal de contribuyentes) a la factura que obtengo para la compra de Microsoft 365 para empresas?

### <a name="during-the-purchase-process-checkout"></a>Durante el proceso de compra (checkout)

Durante el proceso de compra de pago, cuando llegue al paso **4. Pago y facturación**, seleccione la casilla para escribir una RFC. Este paso es necesario para que pueda verlo reflejado en la factura de compra.

### <a name="for-your-future-purchases"></a>Para las compras futuras

Puede Agregar o modificar la RFC para que la información se refleje en futuras compras de Microsoft 365 para empresas. Para agregar o modificar la RFC, vaya a la **Billing**  >  página <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">cuentas de facturación</a> de facturación.

> [!NOTE]
> Si se encuentra en una suscripción de facturación mensual o anual, la adición o modificación de la RFC se refleja en la factura de la siguiente tarifa.

## <a name="3-can-i-add-or-modify-my-rfc-to-an-invoice-that-was-already-generated"></a>3. ¿puedo agregar o modificar mi RFC a una factura que ya se ha generado?

Es importante que agregue o modifique la RFC antes de realizar la compra. Una vez que haya generado su factura de 365 para empresas, no podrá agregar ni modificar la información de RFC.

## <a name="4-why-dont-i-get-a-cfdi-comprobante-fiscal-digital-por-internet"></a>4. ¿por qué no obtengo un CFDI (comprobante fiscal digital por Internet)?

Microsoft Corporation, mientras que el impuesto reside en los Estados Unidos de América, sólo puede emitir facturas en virtud de las leyes de jurisdicción y no está obligado a proporcionar una factura electrónica de CFDI.

## <a name="5-is-the-invoice-i-receive-valid-for-my-accounting-operations-in-mexico"></a>5. ¿la factura que recibo es válida para mis operaciones de contabilidad en México?

La factura generada por entidades extranjeras (en este caso, Microsoft Corporation) que incluye la RFC respectiva es válida para propósitos fiscales en México. (Recomendamos consultar a su contable para obtener más información).
