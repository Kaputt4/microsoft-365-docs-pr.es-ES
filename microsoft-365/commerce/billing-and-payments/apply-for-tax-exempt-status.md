# <a name="apply-for-tax-exempt-status-for-office-365-for-business"></a>Solicitar el estado de exención de impuestos para Office 365 para empresas

Si califica para el estado de exención de impuestos en su país o región, [póngase en contacto con el soporte técnico](../../admin/contact-support-for-business-products.md) para crear una solicitud de servicio para establecer el estado de exención de impuestos para su suscripción. Deberá proporcionar la siguiente documentación cuando envíe su solicitud: 
  
|**Ubicación**|**Documentación**|
|:-----|:-----|
|Estados Unidos y Canadá  <br/> |Certificado de IVA de uso de ventas  <br/> |
|Puerto Rico  <br/> |Certificado de compra exentas  <br/> |
|Irlanda  <br/> |Certificado de exención de impuestos de 13B/56A  <br/> |
|Organizaciones internacionales que mantienen la exención de impuestos  <br/> |Confirmación de la certificación o la carta de las autoridades fiscales locales pertinentes  <br/> |
