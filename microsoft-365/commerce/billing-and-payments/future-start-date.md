# <a name="understand-invoicing-for-future-start-dates"></a>Descripción de la facturación para las fechas de inicio futuras

Las propuestas de ventas pueden incluir suscripciones que comienzan en una fecha futura. Una fecha de inicio futura alinea la fecha de inicio de una nueva suscripción con la fecha de finalización de una suscripción anterior. Si la propuesta contiene suscripciones que comienzan en una fecha futura, recibirá una factura para esas suscripciones solo cuando llegue la fecha de inicio futura. El uso de una fecha de inicio futura garantiza que no se le facturen los elementos que aún no posee. Las nuevas suscripciones comienzan en la fecha de inicio futura especificada en la propuesta. En la fecha de inicio de la nueva suscripción, le enviaremos un correo electrónico para informarle de que la nueva suscripción ya está activa y que la facturación de esta suscripción comenzará inmediatamente. La siguiente factura refleja los nuevos cargos más los impuestos aplicables para la nueva suscripción.
