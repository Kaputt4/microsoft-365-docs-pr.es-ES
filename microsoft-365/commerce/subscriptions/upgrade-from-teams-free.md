# <a name="upgrade-from-microsoft-teams-free-to-microsoft-365-for-business"></a>Actualizar de Microsoft Teams gratuito a Microsoft 365 para empresas

> [!IMPORTANT]
> Este documento solo se aplica a organizaciones gratuitas de Microsoft Teams en India.

## <a name="before-you-begin"></a>Antes de empezar

Debe ser administrador gratuito de Microsoft Teams para actualizar una organización gratuita de Microsoft Teams.Es administrador automáticamente si creó la organización gratuita de Microsoft Teams.

## <a name="step-1-get-your-work-or-school-account-id"></a>Paso 1: Obtener el id. de la cuenta del trabajo o la escuela

Para comenzar el proceso, obtén el identificador de tu cuenta de trabajo o escuela subyacente. Este identificador de cuenta es diferente de la dirección de correo electrónico que usa para acceder a Microsoft Teams Gratuito.

1. Vete a <a href="https://go.microsoft.com/fwlink/p/?linkid=2134797" target="_blank">https://account.activedirectory.windowsazure.com</a>
2. En el encabezado, seleccione el icono de perfil. El identificador de la cuenta se muestra debajo de su nombre y termina con *.onmicrosoft.com*.\
    La dirección de correo electrónico del administrador tendrá *el aspecto admin@yourfreeorgname.onmicrosoft.com.*

## <a name="step-2-reset-your-password"></a>Paso 2: Restablecer la contraseña

Una vez que tenga la cuenta de trabajo o escuela subyacente para su organización gratuita de Microsoft Teams, debe restablecer la contraseña de la cuenta.

1. Vaya al Centro <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">de administración de Microsoft 365.</a> Cuando se le pida que inicie sesión, elija **¿Ha olvidado mi contraseña?**
2. Complete el procedimiento de restablecimiento de contraseña con la cuenta de trabajo o escuela subyacente (la que termina con *.onmicrosoft.com*).
3. Complete los pasos de comprobación. Recibirá un correo electrónico con un código para establecer una nueva contraseña.
4. Escriba el código para completar el proceso de comprobación.

## <a name="step-3-buy-your-new-subscription"></a>Paso 3: Comprar la nueva suscripción

Use la cuenta de trabajo o escuela subyacente que recuperó y la nueva contraseña de administrador para comprar una nueva suscripción a Microsoft 365 para empresas.

1. En el Centro de administración de Microsoft 365, vaya a **Facturación** > <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">Servicios de compra</a>.
2. Busque el producto de Microsoft 365 para empresas que quiere comprar.
    > [!IMPORTANT]
    > Asegúrese de elegir una que incluya Microsoft Teams.
3. Completa la compra. Recuerde comprar suficientes licencias para todos los usuarios de su organización gratuita de Microsoft Teams que desea mover a Microsoft 365 para empresas.

## <a name="step-4-assign-new-licenses-to-users"></a>Paso 4: Asignar nuevas licencias a los usuarios

Ahora que tiene una nueva suscripción a Microsoft 365 para empresas, puede asignar licencias a los usuarios existentes en su organización gratuita de Microsoft Teams.

1. En el centro de administración, vaya a **Usuarios**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">activos.</a>
2. Seleccione los usuarios a los que desea asignar licencias de Microsoft 365 para empresas y, a continuación, **seleccione Administrar licencias de producto.**
3. En el **panel Administrar licencias de producto,** seleccione Reemplazar **asignaciones de licencias existentes.**
4. En el **panel Reemplazar productos** existentes, busque el producto de Microsoft 365 para empresas que compró y cambie el botón de alternancia a **On**.
5. Seleccione **Reemplazar** y, a continuación, **seleccione Cerrar**.
