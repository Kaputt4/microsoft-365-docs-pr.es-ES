# <a name="upgrade-from-microsoft-teams-free-to-microsoft-365-for-business"></a>Actualización de Microsoft Teams gratis a Microsoft 365 para empresas

> [!IMPORTANT]
> Este documento solo se aplica a organizaciones gratuitas de Microsoft Teams en India.

## <a name="before-you-begin"></a>Antes de empezar

Debe ser un administrador gratuito de Microsoft Teams para actualizar una organización gratuita de Microsoft Teams.Si ha creado la organización gratuita de Microsoft Teams, será administrador de forma automática.

## <a name="step-1-get-your-work-or-school-account-id"></a>Paso 1: obtener el identificador de la cuenta profesional o educativa

Para iniciar el proceso, obtenga su identificador de cuenta profesional o educativa subyacente. Este identificador de cuenta es diferente de la dirección de correo electrónico que usa para acceder gratuitamente a Microsoft Teams.

1. Vete a<a href="https://go.microsoft.com/fwlink/p/?linkid=2134797" target="_blank">https://account.activedirectory.windowsazure.com</a>
2. En el encabezado, seleccione el icono del perfil. El identificador de cuenta se muestra debajo de su nombre y termina con *. onmicrosoft.com*. \
    Su dirección de correo electrónico de administrador será similar a *admin@yourfreeorgname.onmicrosoft.com*.

## <a name="step-2-reset-your-password"></a>Paso 2: restablecer la contraseña

Una vez que tenga la cuenta profesional o educativa para su org gratuita de Microsoft Teams, debe restablecer la contraseña de la cuenta.

1. Vaya al <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">centro de administración de Microsoft 365</a>. Cuando se le pida que inicie sesión, elija **olvidé mi contraseña**.
2. Complete el procedimiento de restablecimiento de contraseña con la cuenta de trabajo o escuela subyacente (la que termina con *. onmicrosoft.com*).
3. Complete los pasos de verificación. Recibirá un correo electrónico con un código que se usará para establecer una nueva contraseña.
4. Escriba el código para completar el proceso de verificación.

## <a name="step-3-buy-your-new-subscription"></a>Paso 3: comprar la nueva suscripción

Use la cuenta profesional o educativa subyacente que ha recuperado y la nueva contraseña de administrador para comprar una nueva suscripción a Microsoft 365 para empresas.

1. En el centro de administración de Microsoft 365, vaya a servicios de compra de **facturación**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">Purchase Services</a>.
2. Encuentre el producto de Microsoft 365 para empresas que desea comprar.
    > [!IMPORTANT]
    > Asegúrese de elegir una que incluya a Microsoft Teams.
3. Complete la compra. No olvide comprar suficientes licencias para todos los usuarios de su org gratuita de Microsoft teams que quiera migrar a Microsoft 365 para empresas.

## <a name="step-4-assign-new-licenses-to-users"></a>Paso 4: asignar nuevas licencias a los usuarios

Ahora que tiene una nueva suscripción de Microsoft 365 para empresas, puede asignar licencias a los usuarios existentes de su organización gratuita de Microsoft Teams.

1. En el centro de administración, vaya a **usuarios**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">activos</a>.
2. Seleccione los usuarios a los que desea asignar licencias de Microsoft 365 para empresas y, a continuación, seleccione **administrar licencias de producto**.
3. En el panel **administrar licencias de producto** , seleccione **reemplazar las asignaciones de licencia existentes**.
4. En el panel **reemplazar productos existentes** , busque el producto de Microsoft 365 para empresas que compró y cambie el botón de alternancia a **activado**.
5. Seleccione **reemplazar**y, a continuación, haga clic en **cerrar**.
