# <a name="problems-with-your-microsoft-365-for-business-product-key"></a>¿Tiene problemas con la clave de producto de Microsoft 365 para empresas?

Si hay un mensaje de error al intentar canjear una clave de producto para **Microsoft 365** para empresas, pruebe estas soluciones comunes. 
  
 **¿Necesita ayuda inmediatamente?** [Llamar al Soporte técnico de Microsoft](../admin/contact-support-for-business-products.md). 
  
 Para obtener ayuda con una clave de producto de Microsoft **365 Family o Microsoft 365 Personal,** consulte Usar claves [de producto con Office.](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759.aspx)
  
## <a name="product-key-error-help-with-microsoft-365-for-business"></a>Ayuda de error de clave de producto con Microsoft 365 para empresas

| Mensaje de error al escribir la clave de producto... | Pruebe esto... |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| "Lo sentimos, se trata de una clave de producto no válida. Intente escribirlo de nuevo. Si su clave de producto es para Microsoft 365 Personal o Microsoft 365 Family, canjee por office.com/setup". <br/><br/>Si usa Office 365 Solo en Japón: "Lo sentimos, se trata de una clave de producto no válida. Intente escribirlo de nuevo. Si la clave de producto es para Office 365 Solo, canjee por office.com/setup". | Si está configurando [Microsoft 365 Familia](https://support.microsoft.com/office/28cbc8cf-1332-4f04-9123-9b660abb629e.aspx)o Personal, debe canjear su clave de producto en [https://www.office.com/setup](https://www.office.com/setup) . De lo contrario, para los clientes empresariales, compruebe cuidadosamente los números y caracteres que está especificando. |
| "La clave de producto que escribió no es válida. Intente escribirlo de nuevo". O "Esta clave de producto no es válida. Escriba una clave de producto diferente". | Compruebe cuidadosamente los números y caracteres que escribe. Se pueden provocar errores con 0 y o, 5 y S, l e I, etc. Si el problema continúa, póngase en contacto con el distribuidor donde compró la clave de producto. |
| "Ya ha escrito esta clave de producto. Escriba una clave diferente". | Compruebe las claves del producto que ya ha escrito para ver si la clave ya se ha agregado. Si está renovando una suscripción mediante una clave de producto, debe usar una clave de producto nueva y sin usar.  <br/><br/>¿Necesita comprar una clave nueva? Visite [Microsoft Store](https://go.microsoft.com/fwlink/p/?LinkId=529160) o un revendedor de terceros, o si está trabajando con un partner, póngase en contacto con él.  <br/><br/>Si el problema persiste, póngase [en contacto con el soporte técnico.](../admin/contact-support-for-business-products.md) Si estás trabajando con un partner, ponte en contacto con él para obtener soporte técnico de clave de producto. |
| "La clave de producto que escribió ha expirado. Escriba una clave diferente". | Debe usar una clave de producto nueva y sin usar para renovar una suscripción actual o activar una suscripción nueva.<br/><br/>¿Necesita comprar una clave nueva? Visite [Microsoft Store](https://go.microsoft.com/fwlink/p/?LinkId=529160) o un revendedor de terceros, o si está trabajando con un partner, póngase en contacto con él.   |
| "Esta clave de producto ya se ha usado. Escriba una clave de producto diferente". | Compruebe que la clave no se ha usado todavía por un miembro de la organización o por usted. Si la clave no se ha usado todavía, póngase en contacto con su partner o con el revendedor donde ha comprado la clave del producto. |
| "Lo sentimos, no podemos procesar su solicitud en este momento. Espere unos minutos e inténtelo de nuevo". | Si los intentos posteriores resultan en el mismo mensaje de error durante más de 15 minutos, póngase [en contacto con el soporte técnico.](../admin/contact-support-for-business-products.md) |
| "La suscripción solicitada no está disponible. Uno de los siguientes motivos podría haber causado esto: la oferta no está disponible - El servicio no está disponible en su país - Es imposible usar o seleccionar la misma versión de prueba dos veces. Si el problema persiste, póngase en contacto con el soporte técnico de Microsoft". | Póngase [en contacto con el soporte técnico.](../admin/contact-support-for-business-products.md) Si estás trabajando con un partner, ponte en contacto con él para obtener soporte técnico. |
| "Ha agregado más licencias de usuario de las que permite esta oferta. El máximo son \<x\> las licencias de usuario. Quite esta clave de producto y escriba una que agrega menos licencias de usuario". | Póngase en contacto con su distribuidor o socio. Ha adquirido más licencias de las que se pueden usar con esta suscripción a Microsoft 365. |
| "Debe ser administrador global o de facturación para canjear una clave de producto". | Asegúrese de que los permisos están establecidos como administrador global o de facturación. Para comprobarlo,  [inicie sesión en Microsoft 365](https://support.microsoft.com/office/e9eb7d51-5430-4929-91ab-6157c5a050b4) con su cuenta de trabajo o escuela y vaya al centro de administración. <br/><br/>En el Centro de administración, seleccione **Usuarios** \> **Usuarios activos**. En **Filtros**, elija **Administradores globales** o **Administradores de facturación**.  <br/><br/>Asegúrese de que aparece en los resultados. |
   
> [!NOTE]
> Cuando compra Microsoft 365 a través de un partner, Microsoft le envía un correo electrónico con instrucciones para obtener la clave de producto y activar la suscripción en el Centro de servicios de licencias [por volumen.](https://go.microsoft.com/fwlink/p/?LinkID=282016) Para obtener más información sobre las licencias por volumen, consulte [ayuda para los clientes de licencias por volumen existentes](https://go.microsoft.com/fwlink/p/?LinkId=534992).
