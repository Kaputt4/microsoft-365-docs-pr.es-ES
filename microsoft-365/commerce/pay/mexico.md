# <a name="payment-information-by-deposit-or-electronic-transfer-only-for-mexico"></a>Información de pago mediante depósito o transferencia electrónica (solo para México)

[Buscar otro país o región](../billing-and-payments/pay-for-your-subscription.md).

Para pagar mediante depósito o transferencia electrónica:

1. Seleccione en el momento de la desprotección el método de pago "Factura (pago por cheque o transferencia bancaria)". En las próximas 24 a 72 horas recibirá la factura del producto en el correo electrónico registrado como contacto. La factura incluirá las instrucciones para completar el pago.

**Código SWIFT:** BNMXMM  
**CLABE** 002180002337160225  
**Número de cuenta:** 0233716022  
**Nombre de cuenta:** Microsoft Payments Inc.  
**Moneda:** Pesos mexicanos

## <a name="important-information"></a>Información importante

1. El pago mediante depósito o transferencia electrónica solo está disponible para importes superiores a 3.500 USD MXN.
2. El pago debe realizarse en un plazo de 30 días; de lo contrario, el pedido se cancelará automáticamente.
