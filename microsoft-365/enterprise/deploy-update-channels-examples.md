# <a name="deployment-and-update-channel-example-configurations"></a>Ejemplo de configuraciones de canal de implementación y actualización

La elección de los canales de actualización que se usarán para las aplicaciones de Windows 10 y Microsoft 365 pueden depender del tipo de organización y en función del ciclo de desarrollo en el que quiera implementar y usar nuevas características y funciones. Busque los canales de producción y prelanzamiento que se adapten mejor a sus necesidades.

## <a name="pre-release-channels"></a>Canales de prelanzamiento

| Oferta de clientes y canales | Windows 10 | Aplicaciones de Microsoft 365 para empresas (Windows 10) |
|:-------|:-------|:-----|
| Ideal para usuarios y programadores altamente técnicos. <br><br> Sea el primero en tener acceso a las compilaciones más recientes más pronto en el ciclo de desarrollo con el código más reciente. <br><br> Se aplicarán bordes irregulares y cierta inestabilidad. | Desarrollador | N/D |
| Adecuada para los primeros y los profesionales de TI que quieren compilaciones más confiables que siguen en desarrollo. <br><br> Vea lo que sigue y ayude a validar las nuevas características. | Canal beta | Canal beta |
| Adecuada para las personas que quieren tener acceso anticipado a las versiones futuras. <br><br> Dónde se obtiene una vista previa de las publicaciones para ampliar la implementación. <br><br> Son compatibles. <br>  | Versión preliminar | Canal actual (vista previa) <br><br> Canal empresarial semianual (versión preliminar)|
||||

| Oferta de clientes y canales | Windows 10 | Aplicaciones de Microsoft 365 para empresas (Windows 10) | Ejemplo |
|:-------|:-------|:-----|:-------|

| Apropiado para empresas con necesidad de realizar extensas pruebas de TI antes de cada actualización. | Canal semianual | [Canal Empresarial Semianual](https://docs.microsoft.com/deployoffice/overview-update-channels#semi-annual-enterprise-channel-overview) |  |
