# <a name="step-4-simplify-password-resets"></a>Paso 4: Simplificar los restablecimientos de contraseña

*Este paso es opcional y es válido para las versiones E3 y E5 de Microsoft 365 Enterprise*

![](./media/deploy-foundation-infrastructure/identity_icon-small.png)

En este paso, habilitará el autoservicio de restablecimiento de contraseña (SSPR) para permitir que los usuarios restablezcan o desbloqueen sus contraseñas o cuentas. Para avisarle de usos indebidos o abusos, puede usar los informes detallados, que realizan un seguimiento del acceso de los usuarios al sistema, además de enviar notificaciones.

Vea las [instrucciones para habilitar el restablecimiento de contraseña](https://docs.microsoft.com/azure/active-directory/active-directory-passwords-best-practices).
