# <a name="plan-for-microsoft-365-enterprise"></a>Plan para Microsoft 365 Enterprise

Al mover una organización empresarial a Microsoft 365, es importante planear con antelación y tomar las decisiones de diseño clave que simplificarán la implementación de TI y la adopción de usuarios. 

## <a name="planning-with-microsoft-365-fasttrack"></a>Planificación con Microsoft 365 FastTrack

[FastTrack para Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) es el mejor método para obtener ayuda de Microsoft para planear la implementación de Microsoft 365. FastTrack puede ayudarle a través de las consideraciones de diseño más comunes y puede responder preguntas en el camino. 

>[!Note]
>También puede obtener ayuda de un partner [de Microsoft.](https://www.microsoft.com/solution-providers/home)
>

## <a name="do-it-yourself-planning-for-microsoft-365"></a>Planificar usted mismo para Microsoft 365

Para planear Microsoft 365 por su cuenta, pase por las decisiones de planeación y diseño para estas áreas:

- Su inquilino de Microsoft 365

  Incluye la planeación de las conexiones de red a Internet, las identidades de Microsoft 365 y la integración con aplicaciones, locales, Azure y otros elementos. Para obtener más información, vaya a [Suscripciones, licencias,](subscriptions-licenses-accounts-and-tenants-for-microsoft-cloud-offerings.md) cuentas e inquilinos para obtener más información.

- Compatibilidad con los clientes

  Incluye autenticación basada en certificados, administración de dispositivos móviles, opciones de autenticación y colaboración entre inquilinos. Vaya a [Autenticación basada en certificados](microsoft-365-client-support-certificate-based-authentication.md) para obtener más información.

- Compatibilidad con la autenticación moderna híbrida

  Incluye la planeación de la autenticación moderna al usar configuraciones híbridas de aplicaciones clave de productividad de Microsoft 365. Vaya a Introducción a la autenticación moderna híbrida y a [los requisitos previos](hybrid-modern-auth-overview.md) para obtener más información.

- Clientes y servidores de Office más antiguos

  Incluye información de migración para productos de cliente y servidor de Office 2007 y Office 2010. Vaya a [Planear la actualización](plan-upgrade-previous-versions-office.md) para obtener más información.

También puede iniciar sesión en su suscripción de Microsoft 365 y usar las guías de configuración para los servicios de [Microsoft 365 y Office 365.](setup-guides-for-microsoft-365.md)
