# <a name="cmdlet-references-for-microsoft-365-services"></a>Referencias de cmdlets para los servicios de Microsoft 365 

*Este artículo afecta tanto a Office 365 Enterprise como a Microsoft 365 Enterprise*

En este artículo se proporcionan referencias de cmdlet para los distintos servicios de Microsoft 365 e instrucciones de conexión para cada servicio de Microsoft 365 compatible con PowerShell.
  
> [!NOTE]
> Para conectarse a todos los servicios a la vez, consulte Conectarse a todos los servicios de [Microsoft 365](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)en una Windows PowerShell única.
  
## <a name="azure-active-directory-powershell-cmdlets"></a>Cmdlets de PowerShell de Azure Active Directory

Los temas de referencia del cmdlet de Azure Active Directory PowerShell para Graph se encuentran en la sección Referencia de la documentación de [Azure Active Directory PowerShell para Graph.](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)

El Módulo Azure Active Directory para Windows PowerShell temas de referencia de cmdlets se encuentra en la sección Referencia de la documentación de [Azure Active Directory (MSOnline).](https://docs.microsoft.com/powershell/azure/active-directory/overview?view=azureadps-1.0)

Para obtener instrucciones de conexión de PowerShell de Microsoft 365, vea Conectarse a [Microsoft 365 con PowerShell.](connect-to-microsoft-365-powershell.md)
  
## <a name="exchange-online-powershell-cmdlets"></a>Cmdlets de Exchange Online PowerShell

Los temas de referencia de cmdlets de Exchange Online se encuentran en la sección Referencia de la [documentación de Exchange Online PowerShell.](https://docs.microsoft.com/powershell/exchange/exchange-online-powershell)
  
Para obtener instrucciones de conexión para Exchange Online PowerShell, consulte [Conectarse a Exchange Online PowerShell.](https://go.microsoft.com/fwlink/p/?LinkId=396554)
  
> [!NOTE]
> Los cmdlets de informes para otros servicios, como SharePoint Online, Skype Empresarial Online y la actividad de usuario de Microsoft 365, están disponibles en Exchange Online PowerShell. Para obtener más información, consulte [Cmdlets de informes en Exchange Online](https://go.microsoft.com/fwlink/p/?LinkId=691595). 
  
## <a name="sharepoint-online-powershell-cmdlets"></a>Cmdlets de PowerShell de SharePoint Online

Para los cmdlets de SharePoint Online, vea [Index of Windows PowerShell for SharePoint Online cmdlets](https://go.microsoft.com/fwlink/p/?LinkId=691476).
  
Para obtener instrucciones de conexión para SharePoint Online PowerShell, vea Configurar el shell de administración [de SharePoint Online Windows PowerShell entorno.](https://go.microsoft.com/fwlink/p/?LinkId=691603)
  
## <a name="skype-for-business-online-powershell-cmdlets"></a>Cmdlets de PowerShell de Skype Empresarial Online

For Skype for Business Online cmdlet reference topics, see [Skype for Business Online cmdlets](https://technet.microsoft.com/library/mt228132.aspx).
  
Para obtener instrucciones de conexión para PowerShell de Skype Empresarial Online, consulte [Administrar Skype Empresarial Online con PowerShell.](manage-skype-for-business-online-with-microsoft-365-powershell.md)

## <a name="security--compliance-center-powershell-cmdlets"></a>Cmdlets de PowerShell del Centro de & seguridad y cumplimiento

Las referencias de cmdlet & Centro de seguridad y cumplimiento se encuentran en la sección Referencia de la documentación de [PowerShell de seguridad & cumplimiento.](https://docs.microsoft.com/powershell/exchange/scc-powershell)
  
Para obtener instrucciones de conexión para PowerShell & Centro de seguridad y cumplimiento, vea Conectarse al [PowerShell del](https://docs.microsoft.com/powershell/exchange/connect-to-scc-powershell)Centro de & seguridad y cumplimiento.
