# <a name="cmdlet-references-for-microsoft-365-services"></a>Referencias de cmdlets para los servicios de Microsoft 365

*Este artículo afecta tanto a Office 365 Enterprise como a Microsoft 365 Enterprise*

En este artículo se proporcionan referencias de cmdlet para los distintos servicios de Microsoft 365 e instrucciones de conexión para cada servicio de Microsoft 365 compatible con PowerShell.

> [!NOTE]
> Para conectarse a todos los servicios a la vez, consulte [Conexión a todos los servicios de Microsoft 365 en una sola ventana Windows PowerShell](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md).

## <a name="azure-active-directory-powershell-cmdlets"></a>Cmdlets de PowerShell de Azure Active Directory

Los temas de referencia del cmdlet de PowerShell para Graph de Azure Active Directory se encuentran en la sección Referencia de la [documentación de PowerShell para Graph de Azure Active Directory](/powershell/azure/active-directory/install-adv2).

El módulo de Azure Active Directory para Windows PowerShell temas de referencia de cmdlets se encuentra en la sección Referencia de la [documentación de Azure Active Directory (MSOnline).](/powershell/azure/active-directory/overview)

Para obtener instrucciones de conexión de PowerShell de Microsoft 365, consulte [Conexión a Microsoft 365 con PowerShell](connect-to-microsoft-365-powershell.md).

## <a name="exchange-online-powershell-cmdlets"></a>Cmdlets de Exchange Online PowerShell

Exchange Online temas de referencia de cmdlets se encuentran en la sección Referencia de la [documentación de Exchange Online PowerShell](/powershell/exchange/exchange-online-powershell).

Para obtener instrucciones de conexión para Exchange Online PowerShell, consulte [Conexión a Exchange Online PowerShell](/powershell/exchange/connect-to-exchange-online-powershell).

> [!NOTE]
> Los cmdlets de informes para otros servicios, como SharePoint Online, Skype Empresarial Online y la actividad de usuario de Microsoft 365, están disponibles en Exchange Online PowerShell. Para obtener más información, consulte [Cmdlets de informes en Exchange Online](/powershell/exchange/exchange-online-powershell).

## <a name="sharepoint-online-powershell-cmdlets"></a>Cmdlets de PowerShell de SharePoint Online

Para cmdlets de SharePoint Online, vea [Índice de Windows PowerShell para cmdlets de SharePoint Online](/powershell/module/sharepoint-online/).

Para obtener instrucciones de conexión para SharePoint Online PowerShell, vea [Configurar el entorno de Windows PowerShell del Shell de administración de SharePoint Online](/powershell/sharepoint/sharepoint-online/connect-sharepoint-online).

## <a name="skype-for-business-online-powershell-cmdlets"></a>Cmdlets de PowerShell de Skype Empresarial Online

Para obtener Skype Empresarial temas de referencia de cmdlets en línea, consulte [cmdlets de Skype Empresarial Online](/previous-versions//mt228132(v=technet.10)).

Para obtener instrucciones de conexión para Skype Empresarial PowerShell en línea, consulte [Administración de Skype Empresarial en línea con PowerShell](manage-skype-for-business-online-with-microsoft-365-powershell.md).

## <a name="security--compliance-powershell-cmdlets"></a>Cmdlets de PowerShell de cumplimiento de & seguridad

Las referencias de cmdlet de Security & Compliance Center se encuentran en la sección Referencia de la [documentación de PowerShell de cumplimiento de seguridad &](/powershell/exchange/scc-powershell).

Para obtener instrucciones de conexión para PowerShell de cumplimiento de & de seguridad, consulte [Conexión a PowerShell de cumplimiento de seguridad &](/powershell/exchange/connect-to-scc-powershell).
