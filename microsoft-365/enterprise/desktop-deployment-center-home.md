# <a name="desktop-deployment-center"></a>Centro de implementación de escritorios

<img src="media/desktop-deployment-center-home-media/desktop-deployment-center-home-media-2.png" alt="Desktop Deployment Wheel" height="450" width="802" align="middle" style="background-color: #fff;" />

Siga los siguientes pasos para planear y llevar a cabo la implementación a gran escala de Windows 10 y Office 365 ProPlus. Cada paso debajo es parte de la planificación general y el proceso de implementación con pasos que normalmente se ejecutan en paralelo entre sí en una implementación por fases. Descargue el software gratuito [Kit de laboratorio de implementación y administración de escritorio](https://aka.ms/howtoshiftlabs) para obtener aprendizaje para las herramientas que se resaltaron en el proceso de implementación. También puede [buscar ayuda](https://aka.ms/mddhelp) para la implementación de escritorio de partners de Microsoft y servicios de FastTrack.
