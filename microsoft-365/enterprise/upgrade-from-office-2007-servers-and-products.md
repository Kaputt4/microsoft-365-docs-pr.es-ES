# <a name="resources-to-help-you-upgrade-from-office-2007-servers-and-clients"></a>Recursos para ayudarle a actualizar desde Office servidores y clientes de 2007

*Este artículo afecta tanto a Office 365 Enterprise como a Microsoft 365 Enterprise*

Si usa productos y servicios Office 2007, tenga en cuenta que la compatibilidad con estas aplicaciones ha finalizado. Considere la posibilidad de pasar a la nube de Microsoft, empezando por Microsoft 365. Use este artículo como punto de partida para revisar las opciones y planear la actualización.
      
## <a name="office-2007-planning-roadmaps"></a>Office plan de planificación de 2007
  
El soporte técnico ha finalizado Office productos y servicios de 2007. Las siguientes hojas de ruta pueden ayudarle a planear la actualización ahora.

|**Planeación del contenido**|**Fecha de finalización del soporte técnico**|
|:-----|:-----|
|[Office plan de fin de soporte técnico de 2007](/DeployOffice/office-2007-end-support-roadmap) <br/> |10 de octubre de 2017  <br/> |
|[Plan de fin del soporte técnico de Exchange 2007](exchange-2007-end-of-support.md) <br/> |11 de abril de 2017  <br/> |
|[Plan de fin del soporte técnico de SharePoint Server 2007](sharepoint-2007-end-of-support.md) <br/> |10 de octubre de 2017  <br/> |
|[Plan de fin del soporte técnico de Project Server 2007](project-server-2007-end-of-support.md) <br/> |10 de octubre de 2017  <br/> |
|[Actualización desde Office Communications Server](/SkypeForBusiness/plan-your-deployment/upgrade) <br/> |8 de enero de 2018  <br/> |
|[Plan de fin del soporte técnico de PerformancePoint Server 2007](pps-2007-end-of-support.md) <br/> |9 de enero de 2018  <br/> |
   
Una vez que el soporte técnico finaliza para un producto de Microsoft, ya no hay más:
- Nuevas actualizaciones de seguridad
- Nuevas actualizaciones que no son de seguridad
- Opciones de soporte técnico asistido gratuitas o de pago disponibles (incluidos los contratos de soporte técnico personalizados)
- Nuevas actualizaciones de contenido técnico en línea

Independientemente de si es una organización empresarial o un usuario particular, tiene varias opciones que tener en cuenta.
