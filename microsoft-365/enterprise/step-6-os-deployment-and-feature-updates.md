## <a name="windows-imaging-process"></a><span data-ttu-id="b13ff-115">Proceso de creación de imágenes de Windows</span><span class="sxs-lookup"><span data-stu-id="b13ff-115">Windows Imaging Process</span></span>

<span data-ttu-id="b13ff-p104">Muchas organizaciones usan el proceso de creación de imágenes de equipo para configurar y capturar un duplicado de Windows, incluido un conjunto de algunas aplicaciones estándar instaladas, o incluso una imagen más reducida solo con los tiempos de ejecución y las actualizaciones de las aplicaciones. <span data-ttu-id="b13ff-p105">Si se decide por la captura de la imagen, se recomienda automatizar la mayor parte posible para garantizar la imagen de mejor calidad y un proceso que se pueda repetir. Para la mayoría de las implementaciones, también se recomienda incluir la menor cantidad de personalización posible y de aplicaciones preinstaladas en la imagen de Windows antes de la captura. Es lo que se denomina un enfoque de "imagen ligera", que puede ahorrar ancho de banda general en la red mediante la eliminación del número de aplicaciones en la imagen. This is what is called a ‘thin image’ approach, which can save overall bandwidth on the network by eliminating the number of apps within the image. By starting with a thin base image, you can layer on required apps, languages and configurations dynamically tailored to users.</span></span>

<span data-ttu-id="b13ff-122">Durante el proceso de compilación y captura, herramientas como Microsoft Endpoint Configuration Manager (rama actual) y Microsoft Deployment Toolkit usan la Herramienta de preparación del sistema (Sysprep), junto con el comando "Generalizar" para volver a sellar la imagen antes de capturar la instalación de Windows 10 como una imagen.</span><span class="sxs-lookup"><span data-stu-id="b13ff-122">During the build and capture process, tools like Microsoft Endpoint Configuration Manager (Current Branch) and the Microsoft Deployment Toolkit use the System Preparation Tool – or Sysprep – along with the “Generalize” command to reseal your image before they capture the Windows 10 installation as an image.</span></span>

<span data-ttu-id="b13ff-p106">La imagen capturada tendrá el formato de imagen de Windows, o WIM, como los medios de instalación estándar de Windows.
