# <a name="deploy-microsoft-365-enterprise-for-your-organization"></a>Implementar Microsoft 365 Enterprise para su organización

¿Está listo para implementar Microsoft 365 Enterprise, integrarlo con la infraestructura local y permitir que los empleados aprovechen la productividad y colaboración basada en la nube?

En este artículo se describen dos opciones de implementación, tanto si desea ayuda como si lo está haciendo por su cuenta. En cada caso, Microsoft proporciona una ruta de acceso guiada que puede seguir para que la implementación se realice correctamente.

## <a name="guided-microsoft-365-enterprise-setup-process-with-fasttrack"></a>Proceso Microsoft 365 Enterprise configuración guiada con FastTrack

**[FastTrack para Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365)** es el mejor método para implementar Microsoft 365. FastTrack le guiará por las configuraciones de implementación más comunes e irá respondiendo sus preguntas sobre la marcha. 

Para obtener acceso a una serie de guías de instalación, inicie sesión en su Microsoft 365 suscripción y, a continuación, vaya a [Instrucciones de instalación](https://aka.ms/o365fasttrack).

>[!Note]
>También puede obtener ayuda de un partner [de Microsoft](https://www.microsoft.com/solution-providers/home).
>

## <a name="do-it-yourself-guided-deployment-of-microsoft-365-enterprise"></a>Implementación guiada por usted mismo de Microsoft 365 Enterprise

La implementación Microsoft 365 Enterprise por su cuenta requiere que realice una investigación detallada para tomar las decisiones de diseño que agilizan la configuración Microsoft 365 servicio y la adopción del usuario. Inicie la planeación [aquí](get-your-organization-ready-for-office-365.md).

Para implementar Microsoft 365 Enterprise por su cuenta después de completar la planeación, estos son los pasos recomendados.

1. [Configurar la red](set-up-network-for-microsoft-365.md)

   Incluye agregar los dominios de Internet y optimizar el rendimiento de la red para los usuarios locales.
 
2. [Configurar las identidades](protect-your-global-administrator-accounts.md)

   Incluye determinar un modelo de identidad (solo en la nube o híbrido) y, para la identidad híbrida, configurar la sincronización de directorios entre los Servicios de dominio de Active Directory locales (AD DS) y la suscripción Microsoft 365 cliente.

3. [Implementar la seguridad](/office365/securitycompliance/security-roadmap)

   Incluye la configuración y la instalación de protecciones básicas y mejoradas de seguridad, amenazas e información para el inquilino y las identidades en los primeros 30 días, 90 días y posteriores.
 
4. [Implementar software cliente](/DeployOffice/deployment-guide-microsoft-365-apps)

   Incluye la implementación de Aplicaciones Microsoft 365 para empresas (anteriormente denominado Office 365 ProPlus), la versión actualizada en la nube y siempre actual del conjunto de Office (Word, Excel, PowerPoint y otros) en los dispositivos. Cada Microsoft 365 de cliente incluye una licencia para Aplicaciones Microsoft 365 para empresas.
 
5. [Configurar la administración de dispositivos móviles](https://support.office.com/article/set-up-mobile-device-management-mdm-in-office-365-dd892318-bc44-4eb1-af00-9db5430be3cd)

   Microsoft 365 Enterprise incluye funcionalidades de administración de dispositivos móviles que te ayudan a proteger y administrar los dispositivos móviles de los usuarios.
 
6. [Configurar servicios y aplicaciones](configure-services-and-applications.md)

   Incluye información sobre la migración de los datos y vínculos a artículos que te ayudarán a empezar a trabajar en aplicaciones clave de productividad de Microsoft 365 como Exchange Online, SharePoint Online y Teams.
 
7. [Proporcionar formación a los usuarios](/office365/admin/admin-overview/get-started-with-office-365#training-resources-for-your-users)

   Incluye vídeos cortos que ayudan a los usuarios a sacar el máximo partido de Microsoft 365 rápidamente.
 

>[!Note]
>Estos pasos también son para empresas y organizaciones sin ánimo [de](https://go.microsoft.com/fwlink/?LinkId=627221) lucro que desean empezar con una implementación personalizada de Microsoft 365 Enterprise.
