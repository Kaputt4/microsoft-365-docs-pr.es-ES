## <a name="why-you-cannot-load-test-sharepoint-online"></a><span data-ttu-id="b1d62-120">Por qué no se puede cargar la prueba de SharePoint Online</span><span class="sxs-lookup"><span data-stu-id="b1d62-120">Why you cannot load test SharePoint Online</span></span>
<span data-ttu-id="b1d62-121">Con entornos locales, las pruebas de carga se usan para validar la suposición de escala y, en última instancia, encontrar el punto de quieje de una granja de servidores; saturando con carga.</span><span class="sxs-lookup"><span data-stu-id="b1d62-121">With on-premises environments, load testing is used to validate scale assumption and ultimately find the breaking point of a farm; by saturating it with load.</span></span> 

<span data-ttu-id="b1d62-122">Con SharePoint Online debemos hacer las cosas de forma diferente porque la escala es relativamente fluida y se ajusta, se cargan los aceleradores y los controles, en función de cierta heurística.</span><span class="sxs-lookup"><span data-stu-id="b1d62-122">With SharePoint Online we need to do things differently because the scale is relatively fluid and adjusts, throttles and controls load, based on certain heuristics.</span></span> <span data-ttu-id="b1d62-123">Al ser un entorno de varios inquilinos a gran escala, debemos proteger todos los inquilinos de la misma granja de servidores, por lo que limitaremos automáticamente las pruebas de carga.</span><span class="sxs-lookup"><span data-stu-id="b1d62-123">Being such a large scale multi-tenant environment, we must protect all tenants in the same farm, so we will automatically throttle any load tests.</span></span> <span data-ttu-id="b1d62-124">Sin embargo, si intenta realizar la prueba de carga, además de limitarse, recibirá resultados confusos y potencialmente engañosos, ya que es probable que la granja de servidores que probó hoy haya tenido cambios de escala durante la ventana de prueba o horas después de las pruebas, ya que las acciones de equilibrio de la granja de servidores y la escala se realizan de forma continua.</span><span class="sxs-lookup"><span data-stu-id="b1d62-124">If you do however attempt to load test, besides being throttled, you will receive disappointing and potentially misleading results because the farm you tested today will probably have had scale changes during the testing window or within hours after testing, as scale and farm balancing actions are performed on an on-going basis.</span></span>
