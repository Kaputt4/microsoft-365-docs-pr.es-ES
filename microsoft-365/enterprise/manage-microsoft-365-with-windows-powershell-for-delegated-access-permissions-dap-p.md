# <a name="how-to-manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-partners"></a>Cómo administrar Microsoft 365 con Windows PowerShell para los asociados de permisos de acceso delegado

*Este artículo se aplica tanto a Microsoft 365 Enterprise como a Office 365 Enterprise.*

Los asociados con permiso de acceso delegado (DAP) son asociados de sindicación y proveedor de soluciones en la nube (CSP). Muchos son proveedores de red o de telecomunicaciones. Incluyen las suscripciones de Microsoft 365 en sus ofertas de servicio. Cuando venden una suscripción a Microsoft 365, se les conceden automáticamente permisos de administración en nombre de (AOBO) a los inquilinos del cliente para que puedan administrar e informar sobre dichos arrendamientos. Estas tareas son difíciles de hacer en el centro de administración de Microsoft 365. Es mucho más fácil usar PowerShell para Microsoft 365 para realizar tareas administrativas como:
- Enumerar todos los **TenantIds** del cliente y sus dominios 
- Identificar todos los usuarios en una arrendamiento de clientes y sus licencias asignadas
> [!NOTE]
> Algunas tareas administrativas solo pueden realizarse en PowerShell.
