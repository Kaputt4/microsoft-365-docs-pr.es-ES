# <a name="tune-skype-for-business-online-performance"></a>Optimización del rendimiento de Skype Empresarial Online

En este artículo se proporcionan vínculos para administradores del sistema que desean planear los requisitos de ancho de banda de red o que necesitan mejorar el rendimiento de Skype Empresarial Online. 
  
## <a name="fine-tuning-skype-for-business-online-performance"></a>Ajuste de Skype Empresarial rendimiento en línea

- **Para obtener información sobre la configuración** y las métricas para mantener la calidad de las llamadas para su organización, es importante leer Calidad de medios y Rendimiento de conectividad de red en Skype Empresarial [Online](/skypeforbusiness/optimizing-your-network/media-quality-and-network-connectivity-performance). En este artículo se proporciona información, si desea usar ExpressRoute, sobre los tipos de implementación y otra información que necesitará.
    
- Para las implementaciones locales e híbridas Skype Empresarial Server, puede ver los requisitos de ancho de banda para las **conferencias** de acceso telefónico local e híbrido, vea [Plan for dial-in conferencing in Skype Empresarial Server 2015](/skypeforbusiness/plan-your-deployment/conferencing/dial-in-conferencing).
    
## <a name="more-ways-to-improve-skype-for-business-online-performance"></a>Más formas de mejorar el Skype Empresarial online

- [ExpressRoute y QoS en Skype Empresarial Online:](/skypeforbusiness/optimizing-your-network/expressroute-and-qos-in-skype-for-business-online)Conectar a Office 365 a través de una conexión de red dedicada mediante Azure ExpressRoute para Office 365 y Skype Empresarial Online. 
    
- Activar y usar el Panel de calidad de llamadas en [Skype Empresarial Online:](/SkypeForBusiness/using-call-quality-in-your-organization/turning-on-and-using-call-quality-dashboard)el Panel de calidad de llamadas (CQD) permite supervisar y obtener información sobre la calidad de las llamadas realizadas mediante Skype Empresarial servicios.
