# <a name="plan-your-upgrade-from-office-2007-or-office-2010-servers-and-clients"></a>Planear la actualización desde servidores y clientes de Office 2007 o 2010

*Este artículo se aplica tanto a Microsoft 365 para Enterprise como a Office 365 para Enterprise.*

Si su organización usa versiones anteriores de Office productos y servidores, ahora es un buen momento para empezar a planear la actualización. Office productos y servicios de 2007 han llegado [a su fin de soporte técnico](upgrade-from-office-2007-servers-and-products.md). Para Office productos y servicios de 2010:

- Office 2010 y Exchange 2010 llegaron a su fin el *13 de octubre de 2020*. 
- Se llegará al final del soporte técnico para SharePoint 2010 y Project Server 2010 el *13 de abril de 2021*. 

Para obtener más información, consulte [Actualización desde servidores y clientes de Office 2010](upgrade-from-office-2010-servers-and-products.md).

Use los recursos de este artículo para empezar a actualizar.

## <a name="what-is-microsoft-365"></a>¿Qué es Microsoft 365?

[Microsoft 365](https://www.microsoft.com/microsoft-365) es una combinación de aplicaciones Office innovadoras, servicios en la nube inteligentes y seguridad de primera clase diseñada para ayudarle a lograr más.

Microsoft 365 incluye las licencias y funcionalidades para ayudar a garantizar que su organización está trabajando en el sistema operativo Windows más reciente. También garantiza que los dispositivos Windows, iOS y Android estén inscritos y protegidos por directivas que requieren autenticación y protección de datos. Además, Windows 10 y el Aplicaciones Microsoft 365 para Enterprise software cliente (anteriormente denominado Office 365 ProPlus) se actualizan continuamente para incluir las características y actualizaciones de seguridad más recientes.
  
Microsoft 365 es la manera de transformar digitalmente su negocio con dispositivos y experiencias de productividad que mejoran constantemente y que están habilitados y protegidos por la nube de Microsoft.

## <a name="what-happens-if-i-dont-upgrade"></a>¿Qué ocurre si no actualo?

Puede optar por no actualizar en este momento. Los servidores y las aplicaciones locales seguirán ejecutándose. Pero cuando ya no recibe actualizaciones de seguridad ni opciones de soporte técnico, su organización podría ser vulnerable a infracciones de seguridad. Se recomienda encarecidamente planear la actualización pronto. Puede actualizar a Microsoft 365 o a versiones más recientes de las aplicaciones y servidores locales.

## <a name="what-upgrade-options-are-available"></a>¿Qué opciones de actualización están disponibles?      

Las organizaciones deben tener en cuenta varias opciones de actualización:

- **Actualice los servidores y las aplicaciones locales.** Si usa Office productos y aplicaciones de servidor locales, consulte el siguiente contenido de planeamiento:<br/> 

  |productos y servicios de Office 2007|productos y servicios de Office 2010|
  |---|---|
  |[Office 2007](/DeployOffice/office-2007-end-support-roadmap) (escritorio)|[Office 2010](/DeployOffice/office-2010-end-support-roadmap) (Escritorio)|
  |[Exchange 2007](exchange-2007-end-of-support.md)|[Exchange 2010](exchange-2010-end-of-support.md)|
  |[SharePoint 2007](sharepoint-2007-end-of-support.md)|[SharePoint 2010](upgrade-from-sharepoint-2010.md)|
  |[Office Communications Server](/skypeforbusiness/plan-your-deployment/upgrade)|[Lync Server 2010](/skypeforbusiness/plan-your-deployment/upgrade)|
  |[Project Server 2007](project-server-2007-end-of-support.md)|[Project Server 2010](project-server-2010-end-of-support.md)|
  |[PerformancePoint Server 2007](pps-2007-end-of-support.md)||
 
- **Implemente una solución híbrida con Microsoft 365 o Office 365.** Una solución híbrida usa los servidores y aplicaciones locales y sus equivalentes en la nube. Si va a pasar a la nube en fases o debe mantener algunos servidores y aplicaciones locales, una solución híbrida podría ser adecuada para su organización. Para obtener más información, consulte [Modelos de arquitectura en la nube de Microsoft](../solutions/cloud-architecture-models.md). 
    
- **Vaya a la nube con Microsoft 365 o Office 365.** Para muchos clientes, migrar a la nube es una solución eficaz y rentable. Un traslado completo a la nube facilita la configuración y la administración continua. Esta opción proporciona sin problemas todas las características y actualizaciones de seguridad más recientes. Para obtener más información, consulte la sección [¿Qué es Microsoft 365?](#what-is-microsoft-365) de este artículo.
    
## <a name="can-i-get-help-for-my-organization"></a>¿Puedo obtener ayuda para mi organización?

Si desea ayuda para planear la actualización, considere una o varias de las siguientes opciones:

- Trabaje con un asociado o especialista en licencias por volumen. [Busque su partner o revendedor de Microsoft 365](https://support.office.com/article/b6c18a9b-2aed-4c84-9d75-af709160258c.aspx). 

- Si su organización compra un número calificado de licencias de Microsoft 365, nuestro equipo de FastTrack puede ayudarle a través del proceso de configuración. Para obtener más información, consulte [FastTrack para Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365).

- Si forma parte de una organización pequeña, o si prefiere controlar la actualización Office de su organización, consulte [Actualización de los usuarios de Microsoft 365 Business al cliente de Office más reciente](/office365/admin/setup/upgrade-users-to-latest-office-client). 
  
## <a name="im-a-home-user-what-do-i-do"></a>Soy un usuario doméstico. ¿Qué hago?

Si usa Office 2007 o Office 2010 en casa, tenga en cuenta las siguientes opciones de actualización:

- **Use Office en un explorador de forma gratuita.** Cree, vea y edite archivos Office en el explorador. Obtenga acceso a esos archivos desde casi cualquier dispositivo que tenga acceso a Internet. 

  [Office en la Web](https://products.office.com/office-online/documents-spreadsheets-presentations-office-online) incluye [Word para la Web](https://go.microsoft.com/fwlink/p/?linkid=746664), [Excel para la Web](https://go.microsoft.com/fwlink/p/?linkid=746665), [PowerPoint para la Web](https://go.microsoft.com/fwlink/p/?linkid=746666), [OneNote para la Web](https://go.microsoft.com/fwlink/p/?linkid=746674)[, Sway](https://go.microsoft.com/fwlink/p/?linkid=746675) , [correo electrónico](https://go.microsoft.com/fwlink/p/?linkid=746676), [calendario](https://go.microsoft.com/fwlink/p/?linkid=746678) y [OneDrive](https://go.microsoft.com/fwlink/p/?linkid=746679). Para empezar, visite [Office.com](https://office.com) e inicie sesión con su [cuenta de Microsoft](https://account.microsoft.com/account). Si no tiene una cuenta microsoft, puede crear una en [Office.com](https://office.com).

- **Pruebe Microsoft 365 Familia.** Inicie una prueba de [Microsoft 365 Familia](https://www.microsoft.com/microsoft-365/p/microsoft-365-family/cfq7ttc0k5dm?rtc=2&activetab=pivot:overviewtab) para ver cómo funciona para usted. Con Microsoft 365 Familia, podrá disfrutar del almacenamiento en la nube con OneDrive.

  La compatibilidad con Windows 7 [finalizó el 14 de enero de 2020](https://www.microsoft.com/microsoft-365/windows/end-of-windows-7-support). Las versiones de Word, Excel, PowerPoint, Outlook, Publisher y Access que se proporcionan con Office 365 Home o Office 365 Personal y que se ejecutan en Windows 7 dispositivos reciben actualizaciones de seguridad, pero no actualizaciones de características. Para seguir recibiendo actualizaciones de características para estas aplicaciones, [actualice los Windows 7 dispositivos a Windows 10](https://support.microsoft.com/help/12435/windows-10-upgrade-faq).
    
- **Compra Office Home &amp; Student.** Si eliges esta opción, realizas una compra única y luego instalas Office en tu pc Windows o Mac. Esta compra no es una suscripción; es una licencia de uso único y perpetuo para un equipo. Vea los [requisitos](https://office.com/systemrequirements) y, a continuación, elija una versión.

  - Si el equipo Windows está ejecutando Windows 10, considere la posibilidad de obtener [Office Home & Student 2019](https://www.microsoft.com/p/office-home-student-2019/cfq7ttc0k7c8).

  - Si el equipo Windows ejecuta Windows 7, 8 u 8.1 y no está actualizando a Windows 10 ahora, considere la posibilidad de obtener Office Home & Student 2016 u otra edición de Microsoft Office. Puede obtenerlo de un revendedor autorizado.
     
    La compatibilidad con Windows 7 [finalizó el 14 de enero de 2020](https://www.microsoft.com/microsoft-365/windows/end-of-windows-7-support). Microsoft ya no proporciona actualizaciones de seguridad para él. Actualice los Windows 7 dispositivos a Windows 10 para obtener actualizaciones continuas de seguridad y características y soporte técnico continuo.

Si decide no actualizar ahora, las aplicaciones de Office seguirán ejecutándose según las [escalas de tiempo](https://support.microsoft.com/lifecycle/search/13615). Sin embargo, para obtener actualizaciones de seguridad o características nuevas y mejoradas, debe actualizar.
