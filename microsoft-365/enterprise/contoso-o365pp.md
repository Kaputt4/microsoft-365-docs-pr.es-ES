<span data-ttu-id="1568f-129">Contoso siguió el mismo enfoque de dos fases para las actualizaciones de características que se usaban para la implementación de Office: los dispositivos del grupo piloto recibieron actualizaciones de características cuatro meses antes que los dispositivos del resto de la organización (el grupo amplio).</span><span class="sxs-lookup"><span data-stu-id="1568f-129">Contoso followed the same two-stage approach for feature updates as they used for deploying Office: Devices in the pilot group received feature updates four months earlier than devices in the rest of the organization (the broad group).</span></span> <span data-ttu-id="1568f-130">Para habilitar esta opción para Office, Contoso usó dos [canales de actualización](https://docs.microsoft.com/DeployOffice/overview-update-channels) recomendados:</span><span class="sxs-lookup"><span data-stu-id="1568f-130">To enable this for Office, Contoso used two recommended [update channels](https://docs.microsoft.com/DeployOffice/overview-update-channels):</span></span>

- <span data-ttu-id="1568f-132">Semi-Annual el canal de empresa para las actualizaciones del grupo general</span><span class="sxs-lookup"><span data-stu-id="1568f-132">Semi-Annual Enterprise Channel for updates to the broad group</span></span>

<span data-ttu-id="1568f-133">Como el canal semestral empresarial (versión preliminar) publica una versión de Aplicaciones de Microsoft 365 para empresas cuatro meses antes que el canal empresarial semestral, Contoso tiene tiempo para validar las actualizaciones sin tener que administrarlas.</span><span class="sxs-lookup"><span data-stu-id="1568f-133">Because the Semi-Annual Enterprise Channel (Preview) releases a version of Microsoft 365 Apps for enterprise four months earlier than the Semi-Annual Enterprise Channel, Contoso has time to validate the updates without having to manage them.</span></span>
