# <a name="deploy-and-enable-microsoft-defender-antivirus"></a>Implementación y habilitación en el Antivirus de Windows Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Se aplica a:**

- [Microsoft Defender para punto de conexión Plan 1](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft Defender para punto de conexión Plan 2](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- Antivirus de Microsoft Defender

**Plataformas**
- Windows

En función de la herramienta de administración que use, es posible que tenga que habilitar o configurar específicamente Antivirus de Microsoft Defender protección. 

Consulte la tabla [de Implementación, administración e informe sobre Antivirus de Microsoft Defender](deploy-manage-report-microsoft-defender-antivirus.md#ref2) para obtener instrucciones sobre cómo habilitar la protección con Microsoft Intune, Microsoft Endpoint Configuration Manager, directiva de grupo , Active Directory, Microsoft Azure, cmdlets de PowerShell e instrucciones de administración de Windows (WMI).

Algunos escenarios requieren más instrucciones sobre cómo implementar o configurar correctamente Antivirus de Microsoft Defender protección, como entornos de Infraestructura de escritorio virtual (VDI).

En el artículo restante de esta sección se proporcionan consejos de un extremo a otro y procedimientos recomendados para [configurar Antivirus de Microsoft Defender en máquinas virtuales (VM) en un entorno de VDI o Servicios de Escritorio remoto (RDS).](deployment-vdi-microsoft-defender-antivirus.md)
