# <a name="access-the-microsoft-defender-for-endpoint-community-center"></a>Acceder al Centro de comunidad de Microsoft Defender para punto de conexión

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**Se aplica a:**

- [Microsoft Defender para punto de conexión Plan 1](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft Defender para punto de conexión Plan 2](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)


> ¿Quiere experimentar Defender para punto de conexión? [Regístrese para obtener una prueba gratuita.](https://signup.microsoft.com/create-account/signup?products=7f379fee-c4f9-4278-b0a1-e4c8c2fcdf7e&ru=https://aka.ms/MDEp2OpenTrial?ocid=docs-wdatp-assignaccess-abovefoldlink)

El Centro de comunidad de Defender para punto de conexión es un lugar donde los miembros de la comunidad pueden aprender, colaborar y compartir experiencias sobre el producto.

Hay varios espacios que puede explorar para obtener información específica:

- Anuncios
- Novedades
- Inteligencia sobre amenazas

Hay varias maneras de acceder al Centro de la comunidad:

- En el panel de navegación del portal de Microsoft 365 Defender, seleccione **Centro de la comunidad**. Se abre una nueva pestaña del explorador y le lleva a la página Defender for Endpoint Tech Community.
- Acceda a la comunidad a través de la página [Microsoft Defender para punto de conexión Tech Community](https://techcommunity.microsoft.com/t5/Windows-Defender-Advanced-Threat/ct-p/WindowsDefenderAdvanced)

Puede ver y leer al instante las conversaciones que se han publicado en la comunidad.

Para obtener toda la experiencia dentro de la comunidad, como poder comentar las publicaciones, deberás unirte a la comunidad. Para obtener más información sobre cómo empezar a trabajar en la Microsoft Tech Community, consulte [Microsoft Tech Community: Introducción](https://techcommunity.microsoft.com/t5/Getting-Started/Microsoft-Tech-Community-Getting-Started-Guide/m-p/77888#M15).
