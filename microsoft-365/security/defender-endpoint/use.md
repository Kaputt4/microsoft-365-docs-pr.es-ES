# <a name="overview-of-microsoft-defender-security-center"></a>Introducción al Centro de seguridad de Microsoft Defender

> ¿Quiere experimentar Microsoft Defender para punto de conexión? [Regístrese para obtener una prueba gratuita.](https://signup.microsoft.com/create-account/signup?products=7f379fee-c4f9-4278-b0a1-e4c8c2fcdf7e&ru=https://aka.ms/MDEp2OpenTrial?ocid=docs-wdatp-usewdatp-abovefoldlink)

Centro de seguridad de Microsoft Defender es el portal donde puede acceder a las funcionalidades de Microsoft Defender para punto de conexión.

Use el panel **de Administración de vulnerabilidades de Defender** para ampliar la visibilidad sobre la posición de seguridad general de su organización. Verá dispositivos que requieren atención y recomendaciones que pueden ayudarle a reducir la superficie expuesta a ataques en su organización.

Use el panel **Análisis de amenazas** para evaluar y controlar continuamente la exposición de riesgos a Spectre y Meltdown.

## <a name="microsoft-defender-for-endpoint-interactive-guide"></a>Microsoft Defender para punto de conexión guía interactiva
En esta guía interactiva, aprenderá a investigar las amenazas a su organización con Microsoft Defender para punto de conexión. Verá cómo Microsoft Defender para punto de conexión puede ayudarle a identificar actividades sospechosas, investigar riesgos para su organización y corregir amenazas.
