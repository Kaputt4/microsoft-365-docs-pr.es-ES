## <a name="enable-microsoft-defender-for-cloud-apps-in-microsoft-defender-for-endpoint"></a>Habilitar Microsoft Defender for Cloud Apps en Microsoft Defender para punto de conexión

1. En el panel de navegación, seleccione **Configuración de** \> preferencias **Características avanzadas**.
2. Seleccione **Microsoft Defender for Cloud Apps** y cambie el botón de alternancia a **Activado**.
3. Haga clic en **Guardar preferencias**.

Una vez activado, Microsoft Defender para punto de conexión comenzará inmediatamente a reenviar las señales de detección a Defender for Cloud Apps.
