# <a name="how-to-get-microsoft-defender-for-business-servers"></a>Obtención de servidores Microsoft Defender para Empresas

Microsoft Defender para Empresas servidores es un complemento de Defender para empresas. Esta nueva oferta le permite proteger los sistemas operativos del servidor con la misma protección que obtiene para los dispositivos cliente en Defender para empresas. En este artículo se describe cómo obtener Microsoft Defender para Empresas servidores e incluye los pasos siguientes e información adicional.

> [!IMPORTANT]
> - Para agregar en Microsoft Defender para Empresas servidores, necesitará al menos una licencia de pago para [Microsoft 365 Empresa Premium](../../business-premium/index.md) o [Defender para la empresa](mdb-overview.md) (independiente). 
> - Necesitará una licencia de servidores de Microsoft Defender para Empresas por instancia de servidor.
> - Hay un límite de 60 licencias de servidores de Microsoft Defender para Empresas por suscripción para Microsoft 365 Empresa Premium o Defender para empresas.
> - Como alternativa, podría usar [Microsoft Defender para servidores](/azure/defender-for-cloud/defender-for-servers-introduction) para incorporar los servidores; sin embargo, la experiencia de Defender para empresas podría cambiar al agregar un plan empresarial, como el plan 1 de Defender para servidores o el plan 2. Para obtener más información, consulte [¿Qué ocurre si tengo una combinación de suscripciones de seguridad de puntos de conexión de Microsoft?](mdb-faq.yml#what-happens-if-i-have-a-mix-of-microsoft-endpoint-security-subscriptions) e [Incorporación de dispositivos a Microsoft Defender para Empresas](mdb-onboard-devices.md).

## <a name="get-microsoft-defender-for-business-servers"></a>Obtener servidores Microsoft Defender para Empresas

Use uno de los procedimientos siguientes para obtener servidores Microsoft Defender para Empresas:

| Escenario  | Procedure  |
|---------|---------|
| Actualmente tiene [Defender para empresas](mdb-overview.md) o [Microsoft 365 Empresa Premium](../../business-premium/index.md) y quiere agregar en servidores de Microsoft Defender para Empresas.     | <ol><li>Vaya al Centro de administración de Microsoft 365 ([https://admin.microsoft.com/](https://admin.microsoft.com/)) e inicie sesión.</li><li>En el panel de navegación, elija **Servicios de compra de facturación** > .</li><li>En la lista de resultados, seleccione el cuadro **Detalles** de **Microsoft Defender para Empresas servidores**.</li><li>Revise la información y complete el proceso de compra. </li></ol>*Necesitará una licencia de servidores Microsoft Defender para Empresas para cada instancia de Windows Server o Linux.*        |
| Aún no tiene Defender para empresas ni Microsoft 365 Empresa Premium.     | <ol><li>Vaya a una de las siguientes páginas de productos: <ul><li>[Microsoft Defender para Empresas](https://aka.ms/DefenderforBusiness)</li><li>[Microsoft 365 para empresas](https://www.microsoft.com/en-us/microsoft-365/business-h)</li></ul></li><li>Revise la información e inicie la suscripción de prueba o de pago hoy mismo.</li></ol> |
| Quiere desconectar uno o varios dispositivos, como servidores. | Consulte [Offboard a device from Microsoft Defender para Empresas (Desconectar un dispositivo de Microsoft Defender para Empresas](mdb-offboard-devices.md)). |
