# <a name="get-help-and-support-for-microsoft-defender-for-business"></a>Obtener ayuda y soporte técnico para Microsoft Defender para Empresas

## <a name="get-help-and-support"></a>Obtener ayuda y soporte técnico

1. Si necesita ayuda con Defender para empresas, seleccione el icono de Ayuda (?) en la esquina superior derecha de la pantalla. 

2. Escriba la pregunta o el problema y, a continuación, seleccione la flecha azul, como se muestra en la captura de pantalla siguiente: 

   :::image type="content" source="media/help-pane.png" alt-text="Captura de pantalla del panel de ayuda con una pregunta sobre cómo agregar dispositivos.":::

   Se mostrarán varias opciones, como respuestas rápidas o artículos de ayuda.

3. Seleccione un elemento en la lista de resultados. Si no ve la respuesta a su pregunta, seleccione **Ponerse en contacto con soporte técnico** en la parte inferior del panel flotante. Para obtener más información, consulte [Obtención de soporte técnico](../../admin/get-help-support.md).
