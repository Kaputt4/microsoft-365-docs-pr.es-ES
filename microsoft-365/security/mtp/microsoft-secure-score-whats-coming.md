# <a name="whats-coming-to-microsoft-secure-score"></a>What's coming to Microsoft Secure Score

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

La puntuación de seguridad de Microsoft puede encontrarse https://security.microsoft.com/securescore en el Centro de seguridad de Microsoft [365.](overview-security-center.md)

## <a name="proposed-changes"></a>Cambios propuestos

Estamos realizando algunos cambios en el futuro próximo para que la puntuación de seguridad de [Microsoft](microsoft-secure-score.md) sea un mejor representante de su posición de seguridad y mejorar la facilidad de uso. La puntuación y la puntuación máxima posible pueden cambiar.

### <a name="no-planned-changes"></a>Sin cambios planeados

No hay cambios planeados en este momento.
