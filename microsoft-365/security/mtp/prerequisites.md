# <a name="microsoft-threat-protection-prerequisites"></a>Requisitos previos de la Protección contra amenazas de Microsoft

**Se aplica a:**
- Protección contra amenazas de Microsoft

[!INCLUDE [Prerelease information](../includes/prerelease.md)]

Obtenga más información sobre las licencias, requisitos previos de hardware y software, y otras opciones de configuración para ejecutar y usar los servicios de seguridad de Microsoft 365.

## <a name="licensing-requirements"></a>Requisitos de licencia
La seguridad de Microsoft 365 requiere una de las siguientes licencias:

- Microsoft 365 E5 
- Office 365 E5, Enterprise Mobility + Security E5 y Windows E5

Puede adquirir estas licencias en la [Página de Microsoft 365 Enterprise](https://www.microsoft.com/en-us/microsoft-365/enterprise).

### <a name="check-your-existing--licenses"></a>Comprobar las licencias existentes
Vaya al Centro de administración de Microsoft 365 en [admin.microsoft.com](https://admin.microsoft.com/) para ver las licencias existentes. En el Centro de administración, vaya a **Facturación** > **Licencias**.

Debe tener asignado el rol de **Administrador de facturación** o **Lector global** [en Azure AD](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#available-roles) para poder ver la información de licencias. Si tiene problemas de acceso, póngase en contacto con un administrador global.  

## <a name="browser-requirements"></a>Requisitos de los exploradores
El acceso al Centro de seguridad de Microsoft 365 se lleva a cabo en un navegador. Internet Explorer 10 y Microsoft Edge son compatibles, así como cualquier otro navegador compatible con HTML5.
