# <a name="overview-of--microsoft-365-defender-apis"></a>Información general de las API de Microsoft 365 defender

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Se aplica a:**
- Microsoft 365 defender


>[!IMPORTANT] 
>Parte de la información se refiere a un producto prelanzamiento que puede modificarse de forma sustancial antes de su lanzamiento comercial. Microsoft makes no warranties, express or implied, with respect to the information provided here.

La solución de Microsoft 365 defender se basa en una plataforma lista para la integración. 

Las API de Lop de Microsoft 365 defender le permitirán automatizar flujos de trabajo en función de las tablas incidente compartido y de caza avanzada.

- **Cola de incidentes combinados** : ayuda a los profesionales de la seguridad a centrarse en lo fundamental. Ayuda a garantizar que el ámbito de ataque completo y los activos afectados se agrupan y se exponen de forma puntual en la API de incidentes.

- **Búsqueda de amenazas entre productos** : los equipos de seguridad pueden aprovechar el conocimiento único de la organización para buscar signos de peligro mediante la creación de sus propias consultas personalizadas a través de las API sobre los datos sin procesar recopilados por los distintos productos de protección. 

Además de este conjunto de API, cada uno de los diversos productos de protección exponen API adicionales para ayudarle a innovar en función de cada capacidad de producto.
