# <a name="overview-of--microsoft-threat-protection-apis"></a>Información general sobre las API de Microsoft Threat Protection

**Se aplica a:**
- Protección contra amenazas de Microsoft


>[!IMPORTANT] 
>Parte de la información se refiere a un producto prelanzamiento que puede modificarse de forma sustancial antes de su lanzamiento comercial. Microsoft makes no warranties, express or implied, with respect to the information provided here.

La solución de protección contra amenazas de Microsoft se basa en una plataforma preparada para la integración. 

Las API de protección contra amenazas de Microsoft de nivel Lop le permitirán automatizar flujos de trabajo en función de las tablas incidente compartido y de caza avanzada.

- **Cola de incidentes combinados** : ayuda a los profesionales de la seguridad a centrarse en las tareas críticas al garantizar que el ámbito completo de los ataques y los activos afectados se agrupan y se exponen de manera oportuna en la API de incidentes.

- **Búsqueda de amenazas entre productos** : los equipos de seguridad pueden aprovechar el conocimiento único de la organización para buscar signos de peligro mediante la creación de sus propias consultas personalizadas a través de las API sobre los datos sin procesar recopilados por los distintos productos de protección. 

Además de este conjunto de API, cada uno de los diversos productos de protección exponen API adicionales para ayudarle a innovar en función de las capacidades de cada producto.
