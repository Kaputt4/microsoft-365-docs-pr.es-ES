## <a name="apis"></a>API

 Las API de Microsoft 365 defender se rigen por [licencia de API de Microsoft y condiciones de uso](https://docs.microsoft.com/legal/microsoft-apis/terms-of-use).

### <a name="throttling-limits"></a>Superado la limitación

Cada API tiene sus propios límites de limitación.<br>
Hay límites de limitación por hora y por minuto.<br>
Por ejemplo, los límites de limitación de la API de incidentes son: 50 llamadas por minuto y 1500 por hora.<br>
El código de estado de respuesta HTTP en caso de limitación será 429.
