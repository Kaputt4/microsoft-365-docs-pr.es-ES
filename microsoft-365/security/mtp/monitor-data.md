# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Supervisión e informes de datos en el Centro de seguridad de Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


La categoría **Datos** ayuda a realizar un seguimiento de la actividad de usuarios que pueda conducir a la divulgación no autorizada de datos. Son el trabajo de reelaje de los informes de directivas DLP existentes más un informe de coincidencia de directiva DLP de terceros.

Se puede ver lo siguiente:

- Usuarios que comparten la mayoría de los archivos de las aplicaciones en la nube
- Cuántas coincidencias de directiva DLP se produjeron
- Número de directivas DLP que se anulan o comunican falsos positivos
- Cuántas coincidencias de directiva DLP se han producido en servicios en la nube de terceros a través de Microsoft Cloud App Security
