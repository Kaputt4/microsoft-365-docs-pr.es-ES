# <a name="step-3-enable-and-pilot-microsoft-defender-for-office-365"></a>Paso 3. Habilitar y piloto Microsoft Defender para Office 365

**Se aplica a:**
- Microsoft 365 Defender

En este artículo se describe el proceso para habilitar y pilotar Microsoft Defender para Office 365. Antes de iniciar este proceso, asegúrese de haber revisado el proceso general para evaluar Microsoft 365 Defender [](eval-overview.md) y de haber creado el entorno [Microsoft 365 Defender evaluación.](eval-create-eval-environment.md) 
<br>

Use los siguientes pasos para habilitar y pilotar Microsoft Defender para Office 365.

:::image type="content" source="../../media/defender/m365-defender-office-eval-steps.png" alt-text="Los pasos para agregar Microsoft Defender para Office al entorno de evaluación de Microsoft Defender" lightbox="../../media/defender/m365-defender-office-eval-steps.png":::

En la tabla siguiente se describen los pasos de la ilustración.

| Número de serie|Paso  |Descripción  |
|---------|---------|---------|
|1|[Revisar los requisitos de arquitectura y los conceptos clave](eval-defender-office-365-architecture.md)    | Comprenda defender para Office arquitectura y asegúrese de que el entorno Exchange Online cumple los requisitos previos de la arquitectura.       |
|2|[Habilitar el entorno de evaluación](eval-defender-office-365-enable-eval.md)     |   Siga los pasos para configurar el entorno de evaluación.      |
|3|[Configurar el piloto ](eval-defender-office-365-pilot.md)    |    Cree grupos piloto, configure la protección y familiarícese con las características y paneles clave.     |
