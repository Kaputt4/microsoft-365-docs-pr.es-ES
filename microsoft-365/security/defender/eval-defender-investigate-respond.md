# <a name="step-6-investigate-and-respond-using-microsoft-365-defender-in-a-pilot-environment"></a>Paso 6. Investigación y respuesta mediante Microsoft 365 Defender en un entorno piloto

- Microsoft 365 Defender

En este artículo se describe el proceso para crear incidentes con simulaciones de ataques y tutoriales y usar Microsoft 365 Defender para investigar y responder. Antes de iniciar este proceso, asegúrese de que ha revisado el proceso general para [evaluar Microsoft 365 Defender](eval-overview.md) y ha [creado el entorno de evaluación de Microsoft 365 Defender](eval-create-eval-environment.md).
