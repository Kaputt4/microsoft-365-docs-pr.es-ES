# <a name="whats-coming-to-microsoft-secure-score"></a>What's coming to Microsoft Secure Score

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

Microsoft Secure Score se puede encontrar en https://security.microsoft.com/securescore el [portal de Microsoft 365 Defender.](microsoft-365-defender.md#the-microsoft-365-defender-portal)

Este artículo le permite saber acerca de las próximas mejoras en la puntuación segura de Microsoft.

## <a name="proposed-changes"></a>Cambios propuestos

### <a name="upcoming-improvement-action-additions-december-2021"></a>Próximas adiciones de acciones de mejora (diciembre de 2021)

- Activar los Caja fuerte adjuntos en modo de bloque
- Impedir el uso Exchange Online detalles del calendario con usuarios externos
- Activar documentos Caja fuerte para Office clientes
- Activar la configuración de filtro de datos adjuntos común para directivas antimalware
- Asegúrese de que no hay dominios de remitente permitidos para directivas contra correo no deseado
- Crear Caja fuerte de vínculos para mensajes de correo electrónico
- Crear directivas de purga automática de hora cero para malware
- Activa Microsoft Defender para Office 365 en SharePoint, OneDrive y Microsoft Teams
- Crear directivas de purga automática de hora cero para mensajes de suplantación de identidad
- Crear directivas de purga automática de hora cero para mensajes de correo no deseado
- Bloquear el uso indebido de controladores firmados vulnerables explotados
- Activar el examen de unidades extraíbles durante un examen completo
