# <a name="evaluate-microsoft-cloud-app-security"></a>Evaluar Microsoft Cloud App Security

**Se aplica a:**
- Microsoft 365 Defender


En este artículo se describe el proceso para habilitar y Microsoft Cloud App Security junto con Microsoft 365 Defender. Antes de iniciar este proceso, asegúrese de que [](eval-overview.md) ha revisado el proceso general para evaluar Microsoft 365 Defender y ha creado el entorno [Microsoft 365 Defender evaluación](eval-create-eval-environment.md). 
<br>

Siga estos pasos para habilitar y pilotar la seguridad de aplicaciones en la nube de Microsoft.

![Pasos para agregar Microsoft Defender para Office al entorno de evaluación de Defender](../../media/defender/m365-defender-office-eval-steps.png)



|Paso  |Descripción  |
|---------|---------|
|[Revisar los requisitos de arquitectura y los conceptos clave](eval-defender-mcas-architecture.md)    | Comprenda la Cloud App Security y cómo se integra con Microsoft 365 Defender, Microsoft Defender para endpoint y Azure Active Directory.        |
|[Habilitar el entorno de evaluación](eval-defender-mcas-enable-eval.md)     | Conectar al portal, configure la integración con Defender for Identity y/o los dispositivos de red de su organización y comience a ver y administrar aplicaciones en la nube.         |
|[Configurar el piloto ](eval-defender-mcas-pilot.md)    | Establezca el ámbito de la implementación en determinados grupos de usuarios, configure el Control de aplicaciones de acceso condicional y pruebe tutoriales para proteger su entorno.       |
