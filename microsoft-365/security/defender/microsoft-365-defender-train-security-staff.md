# <a name="train-your-security-staff-for-microsoft-365-defender"></a>Entrenar al personal de seguridad para Microsoft 365 Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

**Se aplica a:**
- Microsoft 365 Defender

Con la urgencia de proteger su organización y sus datos de los atacantes, debe incorporar rápidamente el personal de seguridad para Microsoft 365 Defender. Microsoft proporciona varios tipos de recursos de aprendizaje y auto-estudio para Microsoft 365 Defender y servicios asociados.


## <a name="microsoft-365-defender-ninja-training"></a>Microsoft 365 Defender entrenamiento ninja

[Microsoft 365 Defender entrenamiento ninja](https://techcommunity.microsoft.com/t5/microsoft-365-defender/become-a-microsoft-365-defender-ninja/ba-p/1789376) es un conjunto de secciones y módulos organizados para recorrer las características y funciones de Microsoft 365 Defender. El entrenamiento recorre los orígenes&mdash;de señal de amenazas, como Microsoft Defender para punto de conexión y otros&mdash;, pero no cubre los orígenes individuales. 

El contenido se estructura en tres niveles de conocimiento diferentes, con varios módulos: Fundamentales, Intermedio y Experto. Cada nivel incluye una comprobación de conocimientos para probar su comprensión del material.

## <a name="microsoft-learn-learning-paths"></a>Rutas de aprendizaje de Microsoft Learn

Use estas rutas de aprendizaje de Microsoft Learn y sus módulos para comprender Microsoft 365 Defender y Microsoft Defender para punto de conexión, un módulo y una unidad a la vez.

 - [Detectar y responder a ataques cibernéticos con Microsoft 365 Defender](/learn/paths/defender-detect-respond/)

 - [Proteja su organización con Microsoft Defender para punto de conexión](/learn/paths/defender-endpoint-fundamentals/)  


## <a name="learning-paths-in-the-microsoft-365-defender-portal-learning-hub"></a>Rutas de aprendizaje en el centro de aprendizaje del portal de Microsoft 365 Defender

El [centro de aprendizaje de Microsoft 365 Defender portal](https://security.microsoft.com/learning) incluye estas rutas de aprendizaje:

- Introducción a la Centro de seguridad de Microsoft 365
- Cómo investigar mediante Microsoft 365 Defender
- entrenamiento básico de Microsoft 365 Defender
- Microsoft Defender para punto de conexión entrenamiento básico
- procedimientos recomendados de Microsoft Defender para Office 365
- Instalación

## <a name="hands-on-with-a-trial-environment"></a>Práctica con un entorno de prueba

Una de las mejores maneras de aprender es configurar y probar Microsoft 365 Defender y sus servicios asociados en un entorno de prueba. Consulte [Evaluación y Microsoft 365 Defender piloto](eval-overview.md) para obtener los detalles.

## <a name="simulating-an-attack"></a>Simulación de un ataque

Para probar al personal de seguridad su capacidad de detectar y responder a un ciberataque, use un ataque simulado. Consulte [Simulación de ataques con el portal de Microsoft 365 Defender](eval-defender-investigate-respond-simulate-attack.md#simulate-attacks-with-the-microsoft-365-defender-portal) para obtener más información.
