# <a name="industry-collaboration-programs"></a>Programas de colaboración del sector

Microsoft proporciona varios programas de colaboración en todo el sector con diferentes objetivos y requisitos. Inscribirse en el programa adecuado puede ayudarle a proteger a sus clientes, obtener más información sobre el panorama actual de amenazas o ayudar a interrumpir el ecosistema de malware.

## <a name="virus-information-alliance-via"></a>Virus Information Alliance (VIA)

El programa VIA proporciona a los miembros acceso a información que ayudará a mejorar la protección de los clientes de Microsoft. La telemetría y los ejemplos de malware se pueden proporcionar a los equipos de seguridad para ayudar a identificar las diferencias en su protección, priorizar la nueva cobertura de amenazas o responder mejor a las amenazas.

**Debe ser miembro de VIA si desea solicitar la pertenencia a los otros programas.**

Vaya a la [página del programa VIA](virus-information-alliance-criteria.md) para obtener más información.

## <a name="microsoft-virus-initiative-mvi"></a>Microsoft Virus Initiative (MVI)

MVI está abierto a organizaciones que crean y son propietarios de un producto antimalware de Protección en tiempo real (RTP) de su propio diseño o uno desarrollado con un SDK de antivirus de terceros.

Los miembros obtienen acceso a las API de cliente de Microsoft para los archivos Centro de seguridad de Microsoft Defender, IOAV, AMSI y Cloud, junto con datos de estado y otra telemetría para ayudar a sus clientes a mantenerse protegidos. Los productos antimalware se envían a Microsoft para realizar pruebas de rendimiento con regularidad.

Vaya a la [página del programa MVI](virus-initiative-criteria.md) para obtener más información.

## <a name="coordinated-malware-eradication-cme"></a>Eliminación coordinada de malware (CME)

CME está abierto a organizaciones que están involucradas en ciberseguridad y antimalware o que están interesadas en combatir la ciberdelincuencia.

El programa tiene como objetivo reunir a las organizaciones de ciberseguridad y otras industrias para reunir herramientas, información y acciones para impulsar campañas coordinadas contra malware. El objetivo final es crear resultados eficientes y duraderos para una mejor protección de nuestras comunidades, clientes y empresas.

Vaya a la [página del programa CME](coordinated-malware-eradication.md) para obtener más información.
