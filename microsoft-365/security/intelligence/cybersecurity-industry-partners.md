# <a name="industry-collaboration-programs"></a>Programas de colaboración del sector

Microsoft proporciona diversos programas de colaboración para todo el sector con diferentes objetivos y requisitos. Inscribirse en el programa adecuado puede ayudarle a proteger a sus clientes, obtener más información sobre el panorama actual de amenazas o ayudar a interrumpir el ecosistema de malware.

## <a name="virus-information-alliance-via"></a>Alianza de información de virus (VIA)

El programa VIA proporciona a los miembros acceso a información que ayudará a mejorar la protección de los clientes de Microsoft. Se pueden proporcionar ejemplos y telemetría de malware a los equipos de seguridad para ayudar a identificar brechas en su protección, priorizar la nueva cobertura de amenazas o responder mejor a las amenazas.

**Debe ser miembro de VIA si desea solicitar la pertenencia a los otros programas.**

Vaya a la [página del programa VIA](virus-information-alliance-criteria.md) para obtener más información.

## <a name="microsoft-virus-initiative-mvi"></a>Microsoft Virus Initiative (MVI)

MVI está abierto a las organizaciones que crean y poseen un producto antimalware de Protección en tiempo real (RTP) de su propio diseño, o bien uno desarrollado mediante un SDK antivirus de terceros.

Los miembros obtienen acceso a las [API de Microsoft 365 Defender](../defender/api-overview.md) para el portal de Microsoft 365 Defender, IOfficeAntivirus (IOAV), AntiMalware Scan Interface (AMSI) y Cloud Files, junto con datos de estado y otros datos de telemetría para ayudar a los clientes a mantenerse protegidos. Los productos antimalware se envían a Microsoft para realizar pruebas de rendimiento con regularidad.

Vaya a la [página del programa MVI](virus-initiative-criteria.md) para obtener más información.

## <a name="coordinated-malware-eradication-cme"></a>Erradicación coordinada de malware (CME)

CME está abierto a las organizaciones que participan en la ciberseguridad y el antimalware o están interesadas en luchar contra la ciberdelincuencia.

El programa tiene como objetivo reunir a las organizaciones de ciberseguridad y otros sectores para agrupar herramientas, información y acciones para impulsar campañas coordinadas contra el malware. El objetivo final es crear resultados eficientes y duraderos para una mejor protección de nuestras comunidades, clientes y empresas.

Vaya a la [página del programa CME](coordinated-malware-eradication.md) para obtener más información.
