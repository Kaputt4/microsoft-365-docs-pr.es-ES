# <a name="microsoft-safety-scanner-download"></a>Descargar el Examen de seguridad de Microsoft

Examen de seguridad de Microsoft es una herramienta de examen diseñada para buscar y quitar el malware de equipos Windows. Simplemente descárguela y ejecute un examen para buscar malware e intentar revertir los cambios realizados por las amenazas identificadas.

- **[Descargar Examen de seguridad de Microsoft (32 bits)](https://go.microsoft.com/fwlink/?LinkId=212733)**

- **[Descargar Examen de seguridad de Microsoft (64 bits)](https://go.microsoft.com/fwlink/?LinkId=212732)**

> [!NOTE]
> A partir de noviembre de 2019, Examen de seguridad estará firmado exclusivamente por SHA-2. Los dispositivos que vayan a ejecutar Examen de seguridad deberán actualizarse para admitir SHA-2. Para obtener más información, consulte [Requisito de compatibilidad con la firma de código SHA-2 de 2019 para Windows y WSUS](https://support.microsoft.com/help/4472027/2019-sha-2-code-signing-support-requirement-for-windows-and-wsus).

## <a name="important-information"></a>Información importante

- La versión de actualización de inteligencia de seguridad del Examen de seguridad de Microsoft coincide con la versión descrita [en esta página web](https://www.microsoft.com/wdsi/definitions).

- Examen de seguridad de Microsoft solo examina cuando se desencadena manualmente y está disponible para su uso 10 días después de la descarga. Te recomendamos que descargues siempre la versión más reciente de esta herramienta antes de cada examen.

- Examen de seguridad de Microsoft es un ejecutable portátil y no aparece en el menú Inicio de Windows ni como un icono en el escritorio. Tenga en cuenta dónde guardó esta descarga.

- Esta herramienta no es un sustituto de su producto antimalware. Para obtener protección en tiempo real con actualizaciones automáticas, use [Antivirus de Microsoft Defender en Windows 11, Windows 10 y Windows 8](https://www.microsoft.com/windows/comprehensive-security) o [Microsoft Security Essentials en Windows 7](https://support.microsoft.com/help/14210/security-essentials-download). Estos productos antimalware también proporcionan eficaces funcionalidades de eliminación de malware. Si tiene dificultades para eliminar malware con estos productos, puede consultar nuestro artículo de ayuda para [eliminar amenazas difíciles](https://www.microsoft.com/wdsi/help/troubleshooting-infection).

## <a name="system-requirements"></a>Requisitos del sistema

Examen de seguridad ayuda a quitar software malintencionado de equipos que ejecutan Windows 11, Windows 10, Windows 10 Tech Preview, Windows 8.1, Windows 8, Windows 7, Windows Server 2019, Windows Server 2016, Windows Server Tech Preview, Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2 o Windows Server 2008. Para obtener más información, consulte la [directiva de ciclo de vida de Microsoft](/lifecycle/).

## <a name="how-to-run-a-scan"></a>Cómo ejecutar un examen

1. Descargue esta herramienta y ábrala.
2. Seleccione el tipo de examen que desea ejecutar e inicie el examen.
3. Revise los resultados del examen que se muestran en la pantalla. Para obtener resultados de detección detallados, vea el registro en **%SYSTEMROOT%\debug\msert.log**.

Para quitar esta herramienta, elimine el archivo ejecutable (msert.exe de forma predeterminada).

Para obtener más información sobre Examen de seguridad, consulte el artículo de soporte técnico [Cómo solucionar problemas con Examen de seguridad](https://support.microsoft.com/kb/2520970).
