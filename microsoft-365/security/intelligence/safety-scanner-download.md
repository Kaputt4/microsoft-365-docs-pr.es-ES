# <a name="microsoft-safety-scanner-download"></a>Examen de seguridad de Microsoft descargar

Examen de seguridad de Microsoft es una herramienta de examen diseñada para buscar y quitar malware de Windows equipos. Basta con descargarlo y ejecutar un examen para buscar malware e intentar revertir los cambios realizados por las amenazas identificadas.

- **[Descargar Examen de seguridad de Microsoft (32 bits)](https://go.microsoft.com/fwlink/?LinkId=212733)**

- **[Descargar Examen de seguridad de Microsoft (64 bits)](https://go.microsoft.com/fwlink/?LinkId=212732)**

> [!NOTE]
> A partir de noviembre de 2019, el escáner de seguridad se firmará exclusivamente con SHA-2. Los dispositivos deben actualizarse para admitir SHA-2 para poder ejecutar el Escáner de seguridad. Para obtener más información, consulte [2019 SHA-2 Code Signing Support requirement for Windows and WSUS](https://support.microsoft.com/help/4472027/2019-sha-2-code-signing-support-requirement-for-windows-and-wsus).

## <a name="important-information"></a>Información importante

- La versión de actualización de inteligencia de seguridad del Examen de seguridad de Microsoft coincide con la versión descrita [en esta página web](https://www.microsoft.com/wdsi/definitions).

- Safety Scanner solo examina cuando se desencadena manualmente y está disponible para su uso 10 días después de su descarga. Se recomienda descargar siempre la versión más reciente de esta herramienta antes de cada examen.

- El escáner de seguridad es un ejecutable portátil y no aparece en el Windows menú Inicio ni como un icono en el escritorio. Tenga en cuenta dónde guardó esta descarga.

- Esta herramienta no reemplaza el producto antimalware. Para la protección en tiempo real con actualizaciones automáticas, use Antivirus de Microsoft Defender en [Windows 11, Windows 10 y](https://www.microsoft.com/windows/comprehensive-security) Windows 8 o [Microsoft Security Essentials en Windows 7](https://support.microsoft.com/help/14210/security-essentials-download). Estos productos antimalware también proporcionan potentes capacidades de eliminación de malware. Si tiene dificultades para eliminar malware con estos productos, puede consultar nuestra ayuda para [eliminar amenazas difíciles](https://www.microsoft.com/wdsi/help/troubleshooting-infection).

## <a name="system-requirements"></a>Requisitos del sistema

Safety Scanner ayuda a quitar software malintencionado de equipos que ejecutan Windows 11, Windows 10, Windows 10 Tech Preview, Windows 8.1, Windows 8, Windows 7, Windows Server 2019, Windows Server 2016, Windows Server Tech Preview, Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2 o Windows Server 2008. Para obtener más información, consulte la [Directiva de ciclo de vida de Microsoft](/lifecycle/).

## <a name="how-to-run-a-scan"></a>Cómo ejecutar un examen

1. Descargue esta herramienta y ábrala.
2. Seleccione el tipo de examen que desea ejecutar e inicie el examen.
3. Revise los resultados del examen que se muestran en pantalla. Para obtener resultados de detección detallados, vea el registro **en %SYSTEMROOT%\debug\msert.log**.

Para quitar esta herramienta, elimine el archivo ejecutable (msert.exe de forma predeterminada).

Para obtener más información acerca del escáner de seguridad, consulte el artículo de soporte técnico sobre [cómo solucionar problemas con el escáner de seguridad](https://support.microsoft.com/kb/2520970).
