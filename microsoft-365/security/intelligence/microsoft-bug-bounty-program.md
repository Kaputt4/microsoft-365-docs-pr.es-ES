# <a name="about-the-microsoft-bug-bounty-program"></a>Acerca del programa de recompensas por errores de Microsoft

¿Eres un investigador de seguridad? ¿Encontró una vulnerabilidad en un producto, servicio o dispositivo de Microsoft? ¡Si es así, queremos saber de ti!

Si su informe de vulnerabilidad afecta a un producto o servicio que está dentro del ámbito de uno de nuestros programas de recompensas a continuación, podría recibir un premio de recompensas de acuerdo con las descripciones del programa.

Visite el [sitio del Programa de recompensas de errores de Microsoft](https://www.microsoft.com/en-us/msrc/bounty?rtc=1) para obtener todos los detalles.
