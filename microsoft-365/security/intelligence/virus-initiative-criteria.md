# <a name="microsoft-virus-initiative"></a>Iniciativa de virus de Microsoft

Microsoft Virus Initiative (MVI) ayuda a las organizaciones a desarrollar soluciones de seguridad mejor juntas que sean eficaces, confiables y alineadas con la tecnología y la estrategia de Microsoft.

## <a name="become-a-member"></a>Convertirse en miembro

Puede solicitar la pertenencia si es un representante de una organización que desarrolla y genera tecnología antimalware o antivirus. 

Para calificar para el programa MVI, su organización debe cumplir todos los requisitos siguientes:

1. La solución de seguridad reemplaza o complementa Antivirus de Microsoft Defender.

2. Su organización es responsable de desarrollar y distribuir actualizaciones de aplicaciones a los clientes finales que abordan la compatibilidad con Windows.

3. Su organización debe estar activa en el sector antimalware y tener una reputación positiva, como lo demuestra la participación en conferencias del sector o la revisión en un informe estándar del sector, como AV-Comparatives, OPSWAT o Gartner.

4. Su organización debe firmar un contrato de no divulgación (NDA) con Microsoft.

5. Su organización debe firmar un contrato de licencia del programa. El mantenimiento de este contrato de licencia requiere que se cumplan todos los requisitos del programa para las aplicaciones antimalware. Estos requisitos definen el comportamiento de las aplicaciones antimalware necesarias para garantizar una interacción adecuada con Windows.

6. Debe enviar la aplicación a Microsoft para realizar pruebas periódicas de rendimiento y revisión de características.

7. La solución debe estar certificada mediante pruebas independientes por al menos una organización estándar del sector y debe mantenerse la certificación anual.
