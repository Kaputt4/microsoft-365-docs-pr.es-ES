# <a name="set-language-and-time-zones-in-microsoft-bookings"></a>Establecer zonas horarias y de idioma en Microsoft Bookings

Si usa Microsoft Bookings y las reservas se crean en el momento incorrecto, es posible que sea necesario cambiar la configuración de la zona horaria. Del mismo modo, si algunas reservas están en el idioma incorrecto, es posible que deba cambiar la configuración del idioma.

Hay dos configuraciones de idioma y zona horaria independientes para Bookings. La primera configuración controla el idioma y la zona horaria del calendario de reserva y se establece mediante la configuración de Outlook en la Web para el calendario personal del usuario que ha iniciado sesión. La segunda configuración afecta a la página de reserva de autoservicio que usan los clientes y se establece mediante una página de "configuración regional" que controla el idioma y la zona horaria solo para esa página.

## <a name="setting-language-and-time-zone-for-a-booking-calendar"></a>Configuración de idioma y zona horaria para un calendario de reserva

El calendario de reserva usa la configuración de idioma y zona horaria del usuario que ha iniciado sesión. Por ejemplo, si la zona horaria del usuario que ha iniciado sesión está establecida en Hora estándar oriental (EST), el calendario de reserva mostrará las horas de inicio y finalización de citas existentes en EST. Esta zona horaria se estableció originalmente cuando se crearon las cuentas de Microsoft 365 y Outlook en la Web del usuario.

Para establecer el idioma y la zona horaria del calendario de reserva:

1. Inicie sesión en Microsoft 365 y seleccione el icono de Outlook en la página de aterrizaje (como se muestra en la captura de pantalla siguiente) o en el Iniciador de aplicaciones de Microsoft 365.

   :::image type="content" source="../media/bookings-outlook-tile.png" alt-text="Icono de Outlook en la página de aterrizaje de Microsoft 365.":::

1. Después de abrir Outlook, seleccione el **icono de engranaje** en la esquina superior derecha de la pantalla para abrir la configuración personal y de la cuenta y, a continuación, busque "zona horaria" en el cuadro de búsqueda del panel **Configuración** . El panel se actualizará para mostrar la configuración actual del idioma personal y la zona horaria de esta cuenta. Como se indicó anteriormente, esta configuración también controla el idioma y la zona horaria del calendario de reserva.

1. Cambie el idioma o la zona horaria seleccionando la flecha desplegable en el cuadro **Idioma o Zona horaria actual** y eligiendo la configuración deseada.

1. Haga clic en **Guardar**. El panel Configuración se cierra, Outlook en la Web se reinicia y se aplica la nueva configuración de idioma y zona horaria.

## <a name="setting-the-language-and-time-zone-for-the-booking-page"></a>Establecer el idioma y la zona horaria de la página de reserva

1. En Microsoft 365, seleccione el iniciador de aplicaciones y, a continuación, seleccione **Bookings**.

1. En el panel de navegación, seleccione **Booking page (Página de reserva)** y **seleccione Change language and time zone settings (Cambiar configuración de idioma y zona horaria**).

   ![Captura de pantalla: cambiar el idioma y el vínculo de configuración de zona horaria.](../media/bookings-region-language-timezone-settings.png)

1. Seleccione el idioma y la zona horaria actual y elija **Aceptar**.
