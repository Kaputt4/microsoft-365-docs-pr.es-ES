# <a name="manage-group-attendees-in-bookings"></a>Gestionar asistentes de grupos en Bookings

Los propietarios y miembros del personal de un calendario de Bookings ahora pueden administrar y actualizar a los asistentes para los servicios de grupo. Para obtener más información sobre las ofertas de servicio en Bookings, consulte [Definición de las ofertas de servicio en Bookings](define-service-offerings.md).

Asegúrese de establecer el **número máximo de asistentes** en más de 1 en el servicio Bookings.

## <a name="update-attendee-list"></a>Actualización de la lista de asistentes

> [!NOTE]
> Debe tener permiso de administrador o miembro del equipo para poder actualizar esta configuración.

1. En Microsoft 365, seleccione el iniciador de aplicaciones y, a continuación, seleccione **Bookings**.

2. Elija el calendario, seleccione la reunión de grupo y elija **Editar**.

:::image type="content" source="../media/group-booking-calendar.png" alt-text="Captura de pantalla: Vista de propietario del calendario de reserva de grupos":::

3. Para agregar asistentes a la reserva de grupos, seleccione **Agregar cliente**.

:::image type="content" source="../media/group-booking-add-customer.png" alt-text="Captura de pantalla: Página de detalles del servicio de reserva de grupos: agregar un cliente":::

4. Para quitar a los asistentes, seleccione el usuario que desea quitar de la reserva de grupo y seleccione **Quitar cliente**. Una vez que quites al asistente, obtendrán un correo electrónico para informarle de que se han quitado de esta reserva.

## <a name="common-issues"></a>Problemas comunes

- La edición de un servicio de reserva de grupos desde la aplicación de Teams o desde la aplicación móvil restablece a los asistentes a 1. Use la aplicación web Bookings para editar el servicio de reserva de grupos.

- Las reservas de grupos no admiten reservas de recursos en este momento.
