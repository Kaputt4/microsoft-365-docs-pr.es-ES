# <a name="get-access-to-microsoft-bookings"></a>Obtener acceso a Microsoft Bookings

En este tema se describe cómo obtener acceso a las reservas por primera vez. También le indica cómo activar o desactivar las reservas en su organización. También se explica cómo se pueden asignar licencias a los usuarios para que puedan tener acceso y usar reservas.

## <a name="access-bookings-for-the-first-time"></a>Obtener acceso a las reservas por primera vez

Si el administrador de ti le ha concedido acceso a reservas, puede acceder a la aplicación a través de Office online.

La primera vez que use [reservas](https://outlook.office.com/bookings/onboarding) , creará un calendario y configurará su empresa.

1. Seleccione **obtenerlo ahora** y elija **Agregar un calendario de reserva**. Si no es la primera vez que se encuentra en esta página, puede seleccionar otro calendario de reserva o realizar una búsqueda de otro calendario de reserva.

:::image type="content" source="media/bookings - get it now.png" alt-text="Captura de pantalla: Página de reservas de obtener ahora. Seleccione obtener ahora para empezar a usar reservas":::

2. Escriba el nombre de la empresa y el tipo de empresa y seleccione **continuar**.

Ya está listo para configurar las reservas para su organización. Puede acceder a la página de reservas con este [vínculo](https://outlook.office.com/bookings/onboarding), desde el iniciador de aplicaciones o desde Office.com.
