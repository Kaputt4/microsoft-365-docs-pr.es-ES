# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>Gráfico de comparación: aplicación Web de reservas frente a la aplicación de Microsoft Teams

La aplicación de reservas de Microsoft Teams permite a los programadores controlar sus tareas principales y cambiar algunas opciones de configuración. Sin embargo, la Web de reservas incluye características avanzadas y configuraciones que aún no están disponibles en la aplicación de Microsoft Teams.

Las características de la aplicación Teams se agregan en todo momento y seguirán actualizando la lista. Consulte la sección **¿todas las funciones de la aplicación Web de reservas originales están disponibles en Microsoft Teams?** en las [preguntas más frecuentes](bookings-faq.md) para obtener más información.

| Característica | Aplicación web Bookings | Aplicación de los equipos de reservas |
|:---|:---|:---|
| Crear nuevos calendarios de reserva | Sí | Sí |
| Agregar y quitar personal de un calendario de reserva | Sí | Sí |
| Crear nuevos tipos de cita | Sí | Sí |
| Programar reservas en línea | Sí | Sí |
| Editar detalles profesionales | Sí | Sí |
| Agregar personal con permisos de invitado a personas de fuera de la organización | Sí | No |
| Programar reservas en persona o sin conexión | Sí | No |
| Grupo de programación o reservas para varios clientes | Sí | No |
| Establecer un logotipo para su empresa | Sí | No |
| Establecer horario comercial | Sí | No |
| Publicación de una página de programación de autoservicio | Sí | No |
| Administrar contactos de clientes | Sí | No |
| Establecer el tiempo libre para el personal | Sí | No |
