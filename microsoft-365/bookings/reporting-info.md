# <a name="reporting-info-for-bookings"></a>Información de informes para Bookings

Ahora puede ver una vista de 120 días del calendario de Booking en un archivo TSV. Los archivos TSV son archivos de valores separados por tabulaciones. Puede ver o editar un archivo como este con cualquier editor de texto u programa de hoja de cálculo, como Excel.

## <a name="see-120-days-booking-activity"></a>Ver actividad de reserva de 120 días

1. En el panel calendario de reserva, seleccione Exportar más datos como TSV.

:::image type="content" source="../media/bookings-activity.png" alt-text="Screenshot: 30 days Booking activities":::

1. Guarde el archivo con un nombre nuevo y especifique el formato .xls o xlsx.

1. Abra el archivo para ver la vista de 120 días del calendario de Booking.

El archivo contiene una fila para cada evento del calendario de reserva de los 60 días anteriores y los próximos 60 días. Cada fila incluye la información de fecha y hora del evento, el nombre del cliente, el correo electrónico del cliente, el número de teléfono del cliente, la dirección del cliente, el personal asignado, el servicio seleccionado, la ubicación y la duración de la cita.
