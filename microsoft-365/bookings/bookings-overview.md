# <a name="microsoft-bookings"></a>Microsoft Bookings

Microsoft bookings facilita la programación y la administración de citas de una manera sencilla. Bookings incluye un calendario de reserva basado en Web y se integra con Outlook para optimizar el calendario del personal y proporcionar a los clientes flexibilidad para reservar una hora que funcione mejor para ellos. Los mensajes de correo electrónico de notificación automatizada reducen el número de programas y mejoran la satisfacción del cliente y las organizaciones ahorran tiempo con una reducción de las tareas de programación repetitivas. Con una flexibilidad y capacidad de personalización integradas, las reservas pueden diseñarse para adaptarse a la situación y las necesidades de muchas partes distintas de una organización. El calendario de reservas es un buzón de correo en Exchange Online.

Bookings le ofrece la posibilidad de hacer que las reuniones de su organización sean virtuales con reuniones en línea a través de [Microsoft Teams](https://support.microsoft.com/office/overview-of-the-bookings-app-in-teams-7b8569e1-0c8a-444e-b712-d9968b05110b) y Skype empresarial. Cada cita reservada como reunión en línea crea un vínculo de reunión único que se envía a los asistentes para que puedan unirse a través de un explorador Web, un teléfono de acceso telefónico local o la aplicación de Skype o Teams. Las reservas también están disponibles como una aplicación dentro de Microsoft Teams, lo que le permite crear calendarios de reservas, asignar personal y programar nuevas y administrar citas existentes sin dejar nunca los equipos.

> [!NOTE]
> Las reservas están activadas de forma predeterminada para los clientes que tienen las suscripciones Microsoft 365 Business Standard, Microsoft 365 a3 y Microsoft 365 A5. Las reservas también están disponibles para los clientes que tienen Office 365 Enterprise E3 y Office 365 Enterprise E5, pero están desactivados de forma predeterminada. Para empezar, vea [obtener acceso a Microsoft bookings](get-access.md). Para activar o desactivar las reservas, consulte [activar o desactivar las reservas para su organización](turn-bookings-on-or-off.md).

Las reservas tienen tres componentes principales:

- Una página de reserva en la que los clientes y clientes pueden programar citas con el miembro del personal que debe proporcionar el servicio o ejecutar la cita. Esta página de programación basada en Web se puede compartir a través de un vínculo directo, la página de Facebook e incluso a través de la incrustación de vínculos en el sitio Web.

- Una aplicación web que contiene un conjunto de páginas orientadas a la empresa basadas en Web en las que los propietarios y los administradores de calendario de reservas dentro de una organización pueden definir los detalles y los tipos de citas, administrar la disponibilidad y programación del personal, establecer el horario comercial y personalizar el modo en que se programan las citas. Estas páginas permiten la versatilidad y la capacidad de personalizar un calendario de reservas para ajustarse a las diversas necesidades de la persona u organización.

- Una aplicación móvil orientada a la empresa donde los propietarios y administradores del calendario de reservas pueden ver todas sus citas, acceder a las listas de clientes y a la información de contacto, y realizar reservas manuales en cualquier lugar.
