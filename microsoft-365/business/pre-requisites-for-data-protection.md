# <a name="prerequisites-for-protecting-data-on-devices-with-microsoft-365-for-business"></a>Requisitos previos para proteger datos en dispositivos con Microsoft 365 para empresas

Este artículo se aplica a Microsoft 365 Empresa Premium.

El primer paso para configurar su organización con Microsoft 365 para empresas es asegurarse de que puede cumplir los requisitos previos.
  
## <a name="requirements-for-setting-up-your-organization-with-microsoft-365-for-business"></a>Requisitos para configurar su organización con Microsoft 365 para empresas

- Windows dispositivos deben ejecutarse Windows 7 Professional, Windows 8 Pro o Windows 8.1 Pro.
    
    [Actualizar dispositivos Windows a Windows Pro Creators Update](upgrade-to-windows-pro-creators-update.md)
    
    Si está ejecutando Windows 10 Home, debe **comprar** Windows 10 Pro. Consulte [upgrade Windows 10 Home to Windows 10 Pro](../business-video/upgrade.md) para obtener instrucciones. 
    
- Quite los dispositivos de las soluciones de administración móvil (Mobile Iron, AirWatch, y así sucesivamente). Inscribirá a todas las personas de su organización en Microsoft 365 para la administración de dispositivos móviles empresariales.
    
- Apple iOS 8.0 y versiones posteriores.
    
    Google Android 4.0 y versiones posteriores (incluido Samsung KNOX Standard 4.0 y versiones superiores). Para obtener más información, consulta [Dispositivos compatibles con Intune.](/mem/intune/fundamentals/supported-devices-browsers)
    
- Si tiene aplicaciones de Office existentes en equipos de usuario [Office,](prepare-for-office-client-deployment.md) lea preparar la instalación de un cliente para comprender los pasos que puede que tenga que seguir antes de configurar Microsoft 365 para empresas para instalar Office 2016 en equipos de usuario.
