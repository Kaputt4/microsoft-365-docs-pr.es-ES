# <a name="difference-between-mdm-and-mam"></a>Diferencia entre MDM y MAM

Microsoft 365 Empresa Premium ofrece varias maneras de proteger los datos empresariales. Consulte [Introducción a Microsoft 365 Empresa Premium](../../admin/admin-overview/what-is-microsoft-365.md) para obtener más información sobre las distintas protecciones que se configuran automáticamente y lo que puede configurar para proteger aún más su negocio. También puede configurar directivas que protejan los dispositivos Windows 10 y los datos de los dispositivos móviles.
[Establezca la configuración de protección de aplicaciones para dispositivos Windows 10](../../business-premium/m365bp-protection-settings-for-windows-10-devices.md).

## <a name="mobile-device-management-or-mdm"></a>Administración de dispositivos móviles o MDM

Microsoft 365 Empresa Premium permite configurar directivas que protegen los datos de los dispositivos Windows 10. Cuando un dispositivo está bajo administración de dispositivos móviles, controla todo el dispositivo y puede borrar datos de él y restablecerlos también a la configuración de fábrica. Para obtener más información, consulte [Establecer la configuración de protección de dispositivos para equipos Windows 10](../../business-premium/m365bp-protection-settings-for-windows-10-devices.md).

## <a name="mobile-application-management-or-mam"></a>Administración de aplicaciones móviles o MAM

La administración de aplicaciones móviles le permite controlar los datos empresariales en los dispositivos personales de los usuarios, como iPhones y Android, y sus equipos Win 10 personales. Puede usar directivas de administración de aplicaciones para evitar que los usuarios copien datos empresariales de Office aplicaciones a sus aplicaciones personales. También puede quitar todos los datos de las aplicaciones de Office en sus dispositivos personales. Para obtener más información, vea [Establecer la configuración de protección de aplicaciones para dispositivos Android o iOS](../../business-premium/m365bp-app-protection-settings-for-android-and-ios.md) y [Establecer la configuración de protección de aplicaciones para dispositivos Windows 10](../../business-premium/m365bp-app-protection-settings-for-android-and-ios.md).
