# <a name="difference-between-mdm-and-mam"></a>Diferencia entre MDM y MAM

Microsoft 365 Empresa Premium ofrece varias maneras de proteger los datos empresariales. Consulte [Introducción a Microsoft 365 Empresa Premium](../../admin/admin-overview/what-is-microsoft-365.md) para obtener más información sobre las distintas protecciones que se configuran automáticamente y lo que puede configurar para proteger aún más su negocio. También puede incorporar dispositivos para Microsoft Defender para Empresas para proteger los dispositivos Windows y los datos de los dispositivos móviles. Para obtener más información, consulte [Incorporación de dispositivos inscritos a Microsoft Defender para Empresas](/microsoft-365/business-premium/m365bp-onboard-devices-mdb). 

## <a name="mobile-device-management-or-mdm"></a>Administración de dispositivos móviles o MDM

Microsoft 365 Empresa Premium permite configurar directivas que protegen los datos de los dispositivos Windows 10 y 11. Cuando un dispositivo está bajo administración de dispositivos móviles, controla todo el dispositivo y puede borrar datos de él y restablecerlos también a la configuración de fábrica. Para obtener más información, consulte [Elección de la solución de administración de dispositivos adecuada para usted](/mem/intune/fundamentals/what-is-device-management#choose-the-device-management-solution-thats-right-for-you) y [Configuración y protección de dispositivos administrados](/microsoft-365/business-premium/m365bp-protect-devices).

## <a name="mobile-application-management-or-mam"></a>Administración de aplicaciones móviles o MAM

La administración de aplicaciones móviles le permite controlar los datos empresariales en los dispositivos personales de los usuarios, como iPhones y Android, y sus equipos Windows personales. Puede usar directivas de administración de aplicaciones para evitar que los usuarios copien datos empresariales de aplicaciones de Office en sus aplicaciones personales. También puede quitar todos los datos de las aplicaciones de Office en sus dispositivos personales. Para obtener más información, consulte [Elección de la solución de administración de dispositivos adecuada para usted](/mem/intune/fundamentals/what-is-device-management#choose-the-device-management-solution-thats-right-for-you) y [Configuración y protección de dispositivos administrados](/microsoft-365/business-premium/m365bp-protect-devices).
