# <a name="upgrade-windows-devices-to-windows-pro-creators-update"></a>Actualizar dispositivos Windows a Windows Pro Creators Update

## <a name="upgrade-to-windows-10-pro-creators-update"></a>Actualización a Windows 10 los creadores de Pro actualizar

1. Inicie sesión el [Centro de administración de Microsoft](https://portal.office.com/adminportal/home) y busque la tarjeta de **Actualización de Windows 10** . 
    
    ![Tarjeta de actualización de 10 de Windows en el centro de administración.](media/066f47bf-7b88-4fea-8fd0-82798ea66716.png)
  
2. Para actualizar a Windows 10 Pro los creadores de Update, dispone de varias opciones. Puedes:
    
- Instalar la actualización desde el sitio de descarga de Software de Microsoft: seleccione esta opción si el dispositivo que ha iniciado sesión en que se encuentra en el mismo dispositivo que el que desea actualizar.
    
  a. Seleccione **instalar la actualización**.
    
  b. desde el software del sitio de descarga, haga clic en **Actualizar ahora** para empezar a actualizar el dispositivo a Windows 10 Pro los creadores de actualización. 
    
- Compartir el vínculo de descarga: seleccione esta opción si desea enviar un correo electrónico a las personas de su organización, por lo que puede actualizar a Windows 10 Pro por sí mismos.
 
   a. sitio de descarga seleccione vínculo de **recurso compartido de la descarga** para abrir un correo electrónico que contiene un vínculo al software de Windows. 
    
 - Crear un medio de instalación mediante la herramienta de creación de medios: seleccione esta opción para crear un medio de instalación de actualización de los creadores de Windows 10 Pro (unidad flash USB o archivo ISO) para instalar Windows 10 en un PC que es diferente del que se está utilizando.
    
    a. Seleccione **crear medios de instalación**.
    
    b. Lea las instrucciones acerca de cómo usar la herramienta y crear los discos de instalación. 
    
Vea [configurar los dispositivos de Windows para los usuarios de Microsoft Business de 365](set-up-windows-devices.md) para completar la configuración de dispositivos de Windows 10. 
  
Vea [Set up dispositivos móviles para los usuarios de Microsoft 365 empresarial](set-up-mobile-devices.md) completa configurar dispositivos Android y iOS.
