# <a name="microsoft-365-business-frequently-asked-questions"></a>Preguntas más frecuentes (puede estar en inglés) sobre Microsoft 365 Empresa.

## <a name="general"></a>General

### <a name="what-is-microsoft-365-business"></a>¿Qué es Microsoft 365 Empresa? 
Microsoft 365 es una solución integrada que reúne herramientas de productividad en su clase, seguridad y capacidades de administración de dispositivos para pequeñas y medianas empresas. 

**Un conjunto general de herramientas de productividad y colaboración para empresas** 
* Word, Excel, PowerPoint, Outlook, OneNote, Publisher y Access 
* Exchange, OneDrive, Skype Empresarial, Microsoft Teams, SharePoint 
* Aplicaciones empresariales de Office (Bookings, Outlook Customer Manager, MileIQ<sup>[1](#footnote1)</sup>, Microsoft Listings<sup>[1](#footnote1)</sup>, Microsoft Connections<sup>[1](#footnote1)</sup>, Microsoft Invoicing<sup>[1](#footnote1)</sup>) 
 
**Funcionalidades de seguridad y administración de dispositivos de nivel empresarial** 
- Ayuda a proporciona protección contra las amenazas externas como suplantación de identidad y malware sofisticado con protección de amenaza avanzada de Office 365 y Windows Defender aprovechan Guard 
-  Ayuda a controlar y administrar la información confidencial cómo se tiene acceso y se transmiten con directivas de prevención de pérdida de datos de Office 365 (que estará disponible en verano, 2018) y Azure información protección Plan 1 
-  Ayuda a proteger, conservar y copia de seguridad de los datos con Exchange Online Archiving  
-   Protección de la aplicación para Office y otras aplicaciones móviles con Intune App Protection 
-  Administración de dispositivos de 10 equipos con Windows, Mac OS y dispositivos móviles con la administración de dispositivos Intune 
-  Configuración de seguridad coherente en todos los dispositivos - protección de datos de la compañía en todos los dispositivos - Windows Defender, siempre en y actualizada 
 
 
**Implementación de dispositivos y configuración de usuarios simplificadas** 
* Consola de administración única para configurar y administrar usuarios y dispositivos 
* Instalación automática de aplicaciones de Office en equipos con Windows 10 
* Actualizar siempre Office + Windows 10 
* Implementación optimizada de equipos con Windows AutoPilot

### <a name="who-should-consider-adopting-microsoft-365-business"></a>¿Quién debería considerar la adopción de Microsoft 365 Empresa? 
Microsoft 365 Empresa se diseñó para los clientes pequeños y medianos que tienen pocos recursos de TI, o no tienen ninguno, entre sus empleados y buscan obtener las mejores capacidades de productividad y colaboración de Office 365 junto con soluciones de seguridad y administración de dispositivos para proteger los datos empresariales. El cliente de Microsoft 365 Empresa está listo para trasladar sus operaciones de TI a la nube y está interesado en el mantenimiento de una actitud proactiva para ayudar a proteger los datos de dispositivos tanto de la empresa como de los empleados.

### <a name="how-can-i-get-microsoft-365-business-for-my-business"></a>¿Cómo puedo obtener Microsoft 365 Empresa para mi negocio? 
Microsoft 365 Empresa se puede comprar a través de un <a href="https://www.microsoft.com/solution-providers/search" target="_blank">partner de Microsoft</a> o directamente a <a href="https://www.microsoft.com//microsoft-365/business" target="_blank">Microsoft</a>. A la hora de elegir entre comprar directamente a Microsoft o a través de un partner de Microsoft, deberías tener en cuenta las capacidades de tu personal y si quieres realizar el mantenimiento de una infraestructura de TI. Un partner de Microsoft puede ayudarte a implementar y administrar la infraestructura de TI, incluidas las soluciones de Microsoft.

### <a name="how-much-does-microsoft-365-business-cost"></a>¿Cuánto cuesta Microsoft 365 Empresa? 
Microsoft 365 empresarial se ofrecen al USD$ 20,00 usuario y mes, según un _contrato anual_ si adquirido directamente a Microsoft. Cuando se adquiere a través de un Partner de Microsoft, precios pueden variar en función de los servicios que proporciona el socio y su modelo de precio para la empresa de Microsoft 365. 

### <a name="is-there-a-cap-to-how-many-microsoft-365-business-seats-a-customer-can-have"></a>¿Existe un límite a cuántos puestos de Microsoft 365 Empresa un cliente puede tener? 
Microsoft 365 Empresa se diseñó para pequeñas y medianas empresas con requisitos de complejidad de TI bajos a medianos. Los clientes pueden comprar un máximo de 300 licencias de Microsoft 365 Empresa para su organización. Los clientes pueden mezclar y combinar las suscripciones de nube; como resultado, en función de los requisitos de TI de la organización, los clientes pueden agregar licencias de Microsoft 365 Enterprise a la misma cuenta. 
 
Al considerar un entorno que consta de varios tipos de suscripción, los clientes deberían trabajar con sus asesores de TI de confianza para determinar la mejor forma para administrar y proteger las distintas suscripciones, ya que Microsoft 365 Empresa y Microsoft 365 Enterprise usan diferentes capacidades para proteger y administrar las aplicaciones y los datos.

### <a name="can-i-combine-microsoft-365-business-with-other-microsoft-subscription-offerings"></a>¿Puedo combinar Microsoft 365 Empresa con otras ofertas de suscripción de Microsoft? 
Sí, los clientes pueden combinar sus suscripciones de Microsoft 365 Empresa con planes y complementos de Azure, Dynamics 365, Enterprise Mobility + Security, y Office 365.

### <a name="is-everyone-in-my-business-required-to-have-a-microsoft-365-business-subscription"></a>¿Todos los empleados de mi empresa tienen que tener una suscripción de Microsoft 365 Empresa? 
No, no todos los empleados necesitan una suscripción de Microsoft 365 Empresa. Sin embargo, los beneficios de seguridad y administración solo están disponibles para los usuarios con dispositivos administrados mediante una suscripción de Microsoft 365 Empresa. 
 
La estandarización de un entorno de TI sirve para ayudar a reducir los costos de mantenimiento y seguridad con el tiempo y es un estado que las empresas deberían buscar alcanzar. Sin embargo, sabemos que algunos clientes de tamaño pequeño y mediano actualizan su software principalmente cuando renuevan el hardware, a lo largo de un período prolongado. Las empresas pueden implementar Microsoft 365 Empresa a parte de su organización, pero para lograr la mejor protección de datos empresariales confidenciales y obtener experiencias de colaboración coherentes, se recomienda que la implementación se realice para todos los usuarios.

### <a name="how-can-i-know-if-the-hardware-and-software-i-run-today-is-compatible-with-microsoft-365-business"></a>¿Cómo puedo saber si el hardware y software que ejecuto hoy es compatible con Microsoft 365 Empresa? 
Si el hardware que ejecute hoy se ejecuta Windows 7 Pro o versiones posteriores, es probable que cumpla los requisitos mínimos para la empresa de 365 de Microsoft. Algunas características de Windows 10 como Cortana, Windows Hola y multitoque requieren un hardware específico que sólo está disponible en los equipos más recientes. Consulte los <a href="https://www.microsoft.com/windows/windows-10-specifications" target="_blank">requisitos del sistema para profesionales de Windows 10</a> para obtener más información. 
 
Windows 10 admite la mayoría de las aplicaciones de escritorio (Win32) existentes y la mayoría de las aplicaciones existentes funcionan correctamente sin tener que realizar ningún cambio. Los clientes y sus asesores de TI de confianza deben leer el proceso de pruebas de aplicaciones recomendado sobre la <a href="https://docs.microsoft.com/windows/deployment/planning/windows-10-compatibility#recommended-application-testing-process" target="_blank">compatibilidad con Windows 10</a> y consultar los <a href="https://products.office.com/office-system-requirements#subscription-plans-section" target="_blank">requisitos del sistema de Office</a> para facilitar la transición a Microsoft 365 Empresa. 

### <a name="what-is-windows-10-business"></a>¿Qué es Windows 10 Business? 
Windows 10 Business es un conjunto de servicios en la nube y de capacidades de administración de dispositivos que complementan Windows 10 Pro y habilitan los controles de seguridad y administración centralizados de Microsoft 365 Empresa. Windows 10 Business también incluye Windows, AutoPilot, un servicio que agiliza la implementación de nuevos equipos con Windows 10. Si tienes dispositivos con licencia para Windows 7, 8 y 8.1 Professional, Microsoft 365 Empresa proporciona una actualización a Windows 10 Pro, que es el requisito previo para implementar Windows 10 Business. 

### <a name="how-does-microsoft-365-business-help-support-our-bring-your-own-device-byod-policy"></a>¿Cómo ayuda Microsoft 365 Empresa a apoyar nuestra directiva de BYOD (Bring Your Own Device)? 
Muchos empleados prefieren usar sus propios teléfonos móviles o tabletas para acceder a información personal y del trabajo, en lugar de llevar varios dispositivos para cada propósito. El uso de dispositivos personales para el trabajo, si bien es común, aumenta el riesgo de que la información de la empresa acabe en manos de personas no autorizadas. Muchas soluciones de protección de datos móviles de otros proveedores requieren que los usuarios cambien a un modo concreto en su dispositivo o que usen otro mecanismo complejo que quizás sea intrusivo y, por lo tanto, los usuarios evitan usarlo. 
 
Microsoft 365 Business ofrece a los clientes un medio sencillo pero eficaz de la habilitación de los empleados para usar sus dispositivos personales para el trabajo con la capacidad de ofrecer la empresa para evitar que esos dispositivos desde tener acceso a, retener o uso compartido de información de la empresa . Más concretamente: 
* **Protección de aplicación para aplicaciones móviles de Office** ayuda a proteger los datos de Office, incluidos el correo electrónico, calendario, contactos y documentos en iOS y Android dispositivos móviles, al aplicar las directivas como la eliminación automática de datos profesionales después de una cantidad de tiempo exigida de no se conecta al servicio, que requieren que la información se almacena sólo a OneDrive para la empresa, que requieren una comprobación de huella o PIN para tener acceso a aplicaciones de Office y lo que impide datos de la empresa desde la que se copian desde una aplicación de Office en aplicaciones personal. Administración de aplicaciones móviles para otras aplicaciones móviles a través de Intune también está disponible para los suscriptores de Microsoft 365 Business. 
  
* **Administración de dispositivos para Windows 10 PCs** aallows empresas para elegir establecer y aplicar capacidades tales como la protección de Windows Defender para malware, actualizaciones automáticas y desactivar pantallas después de un período de tiempo establecido. Además, los dispositivos de Windows 10 perdidos o robados pueden ser completamente en blanco de aplicaciones empresariales y datos a través del centro de administración de. 

* Características de **Administración de dispositivos para iOS, Android & Mac OS** ayuda a las empresas administrar de forma segura un ecosistema de diversos dispositivos que incluye dispositivos iOS, Android, Windows y Mac OS. 

### <a name="how-does-microsoft-365-business-help-protect-pcs-in-my-organization-from-malicious-attacks"></a>¿Cómo ayuda Microsoft 365 Empresa a proteger los equipos de mi organización contra ataques malintencionados? 
PC administrados con Microsoft 365 empresa está protegidos con Windows Defender, que es la característica antivirus Nº 1 en 10 de Windows, la protección de más equipos contra virus, malware, spyware y otras amenazas que cualquier otra solución.  Con Microsoft 365 Business, pueden asegurarse de empresas protección de Windows Defender está en ejecución y siempre actualizados en todos sus dispositivos de Windows 10.  
 
Windows 10 Business también incluye a Windows Defender aprovechan Guard, un nuevo conjunto de capacidades de prevención de intrusiones. Uno de sus características, el acceso a la carpeta controlada, detiene ransomware en seco al bloqueo de carpetas e impedir que aplicaciones no autorizadas tengan acceso a archivos importantes de un usuario. 

### <a name="whats-the-difference-between-office-365-business-premium-microsoft-365-business-and-microsoft-365-enterprise"></a>¿Cuál es la diferencia entre Office 365 Empresa Premium, Microsoft 365 Empresa y Microsoft 365 Enterprise? 
Microsoft cuenta con una variedad de ofertas de productividad y administración de seguridad que los clientes pequeños y medianos pueden tener en cuenta a la hora de actualizar su infraestructura de escritorios y dispositivos, y cada una proporciona características y funcionalidades cada vez más eficaces. 

**Office 365 Empresa Premium** ofrece la mejor productividad de su clase con servicios y aplicaciones de Office 365, pero no incluye las capacidades de administración de dispositivos y protección de aplicaciones de Microsoft 365 Empresa.

**Microsoft 365 Business** combina servicios y aplicaciones de Office 365 con la administración de aplicaciones móviles y profesionales de TI de Windows 10 para habilitar la administración remota y ayudar a proteger los dispositivos frente a virus y malware. Incluye una consola de administración simplificada a través de los dispositivos y los datos pueden administrarse de directivas. Muchas empresas pequeñas y medianas se pueden atender mejor con Microsoft 365 empresarial. 
 
**Microsoft 365 Enterprise** es un conjunto de licencias planes que ofrecen mayores niveles de cumplimiento y administración de la seguridad a través de Microsoft 365 Business y están diseñados para los clientes de empresa y los clientes que tienen más de 300 usuarios. Además, los planes de Microsoft 365 Enterprise proporcionan funcionalidad adicional incluida la inteligencia empresarial y las herramientas de análisis.

Sí, los clientes pueden cambiar sus planes de un plan de Office 365 elegible a Microsoft 365 Empresa. En función del plan actual del cliente, es posible que se produzca una disminución o un aumento de los cargos mensuales.

Una versión de evaluación de Microsoft 365 Business está disponible para los CSP. Una versión de prueba para guiar a los clientes estará disponible más adelante. 
 
### <a name="what-should-customers-and-partners-know-before-running-microsoft-365-business-within-their-organization"></a>¿Qué deben saber los clientes y partners antes de ejecutar Microsoft 365 Empresa dentro de su organización? 
Los clientes que deseen experimentar las capacidades completas de negocio de Microsoft 365 deben ejecutar Windows 7, 8.1 o 10 Pro<sup>[2](#footnote2)</sup> en sus equipos de escritorio existentes. ## <a name="deployment"></a>Implementación

### <a name="what-should-customers-consider-when-planning-a-microsoft-365-business-deployment"></a>¿Qué deben tener en cuenta los clientes a la hora de planificar una implementación de Microsoft 365 Empresa? 
El camino más directo a una implementación de Microsoft 365 Empresa correcta es a través de un partner de Microsoft. Él cuenta con amplia entrenamiento y experiencia en una variedad de escenarios de cliente y está mejor preparado para comprender tus necesidades y entorno. ### <a name="does-microsoft-365-business-include-the-full-capabilities-of-microsoft-intune"></a>¿Microsoft 365 Empresa incluye las capacidades completas de Microsoft Intune? 
Sí, los suscriptores de Microsoft 365 Business se licencia para usar Intune funcionalidad completa de las para iOS, Android, Mac OS y otro administración de dispositivos de multiplataforma. Características no disponibles en la consola de administración simplificada en Microsoft 365 empresarial, al igual que 3 de administración de aplicaciones de terceros y la configuración de cosas como perfiles WiFi, certificados VPN, etc., se pueden administrar en la consola de Intune completa.   
  

### <a name="does-azure-active-directory-premium-p1-come-with-microsoft-365-business"></a>¿Se incluyen Azure Active Directory Premium P1 con Microsoft 365 empresarial? 
Microsoft 365 empresarial se basa en la tecnología de a través de Microsoft y mientras comparte algunas de las características con Azure Active Directory, no es una versión completa. La seguridad y las directivas de administración creadas en Microsoft 365 empresarial se basan en algunas de las funciones Azure pero no incluye todas las características (por ejemplo, Self-servicios características, características de acceso condicional y reporting). Los clientes pueden elegir comprar Azure Active Directory Premium P1 o P2 como un complemento de Microsoft 365 empresarial. Consulte la [Descripción del servicio de Microsoft 365 empresarial](https://technet.microsoft.com/library/b042d1fd-c997-46d8-8c02-1df4c932f148) para obtener más información 
 
 
### <a name="does-microsoft-365-business-allow-customers-to-manage-macs"></a>¿Microsoft 365 Empresa permite a los clientes administrar equipos Mac? 
Intune le ayuda a administrar de forma segura los dispositivos iOS, Android, Windows y Mac OS. 
 
### <a name="what-is-windows-autopilot"></a>¿Qué es Windows AutoPilot? 
Piloto automático de Windows es un servicio que simplifica la implementación de nuevo Windows 10 PCs. Este proceso se puede realizar cuando el usuario final inicia sesión en Microsoft Business de 365 por primera vez, sin TI nunca toca el dispositivo: al aprovechar los controles de administración centralizada de negocio de Microsoft 365. También puede usar el piloto automático de Windows para equipos existentes que ejecutan Windows 10 Professional los creadores de actualización (o posterior) y han sido restablecimiento de fábrica. Obtener información detallada sobre el piloto automático de Windows puede encontrarse en <a href="https://blogs.technet.microsoft.com/windowsitpro/2017/06/29/modernizing-windows-deployment-with-windows-autopilot/" target="_blank">la entrada de blog de 2017 este junio</a>.

## <a name="compatibility"></a>Compatibilidad

### <a name="can-i-add-office-365-add-ons-to-microsoft-365-business"></a>¿Puedo agregar complementos de Office 365 a Microsoft 365 Empresa? 
Todos los complementos que pueden agregarse a Office 365 Empresa Premium se pueden agregar a Microsoft 365 Empresa. Esto significa que puedes comprar la protección contra amenazas avanzada, Office 365 Cloud App Security, Cumplimiento avanzado, Inteligencia de amenazas, MyAnalytics, PowerBI Pro y Audioconferencia. 
 
### <a name="can-i-add-phone-system-and-calling-plans-to-microsoft-365-business"></a>¿Puedo agregar planes de llamadas y sistemas telefónicos a Microsoft 365 Empresa? 
En el plan de llamadas y sistemas telefónicos está reservado para los clientes que tienen necesidades más avanzadas. Los clientes que necesiten estas capacidades deberían mirar las ofertas de Microsoft 365 Enterprise. 
 
### <a name="can-microsoft-365-business-customers-use-windows-defender-advanced-threat-protection"></a>¿Los clientes de Microsoft 365 Empresa pueden usar la Protección contra amenazas avanzada de Windows Defender? 
No, los clientes que requieren la Protección contra amenazas avanzada de Windows Defender necesitan Windows 10 Enterprise E5 o Microsoft 365 Enterprise E5. 
 


### <a name="can-customers-create-hosted-windows-10-vms-with-a-microsoft-365-business-subscription"></a>¿Los clientes pueden crear máquinas virtuales Windows 10 hospedadas con una suscripción de Microsoft 365 Empresa? 
No, los clientes que necesitan capacidades de virtualización deberían comprar Windows 10 Enterprise o una suscripción de Microsoft 365 Enterprise.

### <a name="what-is-the-gdpr-and-how-does-microsoft-365-business-help-customers-with-their-compliance-obligations"></a>¿Qué es GDPR y cómo ayuda Microsoft 365 Empresa a los clientes con sus obligaciones de cumplimiento?
El Reglamento de protección de datos General (GDPR) es una nueva ley de privacidad completa que proporciona a los residentes del control mayor de la Unión Europea (UE) a través de sus "datos personales" y requiere que las organizaciones a mantener la integridad de los datos personales. El GDPR requiere que las organizaciones dicho control, o proceso de datos personales vinculadas a los residentes de la UE para usar sólo procesadores de datos de otros fabricantes que cumplen los requisitos de GDPR para el procesamiento de datos personal. En de 2017 de marzo, Microsoft realizados garantías contractuales disponibles que proporcionan estas garantías.
